# 概念-外部来源关联

## 职责边界

- WisdomLedger 负责 SQL、业务口径、字段语义、指标语义和血缘关系。
- `my-dev-brain` 负责监管原文、填报说明、报表页、主题页、规则页和报表清单。
- 两者的关系是“语义层引用原文层”，而不是“在两个仓库各存一份原文副本”。

## 在本库如何记录外部来源

- 在 `知识库/来源/` 页面记录 `external_vault`
- 记录 `external_paths`
- 记录 `search_keywords`
- 记录基于外部来源提炼出的结论
- 不复制监管原文全文

## 适合放在本库的内容

- SQL 结构抽取
- 业务自然语言口径
- 字段与指标的语义映射
- 表到表、字段到字段、指标到字段的血缘

## 不适合放在本库的内容

- 填报说明全文
- 监管制度原文全文
- 报表清单副本
- 在 `my-dev-brain` 已存在的原文摘录大段复制
