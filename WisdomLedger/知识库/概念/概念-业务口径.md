# 概念-业务口径

## 定义

- 业务口径是业务语言对统计对象、时间边界、纳入范围、排除范围、维度和算法的统一描述。
- 它不是 SQL 原样翻译，而是把 SQL 背后的统计意图说清楚。

## 一个完整口径至少包含

- 统计对象是谁
- 在什么时间点或时间段统计
- 统计的是余额、笔数、户数还是金额
- 哪些记录纳入
- 哪些记录排除
- 结果按什么维度输出

## 和 SQL 的关系

- SQL 是技术实现。
- 业务口径是可被业务、监管、数据团队共同复核的语义层。
- 一个业务口径应能追溯到明确的 SQL 证据和源字段。
