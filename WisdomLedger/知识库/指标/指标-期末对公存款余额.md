# 指标-期末对公存款余额

## 自然语言口径

- 指截至报表日营业终了时，符合监管口径的对公活跃存款账户余额之和。
- 该指标按产品类型汇总，用于反映对公存款规模。

## 计算规则

| 项目 | 内容 |
| --- | --- |
| SQL 表达式 | `SUM(balance)` |
| 统计粒度 | `report_date x product_type` |
| 数据来源 | [[数据表-账户余额日快照事实表]] |
| 客户筛选 | 通过 [[数据表-客户维表]] 识别对公客户和内部客户 |

## 纳入与排除

- 纳入 `snapshot_date = ${biz_date}` 的快照记录。
- 纳入 `status = 'ACTIVE'` 的账户。
- 纳入 `customer_type = 'CORP'` 的客户。
- 排除 `is_internal = 0` 之外的内部客户。

## 来源

- [[数据表-账户余额日快照事实表]]
- [[数据表-客户维表]]

## 风险提示

- 如 `balance` 包含冻结、应计或其他扩展金额成分，需要再细化 `balance` 的业务定义。
- 如账户和客户关系存在历史滞后，客户属性与余额快照的时点一致性需要确认。
