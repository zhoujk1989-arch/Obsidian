# 指标-活跃对公账户数

## 自然语言口径

- 指截至报表日，符合监管口径的对公活跃存款账户去重后的账户数。
- 该指标用于反映当前仍处于有效状态、并纳入报表统计范围的对公账户规模。

## 计算规则

| 项目 | 内容 |
| --- | --- |
| SQL 表达式 | `COUNT(DISTINCT account_no)` |
| 统计粒度 | `report_date x product_type` |
| 去重口径 | 以账号 `account_no` 去重 |
| 数据来源 | [[数据表-账户余额日快照事实表]] |

## 纳入与排除

- 纳入口径与 [[指标-期末对公存款余额]] 一致。
- 仅统计活跃、对公、非内部客户账户。

## 来源

- [[数据表-账户余额日快照事实表]]
- [[数据表-客户维表]]
- [[来源-示例-对公存款余额日报]]

## 风险提示

- 如果一个账号在同一天快照内存在多条记录，需要确认 `COUNT(DISTINCT account_no)` 是否已完全避免重复。
- 如监管要求按账户主键而非账号统计，应同步调整口径页和 SQL 实现。
