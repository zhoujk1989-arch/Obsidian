## Key Findings

- 统计时点为 `${biz_date}` 的账户余额日快照。
- 仅纳入活跃账户，即 `status = 'ACTIVE'`。
- 仅纳入对公客户，即 `customer_type = 'CORP'`。
- 排除内部客户，即 `is_internal = 0`。
- 最终按 `product_type` 输出余额和账户数。
- 外部监管来源应进一步确认“余额”的监管定义、报送频率以及是否涉及币种折算。

## Evidence

```sql
FROM dwd_account_balance_snapshot a
WHERE a.snapshot_date = '${biz_date}'
  AND a.status = 'ACTIVE'
```

```sql
INNER JOIN dim_customer c
    ON s.customer_id = c.customer_id
WHERE c.customer_type = 'CORP'
  AND c.is_internal = 0
```

```sql
SUM(balance) AS ending_balance,
COUNT(DISTINCT account_no) AS active_account_cnt
```
