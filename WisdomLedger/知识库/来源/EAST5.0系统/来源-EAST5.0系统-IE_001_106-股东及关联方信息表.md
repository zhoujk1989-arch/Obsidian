## Key Findings

- `IE_001_106` 的业务名称为 `股东及关联方信息表`，本地建表注释为"股东及关联方信息表"。
- DDL 当前包含 `29` 个字段，字段注释中标注 `PK` 的核心标识为：`CJRQ`, `NBJGH`。
- 一表通映射规则（Excel 第 76 行）定义了两种报送分支：
  - 上市银行（A060029='1'）：过滤持股比例≥1% 或 驻派董监事=1
  - 未上市银行（A060029='0'）：全量报送
  - 通用终态过滤：当月无效(00) 且上月也为无效(00) → 不报送
- 27 个 EAST 字段在一表通映射清单中有明确来源；额外 2 个字段（GSFZJG、SENSITIVEFLAG）在 DDL 中存在但映射规则未提供来源。
- 当前 SQL 草案已覆盖全部 29 个字段，其中 GSFZJG 按 NBJGH 兜底，SENSITIVEFLAG 置空，均标注待确认。
