# 报表-对公存款余额日报

## 业务定义

- 这张报表反映截至报表日营业终了时，本行对公客户存款账户的余额情况。
- 报表回答的问题是：在指定报表日，符合监管口径的对公活跃账户余额有多少，以及活跃账户数有多少。
- 输出按 `product_type` 分类，便于监管按产品观察对公存款结构。

## 统计口径

| 维度 | 说明 |
| --- | --- |
| 统计对象 | 对公客户的存款账户 |
| 统计时点 | `${biz_date}` 日快照 |
| 纳入条件 | 账户状态为活跃；客户类型为对公 |
| 排除条件 | 内部客户 |
| 输出维度 | `product_type` |
| 输出指标 | 期末对公存款余额、活跃对公账户数 |
