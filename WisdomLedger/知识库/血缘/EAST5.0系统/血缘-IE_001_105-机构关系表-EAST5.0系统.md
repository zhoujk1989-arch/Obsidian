## 加工逻辑摘要

1. 从一表通机构关系表（T_1_2）读取当日采集日期（A020020 = p_data_date）的所有记录。
2. LEFT JOIN 去重后的机构信息表（T_1_1），通过 A020001 = A010001 关联，获取本机构的支付行号、许可证号、机构名称。
3. IF A020002 ≠ '0'：LEFT JOIN 去重后的机构信息表，通过 A020002 = A010001 关联，获取上级机构的支付行号、机构名称。
4. IF A020002 = '0'：上级管理机构代码、上级管理内部机构号填'0'，上级管理机构名称填本机构名称。
5. 内部机构号由 A020001 从第12位截取。
6. GSFZJG、SENSITIVEFLAG 当前置 NULL。
