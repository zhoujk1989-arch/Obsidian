## 表级链路

```
一表通 T_1_6（股东及关联方信息）
    ├─ LEFT JOIN T_1_6（上月）─── 终态判断
    ├─ LEFT JOIN T_1_1（机构信息）─── 补金融许可证号、银行机构名称
    ├─ LEFT JOIN dim_code_mapping（代码映射表）─── 证件类别转换
    ├─ LEFT JOIN v_ybt_to_east_corp_cust（转east对公客户表）─── 对公客户统一编号
    └─ LEFT JOIN v_ybt_to_east_person_cust（转east个人客户表）─── 个人客户统一编号
         │
         ▼
    EAST5.0 IE_001_106（股东及关联方信息表）
```
