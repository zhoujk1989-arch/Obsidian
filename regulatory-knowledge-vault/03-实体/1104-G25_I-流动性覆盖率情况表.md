# 1104-G25_I-流动性覆盖率情况表

## 简介

G25_Ⅰ《流动性覆盖率情况表》是 1104 系统中的流动性监管核心报表，用于衡量银行在银保监会规定的流动性压力情景下，合格优质流动性资产（HQLA）对未来 30 天净现金流出的覆盖能力。本表与 G25_Ⅱ（净稳定资金比例）同属一张报表的不同部分，分别从短期（30 天压力情景）和长期（1 年稳定资金结构）两个时间维度刻画流动性风险。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银监统0027号（与G25_Ⅱ共用同一报表编号）
- 报表名称：流动性覆盖率及净稳定资金比例情况表（第Ⅰ部分）
- 当前原文依据：[[01-资料库/1104系统/2026-04-22-G25_I-流动性覆盖率情况表-填报说明-原文|G25_I 流动性覆盖率情况表填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、规则和常见口径

## 当前结论

- G25_Ⅰ 的核心指标是流动性覆盖率（LCR）= 合格优质流动性资产（HQLA）/ 净现金流出 ≥ 100%。
- LCR 从两个方向考察流动性：分子为合格优质流动性资产（可在压力情景下快速变现的资产），分母为净现金流出（现金流出减去封顶后的现金流入）。
- 填报机构范围与 G25_Ⅱ（NSFR）高度重合，包括政策性银行、大型商业银行、股份制商业银行、资产规模 2000 亿以上城商行、民营银行、农商行和外资法人银行。
- G25_Ⅰ 报送频度较为复杂：法人口径月报（月后13日）、合并报表季报（季后40日），并表口径手工报表每月报送不晚于月后18天。
- G25_Ⅰ 分子部分（合格优质流动性资产）分为一级资产和二级资产（2A+2B），其中二级资产占比不得超过40%，2B 资产占比不得超过15%。
- G25_Ⅰ 分母部分（净现金流出）= 现金流出 - MIN(现金流入，现金流出的75%)，体现了对现金流入的封顶限制。
- G25_Ⅰ 中我国存款保险制度属于"有效存款保险计划"但不满足"有效存款保险附加标准"，这是境外业务与境内业务的重要区分点。
- G25_Ⅰ 中"业务关系存款"的认定标准极为细化，需同时满足实质性依赖、法律效力合同、30天通知期/高转移成本、专门账户收益不足四个条件。
- G25_Ⅰ 的衍生产品净现金流计算采用逐笔计算→净额结算主协议判断→外汇衍生品本金互换例外三步法。
- G25_Ⅰ 的表间校验关系包括：[1.1.1_A]≤G01_[1.C]（法人口径下现金不超过 G01 资产合计）、[1.1.2A]<G01_[3.C]（法人口径下央行准备金不超过 G01 对应项目）。
- G25_Ⅰ 与 G25_Ⅱ（NSFR）共同构成流动性风险监管的"短期压力+长期结构"双维度框架，两者在现金、存款、准备金、表外项目等概念定义上完全复用。
- G25_Ⅰ 特定项目解释中，理财产品按保本/非保本区分处理：保本类计入无担保批发资金流出（一般为非业务关系存款），非保本类计入非契约性义务。
- G25_Ⅰ 特定项目解释中，大额存单按可转让性区分：不可转让按发行对象分类（零售/小企业/大中型企业），个人与机构间不可转让的分别计入零售/大中型企业，可转让的统一计入大中型企业（认定为非业务关系存款）。
- G25_Ⅰ 空头头寸处理中，由30天内逆回购/证券借入覆盖的超过30天空头头寸假定协议展期，现金流入为0%。
- G25_Ⅰ 中合格优质流动性资产的定义强调"无变现障碍"，即资产未被作为抵质押物或信用增级工具，未被用于支付经营成本。
- G25_Ⅰ 中买断式逆回购和证券借入中收到的符合 HQLA 定义的资产可纳入分子，但若交易对手有权在30天压力期内提取则不得计入。
- G25_Ⅰ 中二级资产的折算率：2A 资产按当前市场价值的85%计入，2B 资产按50%计入。
- G25_Ⅰ 中财政性存款按缴存中央银行财政性存款与财政性存款轧差后的净额填报：资产方反映在超额准备金中，负债方填入无担保批发现金流出。
- G25_Ⅰ 中同业存单处理：发行的30天内到期或提前可支取的同业存单计入"2.1.2.5未包含在以上无担保批发现金流出分类的其他类别"，持有的30天内到期的同业存单计入"2.2.2.6.3金融机构—其他借款和现金流入"。
- G25_Ⅰ 中担保融资交易中，以资产池为抵押品时全部视为非合格优质流动性资产计入"2.1.3.5由其他资产担保的融资交易"，若资产池中含本行持有的 HQLA 则从分子部分对应扣除。
- G25_Ⅰ 中远期交易处理：未来30天内生效并到期的远期回购/逆回购/抵押品互换对 LCR 计算无影响可忽略；未来30天内生效但30天后到期的按折扣系数调整后计入其他现金流入/流出。
- G25_Ⅰ 附注项目[2]用于自动计算二级资产40%和2B资产15%的上限扣减，公式基于各类 HQLA 的"还原后"数量。

## 报表结构

### 第Ⅰ部分：流动性覆盖率

#### 分子：合格优质流动性资产（HQLA）

| 项目 | 含义 | 折算率/约束 |
|------|------|-------------|
| [1.1] 一级资产 | 可在压力情景下无损失或极小损失快速变现的资产 | 100% |
| [1.1.1] 现金 | 未被抵押或用于其他使用安排的库存和运送中现金（不含库存金条） | 100% |
| [1.1.2] 压力条件下可动用的央行准备金 | 超额准备金 + 压力下存款流失导致的法定准备金下降部分 | 100% |
| [1.1.3] 风险权重为零的证券 | 可流通债券市值 | 100% |
| [1.1.3.1-4] 主权/央行/多边机构发行或担保 | 分主权国家发行、主权担保、央行发行或担保、BIS/IMF/多边开发银行 | 100% |
| [1.1.4] 母国或流动性风险所在国本币债券 | 主权风险权重不为0%时的本币债券 | 100%（有限制） |
| [1.1.5] 母国或流动性风险所在国外币债券 | 仅限压力情景下对应外币的现金净流出范围内 | 100%（有限制） |
| [1.2] 二级资产 | 当前市场价值基础上打折计入 | 上限40% |
| [1.2.1-2] 2A资产（公司债/担保债券） | 评级AA-及以上的非金融公司债和担保债券 | 85% |
| [1.2.3] 2A资产（风险权重20%的证券） | 主权/主权担保/央行/公共部门实体/多边开发银行发行或担保 | 85% |
| [1.2.4] 2B资产（公司债券） | 评级BBB-至A+的非金融公司债，含不符合1.1.4/1.1.5定义的主权和央行债券 | 50%，上限15% |

#### 分母：净现金流出

```
净现金流出 = 现金流出 - MIN(现金流入，现金流出 × 75%)
```

##### 现金流出

| 项目 | 含义 |
|------|------|
| [2.1.1] 零售存款流出 | 分稳定（满足/不满足有效存款保险附加标准）和欠稳定（有/无存款保险） |
| [2.1.2] 无担保批发现金流出 | 分小企业、大中型企业（含业务关系/非业务关系）、主权/央行/公共部门/多边开发银行、金融机构（含业务关系）及其他类别 |
| [2.1.3] 担保融资流出 | 与央行和其他交易对手的担保融资，按押品类型（一级/2A/2B/其他）分类 |
| [2.1.4] 其他项目 | 衍生产品净现金流、降级触发条款、非一级资产押品估值变化、市值变动、超额押品收回、交付义务、HQLA押品替换、结构性融资工具、ABCP/管道工具、未提取信用便利、30日内放款义务等 |
| [2.1.5] 或有融资义务 | 无条件可撤销便利、保函、信用证、贸易融资工具、非契约性义务（含理财）、附属交易商债券、客户空头头寸等 |
| [2.1.6] 其他现金流出 | 利息、股息、无担保抵押品借入等（不含运营成本） |

##### 现金流入

| 项目 | 含义 |
|------|------|
| [2.2.1] 逆回购与证券借入 | 分买断式（押品未用于再抵押）、质押式（押品未用于再抵押）、押品用于再抵押 |
| [2.2.2] 正常履约协议性现金流入 | 分零售/小企业/大中型企业/主权实体/央行/金融机构（含业务关系）和到期证券投资 |
| [2.2.3] 其他现金流入 | 衍生产品净现金流入、其他契约性现金流入 |

#### 附注项目

| 项目 | 含义 |
|------|------|
| [1.押品互换还原] | 30日内到期非现金资产互换对 HQLA 的影响（收回押品增加 A 列 / 返还押品减少 B 列） |
| [2.担保融资和押品互换还原] | 自动计算二级资产40%和2B资产15%上限扣减，不需银行填报 |

### 汇总计算公式

```
Ⅱ_1.1A = 一级资产合计（[1.1.1C]+[1.1.2C]+[1.1.3.1C]~[1.1.5C]）
Ⅱ_1.2A = 2A资产合计（[1.2.1C]+[1.2.2C]+[1.2.3.1C]~[1.2.3.5C]）
Ⅱ_1.3A = 2B资产合计（[1.2.4C]）
Ⅱ_1.A = Ⅱ_1.1A + Ⅱ_1.2A + Ⅱ_1.3A - 二级资产调整项 - 2B资产调整项
Ⅱ_2.1A = 现金流出合计
Ⅱ_2.2A = 现金流入合计
Ⅱ_2.A = Ⅱ_2.1A - MIN(Ⅱ_2.2A, Ⅱ_2.1A × 0.75)
Ⅱ_3.A = LCR = Ⅱ_1.A / Ⅱ_2.A
```

### 二级资产上限计算（附注项目[2]）

```
2B资产调整项 = Max{调整后2B资产 - 15/85×(调整后一级资产+调整后2A资产), 调整后2B资产 - 15/60×调整后一级资产, 0}
二级资产调整项 = Max{调整后2A资产+调整后2B资产 - 2B资产调整项 - 2/3×调整后一级资产, 0}
```

## 关键概念与口径规则

### 合格优质流动性资产（HQLA）基本特征

- 必须在压力情景下能够通过出售或抵（质）押方式快速变现。
- 估值可确定且稳定，风险敞口独立于发行人。
- 未被作为抵质押物或信用增级工具（无变现障碍）。
- 不在填报机构的资产负债表内作为担保融资的抵押品。
- 最好是可向央行抵押融资的资产，但向央行可抵押融资≠必然属于 HQLA。

### 流动性风险所在国判定

- 法人口径：填报机构在该国设有分支机构的东道国。
- 并表口径：填报机构在该国设有分支机构或附属公司的东道国（附属公司范围与本表并表范围一致）。
- 若银行在某国无经营实体、只有以该国货币计价的流动性风险暴露，则该国不属于"流动性风险所在国"。

### 中小企业客户定义

- 存款总额（并表口径）不超过800万元并被视同零售存款管理。
- 如对该客户存在风险暴露，还应当满足银保监会资本监管规定中的小企业客户定义。

### 业务关系存款认定四条件

1. 客户在未来30天内对银行的清算、托管和现金管理服务存在实质性依赖（无充足备份安排）。
2. 银行与客户之间签订了具有法律效力的清算、托管和现金管理服务合同。
3. 客户终止服务合同的提前通知期至少为30天，或转移存款的成本较高。
4. 业务关系存款存放于专门账户，账户收益不足以吸引客户存放超出业务需要的多余资金。

### 衍生产品净现金流计算三步法

1. **不同交易对手**：按实际现金流分别计算，不得轧差。
2. **同一交易对手**：判断是否存在有效净额结算主协议，存在则轧差。
3. **例外**：涉及本金互换的外汇衍生产品交易，若现金流同时完成，即使无净额结算主协议也可按轧差后净额计算。

### 存款保险制度处理（中国特殊规则）

- 我国存款保险制度属于"有效存款保险计划"，但**不满足**"有效存款保险附加标准"。
- 同业存款明确不在存款保险覆盖范围内。
- 零售和小企业存款：50万以内部分可在有存款保险项目中反映，超出部分填入无存款保险项目。
- 非业务关系存款：全额50万以内可在有存款保险项目中反映；全额超出则全部填入无存款保险项目。
- 同一客户有业务关系和无业务关系存款同时存在时，可自行选择：（1）存款保险额度先覆盖有业务关系；（2）按两种存款比例拆分。

## 特定项目处理规则

### 理财产品

| 类型 | 期限 | 处理方式 |
|------|------|----------|
| 购买的保本类理财 | ≥30天 | 不纳入计算 |
| 购买的保本类理财 | <30天 | 确定性现金流入计入"2.2.2.6.3金融机构-其他借款和现金流入" |
| 购买的非保本理财 | <30天 | 按最终投向确定部分计入对应流入栏目；无法拆分或回报不确定时不可算作现金流入 |
| 发行的保本类理财 | ≥30天且不可提前支取 | 不纳入计算 |
| 发行的保本类理财 | <30天或可提前支取 | 本金参照存款计入"无担保批发资金流出"（一般为非业务关系存款），客户端收益计入"2.1.6其他现金流出" |
| 发行的非保本理财 | <30天或可提前支取 | 计入"2.1.5.5非契约性义务"→"2.1.5.5.1属于理财产品的部分" |

### 大额存单

| 可转让性 | 投资人类别 | 填报位置 |
|----------|-----------|----------|
| 不可转让 | 个人 | 零售 |
| 不可转让 | 小企业 | 小企业 |
| 不可转让 | 机构 | 大中型企业 |
| 个人与机构间不可转让 | 个人投资人发行部分 | 零售 |
| 个人与机构间不可转让 | 机构投资人发行部分 | 大中型企业 |
| 可在个人和机构间转让 | 全部 | 大中型企业（认定为非业务关系存款） |

### 空头头寸处理

| 覆盖方式 | 现金流入系数 |
|----------|-------------|
| 30天内逆回购/证券借入覆盖的超过30天空头头寸 | 0%（假定展期） |
| 无担保证券借入覆盖的银行空头头寸 | 100%（假定抵押品完全流失） |
| 担保证券融资交易覆盖的银行空头头寸 | 0%（假定未来30天一直持有） |

### 远期交易处理

| 交易类型 | LCR 影响 |
|----------|---------|
| 未来30天内生效并到期的远期回购/逆回购/抵押品互换 | 无影响，可忽略 |
| 生效日早于报告日且到期日在未来30天后的远期交易 | 无影响，可忽略 |
| 合格优质流动性资产的远期销售和购买 | 无影响，可忽略 |
| 未来30天内生效但30天后到期的远期逆回购 | 现金流出计入"其他现金流出"，扣除按折扣系数调整后的押品市值 |
| 未来30天内生效但30天后到期的远期回购 | 现金流入计入"其他现金流入"，扣除按折扣系数调整后的押品市值 |

### 财政性存款

- 按缴存中央银行财政性存款与财政性存款轧差后的净额填报。
- 轧差后为资产方：已反映在超额准备金中，不需单独计量。
- 轧差后为负债方：按净额填入"2.1.2.5未包含在以上无担保批发现金流出分类的其他类别"。
