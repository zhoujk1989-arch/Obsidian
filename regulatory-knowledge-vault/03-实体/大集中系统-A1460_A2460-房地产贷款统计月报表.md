# 大集中系统-A1460_A2460-房地产贷款统计月报表

> [!note] 核心定位
> A1460（人民币）和 A2460（外币）房地产贷款统计月报表是大集中系统中的核心月度信贷监控报表。该报表全面监测和上报金融机构发生的房地产开发贷款（包含土地储备、保障性住房、保障性租赁住房、商业用房等开发贷款）、购房贷款（包含企业、机关团体及个人购房，以及保障性住房收购和住房租赁团体购房）以及证券化的房地产贷款。它是中国人民银行制定房地产信贷与货币政策、评估住房市场“去库存”融资投放进度以及防范系统性金融风险的核心基础数据源。

## 基本信息

- **报表编码**：A1460（人民币业务） / A2460（外币业务折人民币）
- **报表名称**：房地产贷款统计月报表
- **系统归属**：大集中系统
- **报送频度**：月报
- **原文依据**：[[01-资料库/大集中系统/2026-05-18-A1460_A2460-房地产贷款统计月报表-原文|A1460、A2460 房地产贷款统计月报表原文]]

## 指标体系与层级结构

本报表指标体系中，A1460 对应以 `12` 开头的指标编号，A2460 对应以 `22` 开头的指标编号。以下以 A1460（12A系列）为例，完整展示房地产贷款月报的树状逻辑结构：

* **一、房地产开发贷款 (12A09 / 22A01)**：用于土地开发、房屋建设的贷款。
  * **1. 地产开发贷款 (12A10 / 22A02)**：专门用于地产开发、且完成后计划收回的贷款。
    * *其中：政府土地储备机构贷款 (12A11 / 22A03)*：存量土地储备贷款。
  * **2. 房产开发贷款 (12A12 / 22A04)**：金融机构发放的各类房屋建设贷款。
    * **（1）住房开发贷款 (12A13 / 22A05)**：用于住房建设的贷款（含土地开发阶段发放、计划在住宅阶段继续使用的贷款）。
      * *其中：保障性住房开发贷款 (12A14 / 22A06)*：政策性、保障性住宅项目开发贷款。
        * *其中：保障性租赁住房开发贷款 (12A67 / 22A67)*：自持性保障租赁住房开发贷款。
    * **（2）商业用房开发贷款 (12A16 / 22A08)**：商业房屋（包括商住两用房）的建设贷款。
    * **（3）其他房产开发贷款 (12A17 / 22A09)**：除去住房和商业用房开发之外的房产开发贷款。
* **二、购房贷款 (12A18 / 22A10)**：用于收购或长期租赁房屋的贷款。
  * **1. 企业购房贷款 (12A19 / 22A11)**：支持非金融企业收购或长期租赁房屋的贷款。
    * **（1）商业用房贷款 (12A20 / 22A12)**：企业购买或长租商业用房的贷款（含商住两用房）。
    * **（2）住房贷款 (12A21 / 22A13)**：企业购买或长租住房的贷款。
      * *其中：保障性住房收购贷款 (12A2A / 22A2A)*：用于收购已建成未出售商品房用作配售型保障性住房的贷款（银发〔2024〕135号）。
      * *其中：住房租赁团体购房贷款 (12A2B / 22A2B)*：批量购买存量闲置房屋用作租赁住房长期持有运营的贷款（银发〔2024〕2号）。
  * **2. 机关团体购房贷款 (12A80 / 22A80)**：机关团体、事业单位收购或长租房屋的贷款。
    * **（1）商业用房贷款 (12A81 / 22A81)**：机关团体长租或购买商业用房。
    * **（2）住房贷款 (12A82 / 22A82)**：机关团体长租或购买住房。
  * **3. 个人购房贷款 (12A22 / 22A14)**：发放给自然人用于购买房屋的信贷。
    * **（1）个人商业用房贷款 (12A23 / 22A15)**：个人购买商业用房（含商住两用房）贷款。
    * **（2）个人住房贷款 (12A24 / 22A16)**：个人购买住房贷款（按新旧房及担保划分）：
      * **1）新建房贷款 (12A25 / 22A17)**：购买第一次出售住房的贷款。
        * *其中：抵押贷款 (12A26 / 22A18)*：已办理抵押登记的个人新建房贷款。
      * **2）再交易房贷款 (12A27 / 22A19)**：购买二手房（非第一次出售）的贷款。
* **三、证券化的房地产贷款 (12A28 / 22A20)**：金融机构已证券化的房地产贷款的未清偿本金。
  * **1. 证券化个人住房贷款 (12A29 / 22A21)**
  * **2. 其他证券化房地产贷款 (12A30 / 22A22)**
* **附注指标 (12A/22A附)**：
  * **个人购买保障性住房贷款 (12A31 / 22A23)**：个人购买政策性保障住房贷款余额。
  * **企业收购、租赁保障性住房贷款 (12A83 / 22A83)**：企业收购/长租房屋作为廉租房/公租房的贷款。
  * **机关团体收购、租赁保障性住房贷款 (12A84 / 22A84)**：机关团体收购/长租房屋作为廉租房/公租房的贷款。
  * **房地产并购贷款 (12A85 / 22A85)**：房地产开发项目或股权并购活动中向并购方发放的贷款。

## 核心概念与统计边界

### 1. 房地产开发贷款大类
* **房地产定义**：土地、建筑物及不可分割部分，且这些土地和建筑物是**为了进入房地产交易市场而开发**。经营实体自建自用的厂房、酒店、仓储以及医院、学校等公益性建筑均**排除在外**。
* **地产开发 vs 房产开发**：
  * **地产开发贷款 (12A10/22A02)**：土地开发（含土地储备贷款、保障性安居工程中扣除住房开发外的土地开发整理贷款、以土地出让收入为还款来源的贷款）。
  * **房产开发贷款 (12A12/22A04)**：房屋建设阶段发放的贷款（含土地开发阶段发放、但计划在建房阶段继续使用的贷款）。
* **保障性住房开发贷款 (12A14/22A06)**：支持廉租房、公租房、经济适用房、限价商品房、棚户区改造、老旧小区改造安置房的项目建设。**排除农村危房改造**与游牧民定居项目。
* **保障性租赁住房开发贷款 (12A67/22A67)**：必须是取得项目**保障性租赁住房项目认定书**的自持类主体项目贷款。

### 2. 购房与“去库存”再贷款政策支持
* **保障性住房收购贷款 (12A2A/22A2A)**：严格限定为贷款银行向市县政府选定的地方国有企业发放的，用于**收购房地产企业已建成未出售商品房**用作配售型保障性住房的贷款。
* **住房租赁团体购房贷款 (12A2B/22A2B)**：用于支持批量购买存量闲置房屋用作租赁住房长期持有运营的贷款。

### 3. 个人新建房与二手房判定
* **新建房贷款 (12A25/22A17)**：购买第一次出售的住房。
* **再交易房贷款 (12A27/22A19)**：购买非第一次出售的住房（二手房）。

## 核心填报与穿透规则

### （一）混合支持项目的“优先全额填报”规则
对于同一个贷款项目，若既有商品房开发又有保障性住房开发：
1. 应尽量按实际支持对象划分并分别填报。
2. **若业务部门确认确实无法划分，该笔贷款余额应全额填入“保障性住房开发贷款”**。购房贷款的划分判定也完全参照此处理原则。

### （二）经营性物业抵押贷款“穿透投向”规则
金融机构以拥有的物业作抵押、以物业经营收入为还本付息来源发放的“经营性物业抵押贷款”，**必须按实际投向进行穿透分类**：
* **支持开发/物业改造**：计入“一、房地产开发贷款”（12A09/22A01）之下的相应指标。
* **用于置换股东预付款**：计入“企业购房贷款”项下的“（1）商业用房贷款”（12A20/22A12）。
* **支持其他购房需求**：计入“二、购房贷款”（12A18/22A10）的相应购房指标。
* **未用于房地产开发或购房（如用于一般流动资金周转）**：**一律不应纳入房地产贷款统计口径**。

### （三）阶段性保证加抵押贷款的“即时认定”规则
在个人住房贷款当月发放、偿还统计中，金融机构发生的“阶段性保证加抵押”个人住房贷款业务，**在批贷时就应当认定为抵押贷款**，并填报入 12A26/22A18（新建房抵押贷款）中。

## 重要历史修订记录

### 1. 2023 年修订：增设“房地产并购贷款”附报指标
* **修订内容**：为了进一步完善房地产贷款专项统计监测的维度，增设“房地产并购贷款”（12A85/22A85）附报指标。
* **业务内涵**：统计在房地产项目并购或房地产企业股权并购活动中，向并购方发放的用于支付并购交易价款和费用的贷款。

### 2. 2024 年修订：增设保障性住房收购与住房租赁团体购房指标
* **修订背景**：落实《中国人民银行国家金融监督管理总局关于金融支持住房租赁市场发展的意见》（银发〔2024〕2号）、《中国人民银行国家金融监督管理总局关于实施保障性住房再贷款的通知》（银发〔2024〕135号）的政策部署。
* **修订内容**：在大集中系统房地产月报表（A1460/A2460）中，在“企业购房贷款 - 住房贷款”之下，正式增设 **保障性住房收购贷款 (12A2A/22A2A)** 和 **住房租赁团体购房贷款 (12A2B/22A2B)** 统计指标。
* **业务内涵**：反映对已建成存量商品房“去库存”的贷款支持，强化住房保障与租赁体系配套融资监测。
