# 一表通-9.2投融资标的

## 简介

9.2投融资标的是一表通投资信息域中的投资穿透层核心报表，41个字段，1个主键（投融资标的ID），13个强制性校验字段。填报机构在投融资、同业交易等自营业务中涉及的投融资标的的完整信息，包括表层、中间层、底层穿透后的标的。

## 关键信息

- 所属系统：一表通系统
- 报表编码：9.2
- 报表名称：投融资标的
- 当前原文依据：[[01-资料库/一表通系统/2026-04-28-9.2投融资标的-填报说明-原文|2026-04-28-9.2投融资标的-填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

本表是一表通投资信息域中负责采集投融资标的详细信息的穿透层报表，与 9.1 投资标的关系 配合使用：9.1 维护标的之间的层级关系，9.2 维护每个标的的详细信息。本表覆盖自营资金业务的全部投融资标的，包括表层、中间层和底层穿透后的标的。买入返售/卖出回购业务也需填报本表（仅表层），底层押品在 9.3 抵质押品 表填报。本表包含标的范围内的抵债资产。

## 重点口径

1. **层层穿透填报**：投资标的要层层穿透、逐层填报，即包括表层、中间层、底层的投资标的。与 9.1 投资标的关系 配合使用。
2. **穿透后基础资产填报范围**：同业存单、债券、非标准化债权和同业投资下的所有业务穿透后的基础资产需填报"基础资产客户名称"至"基础资产最终投向行业"区间字段；其他投融资标的如能获得相关基础资产信息则也填报，不能获取可以允许为空。
3. **买入返售/卖出回购**：需填报本表，但仅填报表层业务情况，底层押品信息在《9.3抵质押品》表填报。
4. **抵债资产**：本表包含标的范围内的抵债资产。
5. **报数期边界**：报数期开始前该标的物已经不在任何一个协议或者交易相关的，不填报。
6. **拆放/存放同业**：以对某机构为单一金融工具，如"存放××银行"。
7. **基础资产客户行业/投向行业**：境外机构一表通填报"9999"，EAST填报"境外"。
8. **含权类型**：存在多种类型时，使用英文半角分号隔开。
9. **担保协议ID**：同一笔投资业务存在多笔担保协议时，用半角分号隔开共同填报。
10. **市场化债转股/产业基金**：与 G3102 填报规则一致。

## 重点规则

| 规则 | 说明 |
| --- | --- |
| 标的ID生成 | 中登公司登记的金融产品填报中登公司编号；理财登记机构登记的填报理财标识码；底层资产填报理财中心赋予的序号；非标产品银行自定义 |
| 机构ID生成 | 前11位与金融许可证号前11位相同，后13位使用内部机构号，不足部分不用补齐 |
| 利率/收益率 | 百分比为单位，即1/100，保留六位小数。如4.5%则填报为4.5 |
| 币种 | 同时存在多个币种的，以英文半角分号分隔填报 |
| 备注 | 不同备注事项用英文半角分号隔开 |
