# 一表通-8.4信用卡账户状态

## 业务定位

采集填报机构所有信用卡当前状态。信用卡在报数期前已经销户的，不填。属于信用卡信息域，覆盖信用卡账户维度的额度、交易、逾期、分类、状态等全量快照信息。

## 关键字段分组

### 标识字段
- 客户ID（H040001）：机构内唯一客户编码，≤60位数字字母组合
- 机构ID（H040043）：2.1版新增，客户归属银行机构编码
- 信用卡账号（H040003）：银行机构分户账账号
- 核算机构ID（H040004）：信用卡核算所在机构ID
- 币种（H040013）：参照 GB/T 12406-2022，三位数字字母编码
- 采集日期（H040036）：数据采集日期

### 额度字段（本外币拆分）
- 当前本币授信额度（H040005）：含临时额度
- 当前外币授信额度（H040006）：与币种对应金额
- 已使用本币授信额度（H040007）
- 已使用外币授信额度（H040008）
- 其中本币临时额度（H040016）
- 其中外币临时额度（H040017）
- 本币现金支取额度（H040044）：2.1版新增
- 外币现金支取额度（H040045）：2.1版新增
- 本币授信余额（H040047）：2.1版新增，本币可用额度
- 外币授信余额（H040048）：2.1版新增，外币可用额度

### 余额/金额字段
- 免息应收账款（H040009）：免息期应收账款余额，不含手续费和利息
- 应收息费（H040010）：含滞纳金、罚息及利息
- 账户余额（H040011）：含利息和费用，溢缴款时为负
- 分期余额（H040039）：2.0版新增，当前分期业务本金余额
- 溢缴款余额（H040040）：2.0版新增，多缴资金余额
- 逾期金额（H040012）：逾期未还本金之和，区分逾期90天及以上和90天以下
- 透支金额（H040038）：尚未偿还的本金，不含利息
- 冻结金额（H040018）：预授权冻结金额

### 交易字段（自然月累计，非账单月）
- 当月累计交易笔数（H040019）
- 当月累计透支金额（H040020）
- 本月累计消费金额（H040021）
- 本月累计取现转账金额（H040022）
- 本月累计分期交易金额（H040023）：取值范围有差异，一表通允许正值/负值/0，EAST要求非负值
- 本月累计收入（H040024）
- 当年累计信用卡收入金额（H040025）：含利息收入、手续费收入

### 征信/他行字段
- 已有信用卡发卡银行数（H040026）：最近一次人行征信查询时
- 已有他行授信金额（H040027）：最近一次人行征信查询时

### 日期字段
- 逾期起始日期（H040031）：最早一期逾期时间，未逾期默认 9999-12-31
- 最近授信评估日期（H040032）
- 最近征信查询日期（H040033）：默认 9999-12-31
- 最近新增授信日期（H040034）
- 最后交易日期（H040035）
- 账户失效日期（H040046）：2.1版新增，未失效默认 9999-12-31

### 分类/状态/催收字段
- 五级分类（H040015）：01正常/02关注/03次级/04可疑/05损失/00未分级
- 账户状态（H040037）：01正常/02预销户/03销户/04冻结/05止付/[00-XX]其他-银行自定义
- 催收标识（H040028）：0否/1是
- 催收方式（H040029）：01电话/02信函/03外访/04司法/05委外/[00-XX]其他-银行自定义
- 新增授信类型（H040030）：01新发卡授信/02固定额度上调/03专项分期额度上调/[00-XX]其他-银行自定义
- 逾期期数（H040041）：2.1版新增，脱期法记录逾期情况

### 备注字段
- 备注（H040042）：2.1版新增，特殊说明

## 码值

### 五级分类
| 码值 | 含义 |
|------|------|
| 01 | 正常：债务人能够履行合同，没有客观证据表明本金、利息或收益不能按时足额偿付 |
| 02 | 关注：存在可能产生不利影响的因素，但债务人目前有能力偿付 |
| 03 | 次级：债务人无法足额偿付本金、利息或收益，或已发生信用减值 |
| 04 | 可疑：债务人已无法足额偿付，金融资产已发生显著信用减值 |
| 05 | 损失：采取所有可能措施后只能收回极少部分或损失全部 |
| 00 | 未分级：未进行五级分类的金融资产 |

## 重点口径

1. **销户排除**：信用卡在报数期前已经销户的，不填。
2. **本外币拆分**：授信额度、已使用额度、临时额度等字段按本外币币种拆分填报。
3. **账户余额正负**：溢缴款时账户余额为负，其他时候为正数或0。
4. **逾期金额参考 G0102**：区分逾期90天及以上和90天以下的不同取值方式。
5. **自然月口径**：交易字段按自然月（1-31日）累计，非账单月累计，不计算客户还款扣减。
6. **临时额度定义**：固定额度以外的额度，包括临时性取现或消费额度和授信时有效期确定的专项分期额度；即使客户未实际启用，一旦获知存在则有效期内应计算。
7. **日期默认值**：逾期起始日期、最近征信查询日期、账户失效日期未发生对应事件时默认 9999-12-31。
8. **分期交易金额差异**：一表通可根据交易方向区分允许正值/负值/0，EAST 要求合约金额为非负值。

## 版本变化

### 2.1 版
- **新增字段**：机构ID（H040043）、本币现金支取额度（H040044）、外币现金支取额度（H040045）、本币授信余额（H040047）、外币授信余额（H040048）、账户失效日期（H040046）、备注（H040042）
- **修改字段**：
  - 当前本币授信额度：明确按本外币币种拆分填报
  - 当前外币授信额度：修改报送规则，原为按填报时汇率转换后填报
  - 已使用外币授信额度：修改报送规则
  - 逾期金额：补充报送规则，原为"逾期未还本金之和"
  - 其中外币临时额度：修改报送规则
  - 催收方式：码值新增[00-XX]其他-银行自定义，格式代码→文本，长度an→anc..60
  - 新增授信类型：码值新增[00-XX]其他-银行自定义，格式代码→文本，长度2!n→anc..60
  - 账户状态：码值新增[00-XX]其他-银行自定义，格式代码→文本，长度2!n→anc..30
- **逾期期数**：2.1版新增，采用脱期法记录逾期情况

### 2.0 版
- **新增字段**：分期余额（H040039）、溢缴款余额（H040040）

### 1.0 版
- 初始版本，40个字段

## 一表通与 EAST 差异

1. **日期格式**：一表通为 YYYY-MM-DD（10位），EAST 为 YYYYMMDD（8位），需转换
2. **五级分类码值**：一表通为数字码值，EAST 为文本码值，需转换
3. **账户状态码值**：一表通为数字码值，EAST 为文本码值，需转换
4. **催收标识码值**：一表通为数字码值，EAST 为文本码值，需转换
5. **催收方式码值**：一表通比 EAST 码值分类更细，需码值映射
6. **新增授信类型码值**：一表通为数字码值，EAST 为文本码值，需转换
