# 一表通-6.5银团贷款协议

## 简介

采集报送机构与合作方签订的银团贷款合作协议。银团贷款是指由两家或两家以上银行基于相同贷款条件，依据同一贷款合同，按约定时间和比例，通过代理行向借款人提供的本外币贷款或授信业务。

## 关键字段

| 数据项编码 | 数据项名称 | 说明 |
|------------|------------|------|
| F050001 | 协议ID | PK，强制性校验，长度不超过60位数字与字母组合 |
| F050016 | 机构ID | PK，强制性校验，拼接金融许可证号前11位+内部机构号 |
| F050002 | 牵头行行名 | 强制性校验，应与公章名称一致 |
| F050003 | 牵头行行号 | 强制性校验，12位支付行号或SWIFT行号 |
| F050004 | 参加行行名 | 强制性校验，多个行名以英文半角分号隔开 |
| F050005 | 参加行行号 | 强制性校验，多个行号以英文半角分号隔开 |
| F050006 | 代理行行名 | 强制性校验，应与公章名称一致 |
| F050007 | 代理行行号 | 强制性校验，12位支付行号或SWIFT行号 |
| F050008 | 银团成员类型 | 强制性校验，4 种码值（01-03+00-XX） |
| F050009 | 银团贷款总金额 | 强制性校验，协议总额度 |
| F050017 | 银团贷款总金额币种 | 2.0版新增，三位数字字母编码 |
| F050010 | 承担贷款金额 | 强制性校验，本行承担的银团贷款额度 |
| F050011 | 已发放银团贷款金额 | 所有相关银行发放的贷款金额合计 |
| F050012 | 已发放银团贷款余额 | 1.0版新增，贷款余额合计 |
| F050015 | 已发放承担银团贷款金额 | 2.0版新增，本行实际已发放金额 |
| F050013 | 备注 | 特殊说明，英文半角分号隔开 |
| F050014 | 采集日期 | PK，强制性校验，YYYY-MM-DD 格式 |

## 银团成员类型码值

| 码值 | 含义 |
|------|------|
| 01 | 牵头行 |
| 02 | 代理行 |
| 03 | 参加行 |
| 00-XX | 其他-银行自定义（XX 为银行自定义类型） |

> 如果本行同时担任多个角色（如牵头行+代理行），以英文半角分号隔开并列填列，如 `01;02`。
> 一表通为数字码值，EAST 为文本码值，需做码值转换。

## 填报规则

- 报送机构与合作方签订的银团贷款合作协议
- 多个参与行信息一并填报，使用英文半角分号隔开
- 银团成员类型可并列填列，以英文半角分号隔开
- 已发放银团贷款金额和余额若为参加行可允许为空
- 已发放承担银团贷款金额填报未结清贷款总额，不填报循环发放累计金额

## 版本差异

- **2.1 版延长**：F050001 协议ID、F050003 牵头行行号、F050007 代理行行号、F050008 银团成员类型、F050013 备注 数据格式长度
- **2.1 版码值变更**：F050008 银团成员类型，原码值"00-其他"修改为"00-XX 其他-银行自定义"
- **2.0 版新增**：F050016 机构ID（PK）、F050017 银团贷款总金额币种、F050015 已发放承担银团贷款金额

## 数据转换要点

- 日期格式：一表通为 `YYYY-MM-DD`（10 位），EAST 为 `YYYYMMDD`（8 位），需转换
- 银团成员类型：一表通为数字码值（01/02/03/00-XX），EAST 为文本码值，需映射
- 机构ID：一表通需拼接金融许可证号前 11 位 + 内部机构号
