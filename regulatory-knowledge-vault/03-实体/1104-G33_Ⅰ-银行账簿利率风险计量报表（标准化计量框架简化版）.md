# 1104-G33_Ⅰ-银行账簿利率风险计量报表（标准化计量框架简化版）

## 简介

G33_Ⅰ《银行账簿利率风险计量报表（标准化计量框架简化版）》是 1104 系统中的银行账簿利率风险监管报表，用于衡量银行账簿利率风险水平。与 G33_Ⅱ（完整版）相比，G33_Ⅰ 采用简化版标准化计量框架，不要求对行为性期权和嵌入式利率期权进行复杂建模。

## 与 G33_Ⅱ 的关系

G33_Ⅰ 和 G33_Ⅱ 同属银行账簿利率风险计量报表系列，但适用机构和框架复杂度不同：

| 维度 | G33_Ⅰ（简化版） | G33_Ⅱ（完整版） |
|------|----------------|----------------|
| 填报机构 | 政策性银行、城市商业银行、民营银行、农村商业银行、外资法人银行 | 大型商业银行（含邮储银行）、股份制商业银行 |
| 框架复杂度 | 简化版，不处理行为性期权和嵌入式利率期权 | 完整版，需建模行为性期权、嵌入式利率期权 |
| 提前还款/支取权 | 按合同约定现金流填报，不计算情景调整率 | 需计算基准和6种情景下的提前还款/支取率调整 |
| 无到期日存款（核心存款） | [2.2.1] 暂不填报 | 使用十年历史数据识别核心存款和非核心存款 |
| 衍生品处理 | 复式记录法，参见 G4C-1(b) | 复式记录法，参见 G4C-1(b) |
| 利率冲击情景 | 6种标准化情景（平行上移/下移、形状变化） | 6种标准化情景（同 G33_Ⅰ） |
| 报送频度 | 季报第二批次（季后18日） | 季报第二批次（季后18日） |

## 当前结论

- G33_Ⅰ 的核心是基于银行账簿利率敏感性资产与负债的重定价期限，计算特定利率冲击情景下利率波动对经济价值和一年以内净利息收入的影响。
- 本报表依据原银保监会《商业银行银行账簿利率风险管理指引（修订）》制定。
- 填报机构限定为政策性银行（含开发银行）、城市商业银行、民营银行、农村商业银行、外资法人银行，与 G33_Ⅱ 的填报机构形成互补覆盖。
- 报表采用简化版标准化计量框架，不要求对行为性期权（提前还款权/提前支取权）进行情景调整建模，按合同约定现金流填报。
- 报表按主要货币分别填报（占比超过5%的币种），外币折算为人民币后报送。
- 报表结构分为利率敏感性资产、利率敏感性负债、衍生金融产品（不含利率期权）、固定利率贷款承诺、利率期权五大板块，以及附注项目。
- 衍生品交易采用复式记录法：拆分为基础工具的多头和空头头寸分别填报，与 G33_Ⅱ 规则一致。
- [2.2.1] 核心存款暂不填报，无到期日存款全部填入 [2.2]。

## 重点口径

- **报送口径**：法人口径分币种（季报），季报第二批次，季后18日
- **填报机构**：政策性银行（含开发银行）、城市商业银行、民营银行、农村商业银行、外资法人银行
- **数据单位**：万元，保留两位小数
- **填报币种**：按主要货币分别报送（占比超过5%的币种），外币折算为人民币
- **汇率折算**：美元/欧元/日元/港币使用国家外汇管理局基准汇价；其他货币套算确定

## 时间区间结构

报表按重定价期限划分多个时间段（B列至T列），涵盖从隔夜到长期限的完整区间。k值为时间区间中值，无需填报。

## 重点规则

### 重定价日期确定规则

1. **固定利率产品**：本金偿还到期日 = 本金重定价日；利息支付日 = 利息重定价日。
2. **可变利率产品**：距报告日最近的利率调整日 = 本金重定价日。
3. **利率调整时间不确定**：利率调整日视为紧接报告日的工作日。
4. **逾期资产**（未转为非应计）：填报在【隔夜-1个月】内。

### 浮动利率判定规则

- 若浮动利率业务已达到浮动上限或下限，则视同为固定利率。
- 只有重新浮动后，才再作为浮动利率。

### 简化版与完整版的核心差异

- **行为性期权**：G33_Ⅰ 不计算情景调整后的提前还款率/提前支取率，按合同约定现金流直接填报至各时间区间。
- **嵌入式利率期权**：G33_Ⅰ 原文未提及债券投资/发行债券中嵌入式利率期权的剥离处理，与 G33_Ⅱ 不同。
- **无到期日存款**：G33_Ⅰ [2.2.1] 核心存款暂不填报，无到期日存款全部填入 [2.2]。

### 衍生品复式记录法

- **利率掉期**（收浮动、付固定）：浮动利率工具多头（期限=至下一个重定价日）+ 固定利率工具空头（期限=合约剩余期限）。
- **远期外汇**：两种货币的零息政府债券多头和空头（期限=合约期限）。
- **债券期货多头**：可交割债券多头 + 期货交割日到期的零息债券空头。
- **货币期权**：按 delta 等值填报，收到货币为多头，交付货币为空头。
- **非利率期权**：按 delta 等值（本金×delta 或债务工具市值×delta）采用复式记录法填报。
