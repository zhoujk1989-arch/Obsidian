# 一表通-7.4信用卡交易

## 基本信息

|| 项目 | 内容 |
| --- | --- |
| 报表编码 | 7.4 |
| 报表名称 | 信用卡交易 |
| 数据域 | 交易信息 |
| 字段总数 | 37 |
| 版本 | 1.0 版 + 2.1 版新增字段 |

## 业务描述

填报机构所有信用卡的交易，包括预借现金、还款、转账交易、消费类交易的明细。已核销卡的交易明细不再报送。

| G040003 | 分户账号 | 编码类 | 是 | 最细一级账号 |

## 2026 年制度升级变更

- **交易类型（G040009）**：数据类别由代码类改为文本类
- **交易渠道（G040030）**：数据类别由代码类改为文本类
- **分期业务ID（G040025）**：数据格式长度由60改为100
- **IP地址（G040026）**：数据格式长度由39改为40
- **新增字段（2.1版）**：提前结清标志、最迟还款日期、交易账单日期、备注

## EAST 映射要点

- 交易类型：EAST 填报中文码值，一表通填报数字码值，需转换
- 借贷标识：EAST 填报中文码值，一表通填报数字码值，需转换
- 线上线下交易标识：EAST 填报中文码值，一表通填报数字码值，需转换
- 交易渠道：EAST 填报中文码值，一表通填报数字码值，需转换
- 交易金额：EAST 要求非负值，一表通允许正值、负值和0
