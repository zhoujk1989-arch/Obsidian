# 一表通-6.8担保协议

## 简介

采集填报机构签订的所有担保协议（包括表内外业务），覆盖接受担保和对外出具担保两种方向。

## 关键信息

- 所属系统：一表通系统
- 报表编码：6.8
- 报表名称：担保协议
- 字段总数：29
- 当前原文依据：[[01-资料库/一表通系统/2026-04-28-6.8担保协议-填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

1. **覆盖范围**：填报机构签订的所有担保协议（包括表内外业务），包括接受担保和对外出具担保。多个担保人共用一张协议时采多条记录，混合担保时采多种记录。
2. **排除条件**：报数期开始前已失效且被担保主合同已销户终结的担保协议无需填报。
3. **抵质押关联**：当担保类型为抵质押时，担保人即为《抵质押品》表中抵质押物所有权人。
4. **保证金担保**：保证金形式的担保需要在本表报送。
5. **版本演进**：2.0 新增担保人类型和担保能力上限字段；2.1 新增担保人客户ID、担保人分类，并延长部分字段长度。

## 重点口径

### 担保类型分类（F080004）

| 码值 | 名称 | 说明 |
|------|------|------|
| 01 | 抵押 | - |
| 02 | 质押 | - |
| 03 | 单人保证 | 一个或多个保证人分别签订一份保证合同，每份确定个人应承担的实际担保金额 |
| 04 | 多人保证 | 多个保证人分别签订独立保证合同，每个承担相同的总担保金额 |
| 05 | 多人联保 | 多人共同签订一份保证合同，每个承担相同的总担保金额 |
| 06 | 多人分保 | 多人共同签订一份保证合同，每人承担规定的实际担保金额 |
| 07 | 混合 | 混合担保 |

### 担保合同方向（F080005）

| 码值 | 名称 | 说明 |
|------|------|------|
| 01 | 接受担保 | 填报机构接受其他组织或个人提供的担保 |
| 02 | 出具担保 | 填报机构对外出具担保，不包含保函业务 |

### 担保合同类型（F080007）

## 重点规则

### EAST 差异

1. **机构ID拼接**：F080002 一表通需拼接金融许可证号前11位 + 内部机构号后13位，共24位。
2. **码值差异**：担保类型、被担保业务类型、担保合同类型、担保人类别、担保人证件类型、协议状态等字段的码值与 EAST 不一致，一表通码值分类更细或采用数字码值。
3. **日期格式**：一表通为 `YYYY-MM-DD`（10位），EAST 为 `YYYYMMDD`（8位），需格式转换。
4. **新增字段**：2.0 版新增担保人类型（F080026）、担保人担保能力上限（F080027）；2.1 版新增担保人客户ID（F080028）、担保人分类（F080029）。

### 码值映射要求

以下字段需进行码值映射（一表通为数字码值，EAST 为文本码值）：
- F080004 担保类型
- F080006 被担保业务类型
- F080007 担保合同类型
- F080008 担保人类别
- F080010 担保人证件类型
- F080019 协议状态
