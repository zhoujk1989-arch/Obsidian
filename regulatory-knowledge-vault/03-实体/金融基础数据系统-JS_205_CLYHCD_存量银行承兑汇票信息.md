## 业务说明

存量银行承兑汇票指金融机构报告期末存续的，由在本行开立存款账户的存款人签发，并经本行审查同意承兑的票据。可分包票据以票据（包）的子票区间为单位逐笔报送。不可分包票据以单张票据为单位逐笔报送。

## 重点口径

1. **报送时点**：报告期末存续数据，非报告期内发生额。
2. **报送粒度**：可分包票据以票据（包）的子票区间为单位逐笔报送，不可分包票据以单张票据为单位逐笔报送。
3. **票据编号格式**：可分包票据填报格式为"票据（包）号码,子票区间起始编号,子票区间终止编号"，用英文逗号分隔。
4. **折人民币必填**：票面金额折人民币和手续费金额折人民币均为必填项，外币按金融机构会计核算使用汇率折算。
5. **主体识别**：出票人和收款人均需完整报送身份标识（名称+证件类型+证件代码），证件类型必须填写最底层类别。

## 重点规则

1. 金融机构代码优先统一社会信用代码，无则组织机构代码并附说明。
2. 出票人/收款人地区代码：境外客户前3位用"000"填充，后3位用GB/T 2659国别码，港澳台按境外客户处理，不得填报810000/820000等行政区划代码。
3. 出票人经济成分：允许回溯一级（如不能确定A0101/A0102可填A01），但不可填更高层级（不可填A）。
4. 出票人企业规模：境内非企业类单位应填报"CS05"，不得为空，也不得归入其他企业规模类型。
5. 担保方式：含房地产抵押的组合担保统一填报"E01"；不含房地产抵押的组合担保统一填报"E"。

## 与系统内其他票据报表的关系

- 本表（JS_205_CLYHCD）属于票据业务大类下的银行承兑汇票细类，采集报告期末存续的银行承兑汇票存量数据。
- 与 JS_205_CLPJRZ（存量票据融资信息）互补：CLYHCD 聚焦银行承兑汇票（存款人签发、银行承兑），CLPJRZ 聚焦贴现及买断式转贴现业务。
- 与 JS_205_PJRZFS（票据融资发生额信息）互补：CLYHCD 采集存量余额，PJRZFS 采集报告期内发生/收回流量。
- 与 JS_205_CLZTX（存量再贴现信息）互补：CLYHCD 为市场端银行承兑汇票存量，CLZTX 为央行端再贴现存量。
- 与 JS_205_ZTXFS（再贴现发生额信息）互补：CLYHCD 为市场存量，ZTXFS 为央行端发生额流量。
- 金融基础数据系统票据业务覆盖扩展至四大细类：票据融资（CLPJRZ存量+PJRZFS流量）+ 再贴现（CLZTX存量+ZTXFS流量）+ 银行承兑汇票（CLYHCD存量）。
