# 1104-G03 各项资产减值损失准备情况表

## 简介

G03 是 1104 系统中专门反映各项资产减值损失准备提取、冲销、转回等变动情况及各项准备金保有情况的报表，是减值准备全貌的核心报表。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银监统0004号
- 报表名称：各项资产减值损失准备情况表
- 当前原文依据：[[01-资料库/1104系统/2026-04-22-G03-各项资产减值损失准备情况表-填报说明-原文|原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- G03 是 1104 系统中专门反映资产减值损失准备变动全貌的报表，覆盖贷款损失准备和其他资产减值准备两大主线。
- 月报（境内分支机构汇总，月后6日）+ 季报（法人汇总，季后13日；合并报表，季后40日），是唯一同时包含月报和季报双重频度的减值类报表。
- 填报机构范围极广，几乎覆盖所有银行类和非银类金融机构（含政策性银行、大型银行、股份制银行、城商行、农商行、外资法人银行、信托公司、金融租赁公司、汽车金融公司等）。
- 报表结构以"准备金变动表"为核心：A列年初余额 → B列当年新提取 → C列冲销 → D列卖出资产 → E列转回 → F列其他变化 → G列期末余额，满足 G = A + B - C - D + E + F。
- 新会计准则实施状态（4.A/4.B/4.C）决定其他资产减值准备子项的填报口径：4.A=1 时填报 2.a-2.c（新准则三栏），4.C=1 时填报 2.1-2.4（旧准则四栏），4.B=1 时新旧并存。
- G03[3.G] ≥ G01[24.C]，各项资产减值损失准备合计期末余额 ≥ G01 对应减值准备项，形成表间校验。

## 重点口径

### 贷款损失准备计提范围

贷款损失准备计提范围为各项贷款，包括：贷款、贸易融资、票据融资、融资租赁、从非金融机构买入返售资产、透支、各项垫款等。各项贷款统计口径与 G01 资产负债项目统计表中的"各项贷款"保持一致。

### 新会计准则实施状态判定

- 4.A=1：整体实施修订后的《企业会计准则第22号——金融工具的确认和计量》（财会[2017]7号），仅填报 2.a-2.c
- 4.C=1：完全未实施新准则，仅填报 2.1-2.4
- 4.B=1：部分实施（如并表口径下母公司实行新标准、子公司实行旧标准），新旧并存填报

### 准备金变动逻辑

| 栏目 | 含义 | 正负方向 |
|------|------|----------|
| A 年初余额 | 年初结转上年末余额 | - |
| B 当年新提取 | 当年累计提取，计入当期损益 | 正数（可为负） |
| C 冲销 | 资产损失确认后使用准备金冲销 | 正数 |
| D 卖出资产 | 卖出已计提减值准备的资产 | 正数 |
| E 转回 | 已确认并冲销的损失收回后调整准备金 | 正数 |
| F 其他变化 | 汇率、年报审计等造成的差异 | 正/负 |
| G 期末余额 | 报告期期末余额 | G = A + B - C - D + E + F |

## 重点规则

### 表内核对关系

1. 4.A、4.B、4.C 三者互斥：任一为1时，其余两个必须为0
2. 2. = 对应子项之和：4.C=1 时 2. = 2.1+2.2+2.3+2.4；4.A=1 时 2. = 2.a+2.b+2.c；4.B=1 时新旧并存
3. 3. = 1. + 2.（各项资产减值损失准备合计 = 贷款损失准备 + 其他资产减值准备）
4. G = A + B - C - D + E + F（准备金变动平衡公式）

### 表间核对关系

- G03[3.G] ≥ G01[24.C]：各项资产减值损失准备合计期末余额 ≥ G01 对应减值准备项
