# 1104-G44-杠杆率情况表

## 简介

G44《杠杆率情况表》是 1104 系统中用于衡量商业银行杠杆率的专项监管报表，基于巴塞尔银行监管委员会《第三版巴塞尔协议杠杆率框架和披露要求》设计，通过建立基于总风险暴露、简单且非风险敏感性的支持性手段，强化以风险为本的资本监管。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银监统0028号

- 当前原文依据：[[01-资料库/1104系统/2026-04-20-G44-杠杆率情况表-填报说明-原文]]

## 当前结论

- G44 是 1104 系统中衡量银行杠杆率的核心报表，杠杆率 = 一级资本净额 / 杠杆率敞口总额。
- 杠杆率敞口由四部分组成：调整后的表内资产余额（[2.]）、衍生产品资产余额（[3.]）、证券融资交易资产余额（[4.]）、表外业务调整后的余额（[5.]）。
- 填报机构范围广泛：政策性银行、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用社、村镇银行、外资法人银行、金融租赁公司、汽车金融公司、消费金融公司和金融资产管理公司。
- 报送频度：季报。法人机构汇总数据为季后18日内；合并报表数据为季后40日内。
- G44 与 G4A《合格资本情况表》存在强表间校验关系：一级资本净额和一级资本扣减项均从 G4A 取数。
- G44 与 G01《资产负债项目统计表》存在表间校验关系（法人口径下）。
- G44 与 G4B-2《表外信用风险加权资产计算表（权重法）》存在表间校验关系：表外各档转换系数后的资产均从 G4B-2 取数。
- G44 的核心特点是非风险敏感性——它不依赖风险权重，而是以简单的总风险暴露作为分母，作为风险加权资本充足率的风险缓释手段。

## 重点口径

### 杠杆率计算公式

```
杠杆率 = 一级资本净额 / (调整后的表内资产余额 + 衍生产品资产余额 + 证券融资交易资产余额 + 表外业务调整后的余额) × 100%
```

### 表内核对关系

```
[2.] = [2.1] - [2.2] - [2.3] - [2.4]
[3.] = [3.1] + [3.2] + [3.3] - [3.4] - [3.5] + [3.6] - [3.7]
[4.] = [4.1] - [4.2] + [4.3] + [4.4]
[5.] = [5.1] + [5.2] + [5.3] + [5.4]
[6.] = [1.] / ([2.] + [3.] + [4.] + [5.])
```

### 表间核对关系

| 表间关系 | 说明 |
|------|------|
| [1.A] = G4A_[8.2] | 一级资本净额从 G4A 取数 |
| [2.1A] = G01_[25.C] | 表内总资产从 G01 取数（仅法人口径） |
| [2.2A] = G01_[23.2C] | 衍生产品资产从 G01 取数（仅法人口径） |
| [2.4A] = G4A_[2.A] + G4A_[4.A] | 一级资本扣减项从 G4A 取数（核心一级+其他一级） |

| [5.2] | [3.C] + [4.2.1C] + [4.3.1C] | 20% |
| [5.3] | [2.C] + [4.2.2C] + [4.3.2C] + [4.4C] + [4.5C] + [4.6C] | 50% |
| [5.4] | [1.C] + [5.C] + [6.C] + [7.C] + [8.C] + [9.C] + [10.C] | 100% |

### 证券融资交易净额结算规则

- 满足净额结算协议的 SFT：4.2 = max(0, 4.1 - 抵押品价值)
- 不满足净额结算协议的 SFT：4.2 = 0
- 交易对手信用风险暴露 4.3 = max(0, (SFT 资产 + 抵押品价值) - (SFT 负债 + 现金抵押品))

### 卖出信用衍生产品有效名义本金计算

| 衍生产品类型 | 有效名义本金计算方法 |
|------|------|
| 利率和信用衍生产品 | 合约名义本金 × 监管期限（min(合约剩余期限, 1年)，最低10天） |
| 汇率衍生产品 | 标的外汇一端的名义本金换算为人民币，两端均为外币时取较大值 |
| 股票和商品衍生产品 | 单位现期价格 × 总量 |

## 重点规则

- G44 的杠杆率指标是**非风险敏感性**的，作为风险加权资本充足率的风险缓释手段。
- 衍生产品部分包含重置成本、潜在风险暴露、抵质押品扣除、保证金应收、中央交易对手扣除、卖出信用衍生产品名义本金和可扣除信用衍生产品资产余额七个子项，计算逻辑复杂。
- 证券融资交易部分涉及净额结算规则：满足协议时可扣除部分资产，但需计算交易对手信用风险暴露。
- 表外业务部分完全依赖 G4B-2 的信用风险转换系数分类，四档分别为 10%、20%、50%、100%。
- 填报币种为本外币合计人民币，外币折算规则与 G01_V 一致。
