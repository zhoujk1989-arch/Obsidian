# 1104-G18-融资工具发行情况表

## 简介

G18《融资工具发行情况表》是 1104 系统中专门反映银行业金融机构各类融资工具发行情况的报表，将填报机构所发行各类融资工具分为有固定期限和无固定期限两大类，并按品种进行细分，反映各类融资工具的境内外余额、本年发行量以及按剩余期限分布的情况。

## 当前结论

- G18 是 1104 系统中专门反映银行业金融机构融资工具发行情况的报表，覆盖存单、普通债券、专项债券、次级债券、混合资本债、可转换债券、永续债、优先股、普通股等融资工具类型。
- 季报（季后13日，第一批次），报送法人汇总数据，填报机构范围广泛（17类机构）。
- 报表结构分为两大类：有固定期限融资工具（\[1.\]，含存单、普通债券、专项债券、次级债券、混合资本债、可转换债券等7子类）和无固定期限融资工具（\[2.\]，含永续债、优先股、普通股等4子类），以及\[3.\]各类融资工具合计。
- 列维度为8列：A（境内账面余额）、B（境内本年发行）、C（境外账面余额）、D（境外本年发行）、E/F/G/H（按剩余期限分类的境内+境外合计）。
- 表间校验：\[2.1.1A\]+\[2.1.1C\]+\[2.2.1A\]+\[2.2.1C\]+\[2.4A\]+\[2.4C\] ≤ G4A\[3.1A\]（仅适用于法人口径），即可计入其他一级资本的无固定期限融资工具合计 ≤ G4A 其他一级资本。
- G18 与 G4A（资本充足率明细表）存在概念交叉，两者都涉及资本工具合格标准的判定。
- 填报币种规则：所有币种折合人民币填报，按报告期末最后一天国家外汇管理局公布的基准汇价折算。
- 含权债券期限口径：如行权方已行权将赎回或回售，则按赎回日期或回售日期填报，否则按最终到期日填报。

## 重点口径

- **融资工具定义**：填报机构主动发行的、形成自身表内负债或所有者权益的融资工具。被动发行的不在此列。
- **含权债券期限口径**：对于可赎回或回售的含权债券，如行权方已行权将赎回或回售，则按赎回日期或回售日期填报；否则按最终到期日填报。
- **资产支持证券填报**：由发起机构填报，如标的资产仍在发起机构会计账面，按账面余额填报；不在账面的部分，也应按发行余额填报。
- **外币折算规则**：美元、欧元、日元、港币和英镑五种主要货币按国家外汇管理局基准汇价折算；其他货币按美元对人民币基准汇率与同一天上午9时国际外汇市场其他货币兑美元汇率套算确定。
