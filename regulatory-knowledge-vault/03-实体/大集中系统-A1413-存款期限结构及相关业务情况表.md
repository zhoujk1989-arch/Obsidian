# 大集中系统-A1413-存款期限结构及相关业务情况表

> [!note] 核心定位
> A1413 存款期限结构及相关业务情况表是大集中系统中的核心统计报表，用于按月监测和上报单位/个人、非存款类同业机构的**定期及协议存款期限错配**情况，并对**现金管理类理财**相关资金（存款、持仓投资、募集余额、持有的货基等）实施多维度额度及存量监控。它是监管部门防范负债端期限错配风险、流动性风险以及监测理财底层投向的关键报表。

## 基本信息

- **报表编码**：A1413
- **报表名称**：存款期限结构及相关业务情况表
- **系统归属**：大集中系统
- **报送机构**：银行、信用社、企业集团财务公司、农村资金互助社、理财公司。
- **报送范围**：全国境内汇总数据。
- **报送频度**：月报
- **原文依据**：[[01-资料库/大集中系统/2026-05-18-A1413-存款期限结构及相关业务情况表-原文|A1413 存款期限结构及相关业务情况表原文]]

## 指标体系与层级结构

本表的统计指标分为“期限拆分”和“现金管理类理财监控”两大模块。包含 14 个统计指标：

### 1. 定期及其他存款期限拆分树状结构
- **14A20 一、单位和个人定期及其他存款**：产品期限以存款（或存单）协议期限为依据。
  - **14A21 1年以内（含）**
  - **14A22 1至2年（含）**
  - **14A23 2至3年（含）**
  - **14A24 3年以上**
- **14A25 二、境内非存款类金融机构定期及其他存款**：产品期限以存款（或存单）协议期限为依据。
  - **14A26 1年以内（含）**
  - **14A27 1至2年（含）**
  - **14A28 2至3年（含）**
  - **14A29 3年以上**

### 2. 现金管理类理财监控指标
- **14A30 三、现金管理类理财存款**
- **14A31 四、现金管理类理财份额投资**
- **14A32 五、现金管理类理财募集资金**
- **14A33 六、现金管理类理财持有的货币市场基金**

## 重点口径与统计边界

### 1. 单位和个人定期及其他存款 (14A20)
- **包含范围**：定期存款、定活两便存款、通知存款、协议存款，以及单位和个人持有的本机构发行的**大额存单**。
- **排除范围**：信用卡存款、活期存款、协定存款、**保证金存款**、结构性存款、临时性存款。
- **期限划分策略**：必须严格以存款或存单的**协议期限（合同期限）**为划分标准，不得按剩余期限填报。

### 2. 境内非存款类金融机构定期及其他存款 (14A25)
- **非存款类金融机构范围**：境内银行业非存款类金融机构、境内证券业金融机构、境内保险业金融机构、境内结算清算类金融机构、境内金融控股公司、境内特定目的载体（SPV）、境内其他金融机构。
- **存款产品范围**：包含境内非存款类同业存放定期款项，以及这些金融机构持有的本机构发行的同业存单。
- **期限划分策略**：以存款或存单的**协议期限**为依据。

### 3. 现金管理类理财产品认定
- **标准定义**：根据银保监发〔2021〕20号《关于规范现金管理类理财产品管理有关事项的通知》，指仅投资于货币市场工具，每个交易日可办理产品份额认购、赎回的商业银行或者理财公司发行的理财产品。
- **口径一致性要求**：本表 A1413 涉及的全部现金管理类理财指标（14A30-14A33），其统计口径和底层数据**必须与金融机构资管产品统计（资管报送体系）保持一致**。

## 校验关系与勾稽公式

### 1. 表内加总强平衡校验
本表内所有期限细分子项之和必须完全轧平等于对应的大项余额。
$$14A20 = 14A21 + 14A22 + 14A23 + 14A24$$
$$14A25 = 14A26 + 14A27 + 14A28 + 14A29$$

### 2. 跨表勾稽平衡关系（与 A1411 资产负债项目表核对）
A1413 与“全科目”资产负债表 [[03-实体/大集中系统-A1411_A2411-金融机构资产负债项目月报表|A1411（人民币月报）]] 存在强勾稽关系，用以核验定期与理财数据的底层一致性：

#### 校验 A：单位和个人定期存款轧平关系
单位和个人定期及其他存款总量（14A20），在 A1411 表中可通过单位和个人各细分定期协议存款相加，扣除包含在其中的同业和非居民金额实现完全轧平：
$$14A20 = 12MGB + 12MGI + 12MGR + 12MI4 + 12MI6 + 12MI8 + 12MIA + 12MUE - 12MWM - 12MWB$$
- **A1411 右端项业务含义**：
  - `12MGB`：定期存款（单位）
  - `12MGI`：通知存款（单位）
  - `12MGR`：协议存款（单位）
  - `12MI4`：定期存款（个人）
  - `12MI6`：定活两便存款（个人）
  - `12MI8`：通知存款（个人）
  - `12MIA`：协议存款（个人）
  - `12MUE`：大额存单
  - `12MWM`：境内金融机构（需剔除同业存入定期部分）
  - `12MWB`：非居民（需剔除境外非居民定期部分）

#### 校验 B：境内非存款类金融机构存款轧平关系
境内非存款类金融机构定期及其他存款（14A25）的各科目加总轧平公式如下：
$$14A25 = 12MMA + 12MW2 + 12MW3 + 12MWM - 12MMB - 12MME - 12MMR$$
- **A1411 右端项业务含义**：
  - `12MMA`：金融机构存放定期及其他款项（总和）
  - `12MW2`：境内银行业非存款类金融机构存放定期/其他
  - `12MW3`：境内其他金融机构存放定期/其他
  - `12MWM`：境内金融机构
  - `12MMB`：货币当局存放（扣除）
  - `12MME`：境内银行业存款类金融机构存放（扣除）
  - `12MMR`：境外金融机构存放（扣除）

#### 校验 C：现金管理类理财存款上限关系
现金管理类理财属于特定目的载体（SPV）或同业投资范畴，其存入本机构的款项（14A30）不可能超过整个 A1411 负债端中境内 SPV 及其他金融机构的存放总额：
$$14A30 \le 12MKY + 12MMP + 12MW3$$
- **A1411 右端项业务含义**：
  - `12MKY`：境内特定目的载体存放（活期）
  - `12MMP`：境内特定目的载体存放（定期）
  - `12MW3`：境内其他金融机构存放

#### 校验 D：现金管理类理财份额投资上限关系
本机构持有的现金管理类理财资产余额（14A31），在 A1411 资产端中受限于整体非保本理财的资产总值：
$$14A31 \le 12MZS$$
- **A1411 右端项业务含义**：
  - `12MZS`：银行非保本理财投资

#### 校验 E：现金管理类理财募集资金上限关系
本机构发行的现金管理类理财的募集资金余额（14A32），理应从属于并小于等于全行非保本理财资金的总募集规模：
$$14A32 \le 12N09$$
- **A1411 右端项业务含义**：
  - `12N09`：非保本理财资金余额
