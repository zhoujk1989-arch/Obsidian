# 一表通-1.5自助机具

## 简介

`1.5自助机具` 采集上报机构辖内所有在行、离行式的，具有向最终客户提供银行机构服务的自助机具信息。

- 所属系统：一表通
- 数据域：设备/渠道基础信息
- 字段总数：13
- 当前原文依据：[[01-资料库/一表通系统/2026-04-27-1.5自助机具-填报说明-原文]]

## 关键字段与规则

### 主键字段

| 数据项编码 | 数据项名称 | 说明 |
| --- | --- | --- |
| A050001 | 机具ID | 唯一区分该机具，长度不超过32位数字与字母组合 |
| A050002 | 机构ID | 按金融许可证前11位+内部机构号生成 |
| A050012 | 采集日期 | PK，YYYY-MM-DD格式 |

### 核心业务字段

| 数据项编码 | 数据项名称 | 码值/规则 |
| --- | --- | --- |
| A050003 | 机具类型 | 01-ATM、02-CRS、03-自助查询终端、00-XX其他 |
| A050011 | 运营状态 | 01-正常、02-故障、03-撤除、00-XX其他 |
| A050007 | 设备地址 | 需写明省/地/县三级，军队或保密机构可不填（提示性校验） |
| A050009 | 设备启用日期 | YYYY-MM-DD，默认值9999-12-31 |
| A050010 | 设备停用日期 | YYYY-MM-DD，默认值9999-12-31 |

### 版本变更要点

- **2.1版本**：机具类型（A050003）数据类别由代码类改为文本类，码值"00-其他"调整为"00-XX 其他-银行自定义"
- **2.1版本**：运营状态（A050011）数据类别由代码类改为文本类，码值"00-其他"调整为"00-XX 其他-银行自定义"
- **2.1版本**：新增备注字段（A050013），文本类，anc..600

## 报送范围与口径

- **纳入范围**：上报机构辖内所有在行、离行式的，具有向最终客户提供银行机构服务的自助机具
- **半自助型机具**（如智能柜台机，尤其是移动式智能柜台，有银行人员介入的设备）属于自助机具，需要报送
- **第三方设备**：在银行厅堂投放的第三方设备不报送（不属于银行管理，仅使用场地）
