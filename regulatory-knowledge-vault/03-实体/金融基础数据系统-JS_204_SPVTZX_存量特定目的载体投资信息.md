# 金融基础数据系统-JS_204_SPVTZX_存量特定目的载体投资信息

## 简介

存量特定目的载体投资信息，采集金融机构报告期末持有的特定目的载体投资，以产品为单位逐笔报送。

## 关键信息

- 所属系统：金融基础数据系统
- 报表编码：JS_204_SPVTZX
- 报表名称：存量特定目的载体投资信息
- 报表大类：股权及特定目的载体投资
- 报表细类：特定目的载体投资
- 章节编号：3.6.2.1
- 当前原文依据：[[01-资料库/金融基础数据系统/2026-05-04-JS_204_SPVTZX_存量特定目的载体投资信息-原文|原文页]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- JS_204_SPVTZX 属于 JS_204 系列（股权及特定目的载体投资大类），与 JS_204_CLGQTZ（存量股权投资信息）和 JS_204_GQTZFS（股权投资发生额信息）共同构成 JS_204 系列。
- JS_204_CLGQTZ 覆盖"股权投资"（上市股票、非上市股票、其他股权投资），JS_204_SPVTZX 覆盖"特定目的载体投资"（银行理财、信托产品、券商资管、基金专户、公募基金、私募基金等资管产品），两者业务对象不同，互补覆盖金融机构对外投资的全景。
- 本表是金融基础数据系统首张特定目的载体（SPV）维度存量报表，使系统覆盖范围从"贷款+存款+债券+股权投资"扩展至"贷款+存款+债券+股权投资+特定目的载体投资"五大核心资产业务线。
- 报送粒度为"以产品为单位逐笔报送"，与 JS_204_CLGQTZ 的"以单张股权凭证为单位逐笔报送"粒度一致。

## 重点口径

1. **报送范围**：报告期末存续的特定目的载体投资，以产品为单位逐笔报送。不含报告期内已赎回/到期的产品。
2. **特定目的载体类型**：10类码值覆盖银行理财、信托产品、券商资管、基金专户、期货资管、保险资管、金融资产投资公司资管、公募基金、私募基金及其他。99为兜底类。
3. **运行方式**：4类码值区分开放/封闭与净值/非净值两种维度交叉。
4. **投资余额折人民币**：必填项。外币按金融机构会计核算使用汇率折算，人民币直接填余额。

## 重点规则

1. **发行人代码为多场景字段**：境内机构填统一社会信用代码（无则组织机构代码），境内个人填身份证件代码，已纳入资管统计制度的填资管产品统计编码，未纳入的填行业管理部门备案编码，境外机构填LEI码，境外个人/无LEI境外机构填自设编码。共7类场景。
2. **地区代码区分境内外**：境内用 GB/T 2260 县区级代码；境外前3位"000"填充+后3位 GB/T 2659 国别码。港澳台按境外客户填报，不得填报 810000/820000 等行政区划代码。
3. **资管产品统计编码优先级**：已纳入《金融机构资产管理产品统计制度》（银发〔2018〕299号）的境内SPV优先填统计编码；未纳入或暂未获取的填自设唯一编码。
4. **日期格式统一**：认购日期和到期日期均为 YYYY-MM-DD 格式。
