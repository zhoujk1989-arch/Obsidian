# S68-绿色贷款统计表

## 简介

S68《绿色贷款统计表》是金监统系列监管报表之一，用于全面统计和反映银行业金融机构对绿色产业的贷款支持情况。报表依据《绿色金融支持项目目录（2025年版）》进行分类，覆盖节能降碳、环境保护、资源循环利用、能源绿色低碳转型、生态保护修复、基础设施绿色升级、绿色服务、绿色贸易、绿色消费等九大境内绿色产业，以及境外绿色贷款和"一带一路"绿色贷款。

## 当前结论

- S68 是监管层追踪绿色金融发展情况的核心报表，覆盖境内绿色贷款、境外绿色贷款、"一带一路"绿色贷款和环境权益贷款四大板块。
- S68 为季报（季后13日），填报机构范围广泛，覆盖主要银行类金融机构。
- S68 采用极为细化的三级分类体系，境内绿色贷款按 9 大产业分类，每类下再分多子类，部分子类继续细分至三级。
- S68 的分类标准依据《绿色金融支持项目目录（2025年版）》，由中国人民银行、金融监管总局、中国证监会联合印发。
- 原文未明确列出与 S63/S71/S70/G01_Ⅶ 等报表的表间校验关系，但绿色贷款与 G01_Ⅶ 贷款投向分行业数据可能存在逻辑关联，需进一步确认。
- 环境权益贷款（[12]-[14]）作为附录项，涉及碳排放权、排污权、水权、用能权等环境权益的采购和抵押，是 S68 的独特维度。

### 填报机构

政策性银行（含开发银行）、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、农村商业银行、村镇银行、外资法人银行、外国银行分行、民营银行、直销银行、农村合作银行、农村信用合作社、企业集团财务公司、金融租赁公司、汽车金融公司。

### 绿色贷款定义

绿色贷款指金融机构为支持**环境改善、应对气候变化和资源节约高效利用**等领域的经济活动，发放给企（事）业法人、国家规定可以作为借款人的其他组织或个人，用于**生产、贸易、消费**等环节的贷款。

### 分类标准依据

《绿色金融支持项目目录（2025年版）》，由中国人民银行、金融监管总局、中国证监会联合印发。

### 环境权益贷款定义

指资金用于购买环境权益（如排污权、碳排放权、水权、用能权等）或以环境权益作为抵质押担保物的贷款，包括：

| 类型 | 说明 |
|------|------|
| 采购碳排放权贷款 | 采购全国统一碳市场碳排放权配额和CCER |
| 以碳排放权为抵押的贷款 | 以碳排放权为抵押担保 |
| 采购其他环境权益贷款 | 采购排污权、水权、用能权等 |
| 以其他环境权益为抵押的贷款 | 以排污权、水权、用能权等抵押担保 |

### 境外绿色贷款认定标准

在《目录》规定范围内，符合以下任一条件：

1. 符合联合国相关公约、世界银行集团及其他国际组织机构颁布的相关国际标准
2. 公开承诺遵守国际良好实践
3. 采用达到国际水平的中国标准（实质上不低于国际标准）

### 当年累放贷款年化利息收益计算

```
年化利息收益 = 贷款金额 × 合同年化利率 × (12/实际月数)

例：发放10万元贷款，2个月产生1000元利息
    年化利息收益 = 0.1 × 12/2 = 0.6万元
```

- 仅统计当年发放贷款的利息收入，往年不计入
