# 一表通-4.4分户账变动情况

## 简介

采集填报机构的所有分户账的变动情况，包括发生额和余额，应覆盖所有总账会计全科目涉及的科目项下的分户账（包括涉及的客户账变动）。共 15 个字段。

## 关键信息

- 所属系统：[[03-实体/一表通系统|一表通系统]]
- 报表编码：4.4
- 报表名称：分户账变动情况
- 数据域：账务信息
- 字段总数：15
- 当前原文依据：[[01-资料库/一表通系统/2026-04-27-4.4分户账变动情况-填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- 4.4分户账变动情况是账务信息数据域中记录分户账**变动过程**的报表，与 4.1 总账会计全科目、4.2 科目信息、4.3 分户账信息共同构成科目-账户两层映射体系中的"变动流水"层。
- 4.3 分户账信息记录分户账的**静态快照**（账户属性、当前余额），4.4 记录分户账的**动态变动**（本期借贷发生额、期初/期末余额）。
- 4.4 的覆盖范围应与 4.3 的分户账范围保持一致：所有存在分户账核算的科目项下分户账，原则上都应报送其期初/期末余额、借贷发生额及应收/应付利息。
- 主键由分户账号 + 机构ID + 采集日期 联合构成。
- 所有强制性校验字段共 13 个（除钞汇类别和备注外均为强制）。
- 应收利息覆盖发放贷款、存放同业、拆出资金和投资等生息资产；应付利息覆盖吸收存款、拆入资金、卖出回购证券及各种借款。

## 报送范围与业务场景

4.4 是 4.3 分户账的期间变动层，不应在每次 query 时重新从业务表散推。默认口径为：凡 4.3 已纳入的分户账，只要需要反映会计期间的期初余额、本期借贷发生额、期末余额或利息余额，就应进入 4.4。

| 场景 | 4.4 应反映的内容 | 典型业务含义 |
|------|------------------|--------------|
| 客户存款分户账 | 期初/期末贷方余额、本期借方/贷方发生额、应付利息 | 客户存入、支取、转账、结息、销户等引起账户余额或利息变化；外币存款还需关注钞汇类别。 |
| 信贷与垫款分户账 | 期初/期末借方余额、本期发放/收回/转逾期/核销等借贷发生额、应收利息 | 贷款发放、还本、还息、转逾期、垫款形成、垫款收回、核销或转让等账务变动。 |
| 信用卡分户账 | 账户余额变化、费用和利息形成、还款或消费后的余额变化 | 信用卡专门报表记录卡、交易和状态；4.4 用于承接其账务分户账期间余额和发生额。 |
| 同业分户账 | 同业资产或负债的期初/期末余额、本期拆入拆出/存放/回购变动、应收或应付利息 | 存放同业、同业存放、拆出拆入、买入返售、卖出回购等科目项下分户账的期间变化。 |
| 投资和融资分户账 | 投资本金或融资本金余额变动、买入/卖出/到期/发行/兑付发生额、应收或应付利息 | 债券投资、资管/SPV投资、同业存单、发行债券、其他融资工具等账务层变动。 |
| 表外与或有事项分户账 | 表外承诺、信用证、保函、承兑、担保等科目项下的发生额和余额变化 | 反映表外科目下分户账的开立、变更、释放、到期或转垫款前后的账务变化。 |
| 内部分户账 | 内部往来、清算、过渡、待清算、费用收入、税费、应收应付等内部明细账的余额和发生额 | 用于总账核对和内部明细核算，不能因为没有外部客户就排除。 |

### 查询默认口径

- 先用 [[03-实体/一表通-4.3分户账信息|4.3分户账信息]] 判断该账户是否属于分户账范围；属于的，再判断 4.4 是否需要报期初/期末余额、借贷发生额和利息余额。
- 不因某业务已有 7.x 交易流水、8.x 状态表或 6.x 协议表而排除 4.4；那些表描述业务事件，4.4 描述账务期间汇总。
- 若某业务只存在总账科目余额、没有分户账号或分户账核算，不能仅凭业务名称推定进入 4.4，应回到 4.2 分户账标识或系统核算口径确认。

### 字段分组

- **标识类**：分户账号、机构ID、币种、会计日期、采集日期
- **余额类**：期初借方余额、期初贷方余额、期末借方余额、期末贷方余额
- **发生额类**：本期借方发生额、本期贷方发生额
- **利息类**：应收利息、应付利息
- **辅助类**：钞汇类别、备注

## 重点口径

### 期初/期末余额与发生额的关系

- 本期借方发生额 = 期末借方余额 - 期初借方余额（借方余额类账户）
- 本期贷方发生额 = 期末贷方余额 - 期初贷方余额（贷方余额类账户）
- 对于非借贷类账户（如资产负债共同类），需根据科目性质判断借贷方向

### 应收利息（D040011）

- 覆盖范围：发放贷款、存放同业、拆出资金和投资等生息资产
- 填报口径：该分户账在会计日期内对应的应收利息余额
- 与 4.3 分户账信息的应收利息口径一致

### 应付利息（D040012）

- 覆盖范围：吸收的存款、拆入资金、卖出回购证券及各种借款
- 填报口径：应付利息余额，包括应付未付利息
- 与 4.3 分户账信息的应付利息口径一致

### 钞汇类别（D040014）

| 码值 | 含义 |
|------|------|
| 01 | 钞 |
| 02 | 汇 |
| 03 | 可钞可汇 |

> 用于区分外币存款通过实物或非实物形式存入银行的代码，人民币存款无钞汇标志。2.1 版本由"代码类"修改为"文本类"，格式由"2!n"修改为"anc..12"。

### 涉密法人处理

- 客户是境内涉密法人的，分户账号填写"*********"

### 机构ID 生成规则（D040002）

- 长度不超过 24 位数字与字母组合
- 前 11 位与金融许可证号前 11 位相同
- 后 13 位使用内部机构号，不足部分不用补齐
- 理财登记中心填报机构ID统一使用金融许可证
- 发行地涉及多个国家的，填写共同上级行机构ID

### 币种（D040004）

- 参照金融监管总局（或原银保监会）发布《公共数据标准》
- 遵循《表示货币和资金的代码》（GB/T 12406-2008），采用三位数字字母编码，如 CNY
- 多个币种合计折算为人民币（本外币合计）的，填报为 BWB
- 财务类数据应该按分币种填报

### 日期格式

- 一表通采用 `YYYY-MM-DD`（10位），EAST 采用 `YYYYMMDD`（8位），需进行格式转换
- 会计日期默认值为 9999-12-31

## 与 4.3 分户账信息的对比

| 维度 | 4.3 分户账信息 | 4.4 分户账变动情况 |
|------|----------------|---------------------|
| 记录类型 | 静态快照（账户属性+当前余额） | 动态变动（发生额+期初/期末余额） |
| 覆盖边界 | 所有存在分户账核算的科目项下分户账 | 与 4.3 分户账范围保持一致，记录其期间余额、发生额和利息变化 |
| 字段数 | 19 | 15 |
| 核心内容 | 分户账类型、计息方式、开户/销户日期等账户属性 | 期初/期末借贷余额、本期借贷发生额、应收/应付利息 |
| 主键 | 机构ID + 分户账号 + 采集日期（联合） | 分户账号 + 机构ID + 采集日期（联合） |
| 用途 | 描述分户账当前状态 | 描述分户账本期变动过程 |

> 两表通过分户账号 + 机构ID 关联，形成"账户状态 + 变动流水"的完整账务视图。
