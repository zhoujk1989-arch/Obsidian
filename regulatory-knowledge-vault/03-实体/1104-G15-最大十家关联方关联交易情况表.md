# 1104-G15-最大十家关联方关联交易情况表

## 简介

G15《最大十家关联方关联交易情况表》是 1104 系统中专门用于收集填报机构与关联方各类关联交易数据的报表，依据《银行保险机构关联交易管理办法》制定，旨在监测填报机构关联交易的集中度与风险敞口。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银监统0011号
- 报表名称：最大十家关联方关联交易情况表
- 频度：季报（法人汇总数据：季后18日内；合并报表数据：季后40天内）
- 填报机构：国家开发银行、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用合作社、村镇银行、外资法人银行、外国银行分行、金融租赁公司、汽车金融公司
- 数据单位：万元、百分比
- 当前原文依据：[[01-资料库/1104系统/2026-04-22-G15-最大十家关联方关联交易情况表-填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- G15 是 1104 系统中专门监测关联交易集中度的核心报表，覆盖最大十家关联方（含关联集团）的表内外授信情况。
- 报表编码明确为"银监统0011号"。
- 季报（法人汇总：季后18日；合并报表：季后40日），与 G51（国别风险敞口）同为季后40日报送。
- 填报机构范围广泛，覆盖政策性银行至外资法人银行及非银金融机构（金融租赁、汽车金融）。
- 第Ⅰ部分（1-10行 + G1-G10行）按授信净额从大到小排序，反映最大十家关联方/关联集团的授信情况。
- 第Ⅱ部分汇总全部关联方的表内外授信净额、关联交易金额、不良/逾期贷款余额。
- 表间校验：G15[11.C] = G40[3.A]（资本净额），G40 尚未在知识库中建立实体页。
- 授信定义涵盖表内外全部信用风险敞口，包括贷款、债券投资、特定目的载体投资、票据承兑、信用证、保理、担保、保函、贷款承诺、证券回购、拆借等。

## 重点口径

### 授信定义

根据《银行保险机构关联交易管理办法》，本表中授信的含义包括：各项贷款（含贸易融资、票据贴现、透支）、债券投资、特定目的载体投资、票据承兑、开立信用证、保理、担保、保函、贷款承诺、证券回购、拆借以及其他实质上由银行机构承担信用风险的表内外业务。

商业银行应当按照实质重于形式和穿透原则，识别、认定、管理关联交易及计算关联交易金额。

### 关联方定义

银行保险机构的关联方，是指与银行保险机构存在一方控制另一方，或对另一方施加重大影响，以及与银行保险机构同受一方控制或重大影响的自然人、法人或非法人组织。

**关联自然人包括：**
1. 自然人控股股东、实际控制人，及其一致行动人、最终受益人
2. 持有或控制银行保险机构5%以上股权的，或持股不足5%但对银行保险机构经营管理有重大影响的自然人
3. 董事、监事、总行和重要分行的高级管理人员，以及具有大额授信、资产转移、保险资金运用等核心业务审批或决策权的人员
4. 第1-3项所列关联方的配偶、父母、成年子女及兄弟姐妹
5. 法人控股股东、实际控制人等关联方的董事、监事、高级管理人员
6. 持有或控制5%以上股权的法人组织及其控股股东的董事、监事、高级管理人员

**关联法人或非法人组织包括：**
1. 法人控股股东、实际控制人，及其一致行动人、最终受益人，及上述关联方控制或施加重大影响的法人
2. 持有或控制5%以上股权的法人组织及其控股股东的法人
3. 银行保险机构控制或施加重大影响的法人
4. 自然人控股股东等关联方控制或施加重大影响的法人
5. 关联自然人中第2-4项所控制或施加重大影响的法人

**实质重于形式可认定的关联方：**
1. 过去或未来12个月内存在关联情形的
2. 关联自然人的其他关系密切的家庭成员
3. 银行保险机构内部工作人员及其控制的法人或组织
4. 关联自然人第2-3项、关联法人第2项所列关联方可施加重大影响的法人
5. 与银行保险机构发生或可能发生未遵守商业原则、有失公允交易的主体

### 资本净额口径

资本净额与资本充足率计算表中的资本净额相同，填报口径为银行保险机构法人口径。适用于《商业银行资本管理办法（试行）》的填报机构，资本净额项目应与新资本充足率G40《资本充足率汇总表》中的【3.资本净额】项目保持一致。

### 净额计算

净额 = 表内授信（H+I+J+K）+ 表外授信（L+M）- 保证金、银行存单、国债（O）

### 排序规则

- 第1-10行：按单一关联方表内外授信净额（N列）从大到小排序
- 第G1-G10行：按关联集团表内外授信净额（N列）从大到小排序
- 无集团的单一关联方也需要纳入关联集团的范畴进行排序
