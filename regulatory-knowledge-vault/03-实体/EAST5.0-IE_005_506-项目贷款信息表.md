## 关键字段

### 主键/关联键

- `XDHTH`（信贷合同号）：关联 `IE_005_501` 信贷合同表
- `XDJJH`（信贷借据号）：关联 `IE_005_503` 个人信贷业务借据表 或 `IE_005_504` 对公信贷业务借据表
- `NBJGH`（内部机构号）：关联 `IE_001_101` 机构信息表
- `JKRBH`（借款人编号）：关联 `IE_002_203` 对公客户信息表

### 金额字段

- `DKJE`（贷款金额）：借款人应还本金总额，与币种相对应金额
- `DKYE`（贷款余额）：贷款账户截止到目前未还的本金余额，包括逾期未还本金，已转让/核销贷款余额为 0
- `XMZTZ`（项目总投资）：与币种相对应金额
- `XMZBJ`（项目资本金）：与币种相对应金额

### 项目五证（房地产项目专用）

- `TDSYZBH` / `TDSYZRQ`：土地使用证编号/日期（非房地产类项目贷款允许为空）
- `YDGHXKZBH` / `YDGHXKZRQ`：用地规划许可证编号/日期（非房地产类项目贷款允许为空）
- `GCGHXKZBH` / `GCGHXKZRQ`：工程规划许可证编号/日期（非房地产类项目贷款允许为空）
- `SGXKZBH` / `SGXKZRQ`：施工许可证编号/日期（非房地产类项目贷款允许为空）
- `QTXKZ` / `QTXKZBH`：其他许可证/编号（允许为空，多个用英文半角分号隔开）

### 立项批文

- `PWWH`（批文文号）：银行自定义，无相关批文允许为空
- `LXPW`（立项批文）：批文标题，无相关批文允许为空

### 日期字段

- `KGRQ`（开工日期）：YYYYMMDD，默认值 99991231
- `CJRQ`（采集日期）：YYYYMMDD，默认值 99991231

### 状态/分类字段

- `XMLX`（项目类型）：`EAST_XMFL` 码表，枚举包括基础设施建设项目、房地产项目、技术改造项目、科技开发项目、并购贷款、其他-银行自定义
- `SFYT`（是否银团）：`EAST_SFBZ` 码表，是/否
- `DKZT`（贷款状态）：`EAST_DKZT` 码表，正常/结清/逾期/核销/转让
