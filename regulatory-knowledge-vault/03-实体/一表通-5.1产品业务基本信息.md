# 一表通-5.1产品业务基本信息

## 简介

采集填报机构在经营活动中涉及的所有产品、业务的共性特征数据。应填报能为金融消费者提供最小定制、最细分且能唯一识别的产品或业务。

## 基本信息

| 项目 | 内容 |
|------|------|
| 报表编码 | 5.1 |
| 报表名称 | 产品业务基本信息 |
| 所属系统 | 一表通 |
| 字段总数 | 17 |
| 主键 | 产品ID(E010001)、机构ID(E010002)、采集日期(E010017) |

## 业务范围

- 填报机构在经营活动中涉及的所有产品、业务的共性特征数据
- 应包含存款、贷款、债券、理财、保险和卡产品
- 应包括自营、代理代销、投融资涉及的所有产品、业务
- 分类产品表为本表的补充表，通过产品ID关联

## 关键字段与规则

### 产品ID（E010001）— 主键

- 唯一识别银行机构在经营活动中满足金融消费者定制化需求、最细颗粒度的产品或业务
- 理财产品：前14/15位使用产品登记编码（理财登记中心赋予），后不足部分不用补齐
  - 公募：14位 = 6位金融机构代码 + 2位年度码 + 6位顺序码
  - 私募：15位 = 6位金融机构代码 + 2位年度码 + 1位标识码 + 6位顺序码
- 其他产品：使用内部产品编号，不足部分不用补齐
- 强制性校验，整表新增

### 机构ID（E010002）— 主键

- 分配给实际业务经办最小单位机构的唯一编码
- 前11位与金融许可证号前11位相同，后13位使用内部机构号
- 理财登记中心填报机构ID统一使用金融许可证
- 发行地涉及多国时，填写共同上级行机构ID
- 强制性校验，主键

### 科目类型（E010005）

- 代码类，提示性校验
- 码值：01-资产、02-负债、03-所有者权益、04-损益、05-资产负债共同类、06-表外、00-其他
- 债券借贷科目类型填"06表外"
- 产品类别可以置空

### 自营标识（E010008）

- 01-自营产品、02-代客产品、03-混合
- 用于区分产品是否为该银行机构的自营产品
- 强制性校验

### 产品期限（E010010）

- 一次性产品：填写产品从成立至到期的持续天数
- 持续性产品：填写0
- 存款类产品为持续性产品，债券类产品为一次性产品
- 强制性校验

### 产品到期日期（E010012）

- 一次性产品：填写约定到期日期
- 持续性产品：默认值 9999-12-31
- 强制性校验

### 代客产品所属机构名称（E010018）

- 2.0试用版新增字段
- 仅限代客产品填报

### 备注（E010016）

- 2.1版本数据格式由 anc 缩短为 anc..600
- 不同备注事项用英文半角分号隔开

## 产品类别填报规则

- 参照产品类别通用字段要求
- 同一产品涉及多个产品类别的，应将多个代码一并填报，使用英文半角分号隔开

## 产品币种填报规则

- 遵循 GB/T 12406-2008《表示货币和资金的代码》，采用三位字母编码（如CNY）
- 多币种以英文半角分号分隔
- 涉及所有币种填报"ALL"，仅涉及外币可填"WB"

## 利率类型码值

- 01-固定利率
- 02-浮动利率
- 03-固定或浮动利率（产品包含固定或浮动利率，可由客户选择）

## 产品状态码值

- 01-正常：该产品处于正常服务状态
- 02-停用：该产品已停止新提供服务状态
