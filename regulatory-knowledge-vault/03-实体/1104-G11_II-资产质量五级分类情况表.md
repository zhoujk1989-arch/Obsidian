# 1104-G11_II-资产质量五级分类情况表（第Ⅱ部分）

## 简介

G11_II 是 G11《资产质量五级分类情况表》的第Ⅱ部分"资产质量及准备金"，专门反映填报机构承担信用风险的各类资产的五级分类分布情况、减值准备计提情况以及表外承诺和或有负债的信用风险敞口。

- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- G11_II 是 1104 系统中专门反映信用风险资产质量的**核心报表**，覆盖从各项贷款到各类投资、同业资产、表外承诺及或有负债的全口径信用风险敞口。
- 第Ⅱ部分与第Ⅰ部分（按行业分类的贷款）相互独立又关联：第Ⅱ部分\[1.\] = 第Ⅰ部分\[1.\]（各项贷款）。
- 报送频度：**季报**（法人机构汇总数据），季后 13 日内。
- 填报机构范围广泛，但**农村资金互助社、贷款公司和货币经纪公司不需要报送第Ⅱ部分**。
- 五级分类定义直接引用《商业银行金融资产风险分类办法》（2023年第1号），与旧版《贷款风险分类指引》口径有更新。
- 减值准备计提依据《商业银行贷款损失准备管理办法》（2011年第4号）和新企业会计准则。

## 重点口径

### 五级分类定义

| 类别 | 定义 | 来源 |
|------|------|------|
| 正常类 | 债务人能够履行合同，没有客观证据表明本金、利息或收益不能按时足额偿付 | 《商业银行金融资产风险分类办法》 |
| 关注类 | 虽然存在可能对履行合同产生不利影响的因素，但债务人目前有能力偿付 | 同上 |
| 次级类 | 债务人无法足额偿付本金、利息或收益，或金融资产已经发生信用减值 | 同上 |
| 可疑类 | 债务人已经无法足额偿付本金、利息或收益，金融资产已发生显著信用减值 | 同上 |
| 损失类 | 在采取所有可能的措施后，只能收回极少部分金融资产，或损失全部金融资产 | 同上 |

- **正常资产** = 正常类 + 关注类
- **不良资产** = 次级类 + 可疑类 + 损失类
- **逾期资产**：未能按合同规定的期限兑付本金、利息或其他收益的资产余额（独立于五级分类）
- **金融资产已发生信用减值**：根据《企业会计准则第22号》第四十条，因债务人信用状况恶化导致的金融资产估值向下调整

### 减值准备口径

- 贷款损失准备：商业银行在成本中列支、用以抵御贷款风险的准备金，**不包括**利润分配中计提的一般风险准备。
- 非贷款类资产减值准备：基于未来现金流量现值低于账面价值的差额确认。
- 计提范围：拆放同业、存放同业、买入返售资产、各类投资、应收利息、其他应收款等。
- 减值准备计提比例：根据以往经验、债务单位实际财务状况和现金流量合理估计。

### 抵质押品覆盖规则

- 保证金和抵质押品价值**先覆盖本金**，超额部分可覆盖应收利息。
- 抵质押品覆盖金额**不超过应收利息余额**。
- 对每一交易对手，账面余额和公允价值**不超过债务本金和应计利息合计**。

### 境内/境外定义

- **境内**：除香港、澳门和台湾地区以外的中华人民共和国境内。
- **境外**：中华人民共和国以外的国家和地区，以及香港、澳门特别行政区和台湾地区。

## 重点规则

### 表内核对关系

| 关系 | 说明 |
|------|------|
| A = B + E = C + D + F + G + H | 列维度：账面余额 = 正常 + 不良 = 正常类 + 关注类 + 次级类 + 可疑类 + 损失类 |
| B = C + D | 正常资产 = 正常类 + 关注类 |
| E = F + G + H | 不良资产 = 次级类 + 可疑类 + 损失类 |
| 23. = 1. + 2. + ... + 22. | 行维度：信用风险资产小计 = 各资产项目合计 |
| 23.1 = 1.1 + 2.1 + ... + 22.1 | 不良资产减值准备合计 = 各项目减值准备 |
| 23.2 = 1.2 + 2.2 + ... + 22.2 | 全部资产减值准备合计 |
| 25. = 23. + 24 | 表内外信用风险资产合计 = 信用风险资产小计 + 表外承诺及或有负债 |
