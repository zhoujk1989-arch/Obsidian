# 大集中系统-A1433_A2433-涉农贷款月报表

> [!note] 核心定位
> A1433 和 A2433 涉农贷款月报表是大集中系统中的核心月度统计报表，用于按月上报涉农信贷余额、结构分布及特定支农专项贷款数据。与季报侧重于五级分类质量与担保维度不同，月报重点关注**普惠性涉农贷款（单户授信额度分档）**、**特定用途支农贷款（农田建设、农产品加工、农资流通等）**以及**粮食安全与特色农业专项信贷**，是评估金融机构支持乡村振兴、普惠涉农贷款投放进度的核心月度监测工具。

## 基本信息

- **报表编码**：A1433（法人/汇总级） / A2433（分支/独立级）
- **报表名称**：涉农贷款月报表
- **系统归属**：大集中系统
- **报送频度**：月报
- **原文依据**：[[01-资料库/大集中系统/2026-05-18-A1433_A2433-涉农贷款月报表-原文|A1433、A2433 涉农贷款月报表原文]]

## 指标体系与层级结构

本报表指标体系以 A1433 对应以 `12` 开头的指标编号，A2433 对应以 `22` 开头的指标编号。以下以 A1433（12系列）为例展示月报的核心指标树状结构：

### 1. 涉农贷款总额 (12P01 / 22P01)
涉农贷款总额由以下四大核心板块构成：
* **12P02 农林牧渔业贷款**：承贷主体从事农、林、牧、渔业活动的所有贷款。
  * **12P03 其中：非农户个人农林牧渔业贷款**（除农户以外的个人从事农业活动的贷款）。
* **12P04 农户贷款**：发放给符合监管定义农户的所有贷款。
  * **12P06 农户消费贷款**
  * **12P1H 农户经营贷款**：按单户授信额度分档统计（**2025年口径明确不含消费类授信**）：
    * **12P1I 单户授信小于500万（含）**
    * **12P1J 单户授信500-1000万（含）**
    * **12P1K 单户授信1000-2000万（含）**
    * **12P2I 单户授信2000-3000万（含）**
    * **12P1M 单户授信大于3000万**
  * **12P2J 农户经营贷款户数**（*仅 A1433 采集*，反映涉农经营实体覆盖面）
    * **12P2K 其中：单户授信小于500万（含）的户数**（*仅 A1433 采集*）
* **12P07 农村非金融企业及机关团体贷款**：注册地位于农村区域的非金融企业与团体的所有贷款。
  * **12P1N 其中：农林牧渔业贷款**（注册地在农村且从事农业活动）
  * **12P1P 其中：支农贷款**：包括以下六大特定用途贷款：
    * **12P1Q 农田基本建设贷款**
    * **12P1R 农产品加工贷款**
    * **12P1S 农业生产资料制造贷款**
    * **12P1T 农用物资和农副产品流通贷款**
    * **12P1U 农业科技贷款**
    * **12P1V 农村基础设施建设贷款**
* **12P0C 城市非金融企业及机关团体涉农贷款**：注册地位于城市区域，但从事农业活动或支持农业农村发展的贷款。
  * **12P1W 农林牧渔业贷款**
  * **12P1X 支农贷款**：包括以下六大特定用途贷款：
    * **12P1Y 农田基本建设贷款**
    * **12P1Z 农产品加工贷款**
    * **12P20 农业生产资料制造贷款**
    * **12P21 农用物资和农副产品流通贷款**
    * **12P22 农业科技贷款**
    * **12P23 农村基础设施建设贷款**

### 2. 附表及专项信贷指标 (附表板块)
* **县城城区非金融企业及机关团体信贷结构**：
  * **12P24 县城城区非金融企业及机关团体贷款**
    * **12P25 其中：农林牧渔业贷款**
    * **12P26 其中：支农贷款**（细分为：12P27 农田建设、12P28 农副加工、12P29 农资制造、12P2A 农资流通、12P2B 农业科技、12P2C 农村基建）
* **乡村振兴与特色农业专项贷款**：
  * **12P1E 林权抵押贷款**：以林权作为抵押物的贷款。
  * **12P2D 粮食生产贷款**：支持谷物、豆类和薯类种植活动的贷款。
  * **12P2E 其他重要农产品生产贷款**：支持油菜、棉花、蔬菜、生猪饲养、水产养殖等。
  * **12P2F 种业振兴贷款**：支持种质资源保护、育种创新和繁育推广。
  * **12P2G 农产品初加工、精深加工贷款**：支持农林牧渔产品加工与制造。
  * **12P2H 农业机械贷款**：支持农机研发、制造、流通与租售维修。

## 核心概念与统计边界

### 1. 农户及区域划分边界
本表所涉及的“农户”、“城市区域”、“农村区域”与季报完全保持一致：
* **农户的“时点判定”规则**：以贷款发放时承贷主体的身份为准，重点包含乡镇行政区住户、城关镇所辖行政村住户、国有农场职工和农村个体工商户等，排除举家外出谋生一年以上的住户（详见季报 [[03-实体/大集中系统-A1333_A2333-涉农贷款季报表|大集中系统-A1333_A2333-涉农贷款季报表]]）。
* **城市与农村界定**：以注册地为准，城市区域涵盖地级以上城市市区、市辖建制镇、县城城区和国务院/省级审批的各类开发区；除城市区域外即为农村区域。

### 2. 支农贷款的六大用途硬约束
支农贷款（如 12P1P / 12P1X / 12P26）仅涵盖以下六种**特定用途**：
1. **农田基本建设**：用于小型水利、大中型灌区改造、中低产田改良、防灾减灾等。
2. **农产品加工**：指以农林牧渔产品为原料的普通加工活动（如谷物磨制、屠宰、中药饮片、纺织加工等）。**排除**精深加工和《国民经济行业分类》中的初加工小类（0514/0523小类），该部分由附表“农产品初加工、精深加工贷款”(12P2G)承接。
3. **农业生产资料制造**：用于化肥、农药、农膜、农用机械等的制造贷款。
4. **农用物资和农副产品流通**：用于农产品收购、储备、批发零售以及农副产品出口。
5. **农业科技**：用于农业科研试验、农技推广、科技中介服务。
6. **农村基础设施**：用于农村安全饮水、清洁能源、农村电网、公路信息网以及位于农村的教育、医疗等公共设施建设。

## 重点校验与表内逻辑关系

在月度涉农贷款数据报送中，存在以下强表内逻辑校验关系：

### 1. 四大板块求和轧平关系
涉农贷款总额等于农林牧渔业贷款、农户贷款、农村企业贷款、城市企业涉农贷款之和。
$$12P01 = 12P02 + 12P04 + 12P07 + 12P0C$$
$$22P01 = 22P02 + 22P04 + 22P07 + 22P0C$$

### 2. 农户贷款细分结构逻辑
* **消费与经营求和轧平**：农户贷款总额等于消费贷款与经营贷款之和。
  $$12P04 = 12P06 + 12P1H$$
  $$22P04 = 22P06 + 22P1H$$
* **授信额度分档全轧平**：农户经营贷款总额必须等于五个单户授信分档额之和。
  $$12P1H = 12P1I + 12P1J + 12P1K + 12P2I + 12P1M$$
  $$22P1H = 22P1I + 22P1J + 22P1K + 22P2I + 22P1M$$
* **户数校验逻辑（仅 A1433）**：农户经营贷款总户数必须大于或等于小于500万分档的户数。
  $$12P2J \ge 12P2K$$

### 3. 企业与机关团体支农贷款特定校验
* **农村企业支农贷款求和轧平**：
  $$12P1P = 12P1Q + 12P1R + 12P1S + 12P1T + 12P1U + 12P1V$$
  $$22P1P = 22P1Q + 22P1R + 22P1S + 22P1T + 22P1U + 22P1V$$
* **城市企业涉农贷款求和轧平**：城市涉农总额仅由农林牧渔与支农两项构成。
  $$12P0C = 12P1W + 12P1X$$
  $$22P0C = 22P1W + 22P1X$$
  * *支农明细求和关系*：
    $$12P1X = 12P1Y + 12P1Z + 12P20 + 12P21 + 12P22 + 12P23$$
    $$22P1X = 22P1Y + 22P1Z + 22P20 + 22P21 + 22P22 + 22P23$$

### 4. 附表县城城区企业信贷校验
* **县城城区支农贷款求和轧平**：
  $$12P26 = 12P27 + 12P28 + 12P29 + 12P2A + 12P2B + 12P2C$$
  $$22P26 = 22P27 + 22P28 + 22P29 + 22P2A + 22P2B + 22P2C$$
* **大类与子项大小逻辑**：
  $$12P24 \ge 12P25 + 12P26$$
  $$22P24 \ge 22P25 + 22P26$$

## 2024 & 2025 年重要修订

### 1. 2024 年修订：普惠信贷授信上限提升与指标增设
* **修订背景**：配合国家普惠小微贷款单户授信认定标准由“不超过1000万”放宽至“不超过2000万”的监管要求。
* **月报变更**：涉农月报增设并调整了授信额度划分区间，增设了 **12P1K/22P1K（1000-2000万）** 与 **12P2I/22P2I（2000-3000万）** 两个指标，用于精准刻画在额度标准放宽后，中等授信农户经营实体的信贷分布，确保普惠涉农信贷监测政策的科学性。

### 2. 2025 年修订：单户授信口径剔除消费性授信（“五篇大文章”硬约束）
* **政策要求**：按照金融“五篇大文章”总体统计制度的规范性硬约束。
* **统计要求**：明确涉农贷款月报表（A1433/A2433表单）中，农户经营贷款“单户授信”的统计口径**一律不含消费类授信**。
* **业务落地影响**：
  * 金融机构在将单个农户客户归入特定授信区间（如 $<500$ 万、 $500\text{-}1000$ 万等）时，必须在后台**剔除其名下的纯消费性质额度**，仅计算其用于农业生产、工商经营等经营性质的统一授信额度。
  * 该修订有效防止了消费金融信用卡的授信膨胀扭曲“普惠涉农经营贷款”的真实画像，保障了实体支农资金的纯洁性。
