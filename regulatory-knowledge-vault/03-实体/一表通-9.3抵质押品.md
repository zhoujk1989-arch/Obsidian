# 一表通-9.3抵质押品

## 简介

9.3抵质押品是一表通协议域中的抵质押品属性报表，36个字段，2个主键（押品ID、担保协议ID），16个强制性校验字段。填报机构涉及的抵质押品信息，覆盖抵质押物类型、估值、抵质押率、登记信息、权证信息、股票类押品预警/平仓线标识等。

## 当前结论

本表是一表通协议域中负责采集抵质押品详细信息的报表，与 6.8 担保协议配合使用：6.8 记录担保协议本身，9.3 记录担保协议下具体抵质押物，通过 `J030002` 担保协议ID 关联。本表覆盖全部抵质押品类型（48类），包括金融质押品、房地产、存货、知识产权、收费权等。一表通比 EAST 多报送保证金形式的押品；抵质押物类型和状态码值一表通为数字码值、EAST 为文本码值；采集日期格式一表通为 YYYY-MM-DD、EAST 为 YYYYMMDD。

## 重点口径

1. **填报范围**：填报机构涉及的抵质押品信息。报数期开始前该抵质押品已经解除抵质押状态或已经完成处置的，不填报。失效担保协议的押品信息不再报送。
2. **一物一报**：对于同一担保协议有不同的抵质押物的，需以不同的质押或抵押物编号分多条记录报送。
3. **保证金押品**：以保证金和存单质押形式的押品需要填报。一表通需报送保证金形式的押品，而 EAST 不报送保证金形式的担保。
4. **公积金转商业贷款**：对公积金住房贷款转成商业住房按揭贷款涉及的相关抵质押品信息允许为空。
5. **抵质押率计算**：根据《商业银行押品管理指引》规定：抵质押率＝押品担保本金余额÷押品估值×100%。按合同内整体押品计算，不是单件押品计算。
6. **已抵押价值**：在办理该笔信贷业务前已进行过抵押业务的，填报押品已经抵押的价值；第一顺位时填报为0。
7. **所有权人**：对应多个所有权人时，只需报送一个，可取占有份额最大的报送。法人填法人名称，自然人填脱敏后姓名。
8. **日期默认值**：首次估值日期、最新估值日期、估值到期日期、登记日期默认值均为 9999-12-31。
9. **唯一担保协议标识**：描述抵质押物是否对应唯一的担保协议，0-否，1-是，范围限定于本行。
10. **备注格式**：不同备注事项用英文半角分号隔开。

## 重点规则

| 规则 | 说明 |
| --- | --- |
| 押品ID生成 | 能唯一识别该押品的编号，最长60位，数字与字母组合 |
| 担保协议ID生成 | 按照既定规则生成，长度不超过60位数字与字母组合，不足部分不用补齐 |
| 机构ID生成 | 前11位与金融许可证号前11位相同，后13位使用内部机构号；发行地涉及多国时填共同上级行机构ID |
| 币种编码 | 参照 GB/T 12406-2008，三位数字字母编码；多币种合计折算为人民币时填报 BWB |
| 抵质押率格式 | 百分比为单位，如 4.5% 填报为 4.5 |
| 权证登记面积 | 单位为 ㎡，如 1000 平方米填报为 1000 |
