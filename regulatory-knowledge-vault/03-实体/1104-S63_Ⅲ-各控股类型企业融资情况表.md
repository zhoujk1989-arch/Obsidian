# S63_Ⅲ-各控股类型企业融资情况表

## 简介

S63_Ⅲ《各控股类型企业融资情况表》（原名"企业贷款情况表（按企业性质划分）"）是 1104 系统监管报表之一，旨在收集各银行在报告日对企业贷款按**企业控股性质**（国有/集体/私人/港澳台商/外商/其他）划分的有关情况。报表以控股类型为核心维度，覆盖贷款余额、五级分类、担保方式、逾期情况、贷款户数、累计发放额、不良处置、表外业务（银行承兑汇票/跟单信用证/保函/贷款承诺）及 PPP 项目贷款等指标。月报（月后6日），与 S63_I 形成表间校验关系。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统S63_Ⅲ
- 报表名称：企业贷款情况表（按企业性质划分）/ 各控股类型企业融资情况表

## 当前结论

- S63_Ⅲ 是 1104 系统中以**"控股类型"**为核心维度的企业贷款报表，弥补了 S63（按企业规模）和 S64（按行业）和 S65（按地区）缺乏股权性质视角的空白。
- S63_I、S63_Ⅲ、S64、S65 共同构成**"规模×控股性质×行业×地区"**的四维企业贷款观察框架。
- S63_Ⅲ 的列维度为 A-H（8列），覆盖国有控股、集体控股、私人控股（含小微企业子项）、港澳台商控股、外商控股、其他、个人经营性贷款。
- S63_Ⅲ 的 H 列是汇总列，满足 H = A+B+C+D+E+F+G（贷款类行）或 H = A+B+C+D+E+F（表外业务行）。
- S63_Ⅲ 的核心表间校验关系：A+B+C+D+E+F = S63_I 的国有+集体+私人+港澳台商控股合计，体现"控股维度"与"规模维度"的校验关系。
- S63_Ⅲ 的 [1.G]（个人经营性贷款余额）= S63_I[1.E]（小微型企业贷款余额），这一等式关系需要进一步确认业务含义。
- S63_Ⅲ 包含 PPP 项目贷款统计（[12]），分为新机制前和新机制两部分，体现了对政府和社会资本合作项目的专项追踪。
- S63_Ⅲ 包含知识产权质押贷款统计（[4.a] 和 [3.a]），体现了对知识产权质押融资的专项追踪。
- S63_Ⅲ 的控股类型定义极为详细：国有控股（绝对控股>50% + 相对控股）、集体控股、私人控股、港澳台商控股、外商控股，每类均区分绝对控股和相对控股。

## 字段概览

### 列项目（8列 A-H）

| 列编码 | 列名称 | 说明 |
|------|------|------|
| A | 国有控股 | 含国有绝对控股（>50%）与国有相对控股 |
| B | 集体控股 | 含集体绝对控股（>50%）与集体相对控股 |
| C | 私人控股 | 含私人绝对控股（>50%）与私人相对控股 |
| C1 | 私人控股-小微企业 | 私人控股中的小型+微型企业之和（C ≥ C1） |
| D | 港澳台商控股 | 含港澳台商绝对控股（>50%）与港澳台商相对控股 |
| E | 外商控股 | 含外商绝对控股（>50%）与外商相对控股 |
| F | 其他 | 信托计划等非法人机构持有企业股权 |
| G | 个人经营性贷款 | 个体工商户+小微企业主+农户经营贷款 |
| H | 合计 | H = A+B+C+D+E+F+G（贷款类行）或 H = A+B+C+D+E+F（表外业务行） |

> 待确认：C1 列是否在所有行中都存在，还是仅在部分行（如 [1]、[2]、[3]、[4]）中存在。

### 控股类型定义

| 控股类型 | 绝对控股 | 相对控股 | 特殊规则 |
|------|------|------|------|
| **国有控股** | 国有经济成分出资人实收资本占比 > 50% | 占比虽≤50%但相对最大，或协议拥有实际控制权 | 投资双方各占50%且一方为国有经济成分的，一律按国有控股 |
| **集体控股** | 集体经济成分出资人实收资本占比 > 50% | 占比虽≤50%但相对最大，或协议拥有实际控制权 | 集体经济与国有经济各占50%的，按国有控股；与除国有经济以外的经济类型各占50%的，按集体控股 |
| **私人控股** | 私人经济成分出资人实收资本占比 > 50% | 占比虽≤50%但相对最大，或协议拥有实际控制权 | - |
| **港澳台商控股** | 港澳台商经济成分出资人实收资本占比 > 50% | 占比虽≤50%但相对最大，或协议拥有实际控制权 | - |
| **外商控股** | 外商经济成分出资人实收资本占比 > 50% | 占比虽≤50%但相对最大，或协议拥有实际控制权 | - |
| **其他** | - | - | 信托计划等非法人机构持有企业股权 |

### PPP 项目贷款定义

- 投向政府和社会资本合作（PPP）项目的贷款余额
- 包括新机制前发放的贷款和按照新机制发放的贷款两部分
- 新机制指 2023 年 11 月《国务院办公厅转发国家发展改革委、财政部<关于规范实施政府和社会资本合作新机制的指导意见>的通知》（国办函〔2023〕115号）

### 知识产权质押贷款

- [3.a]：抵质押物中包含专利权、商标权、著作权等知识产权的贷款户数
- [4.a]：抵质押物中包含专利权、商标权、著作权等知识产权的贷款余额
- 如果知识产权不是唯一担保方式，则根据最新估值对贷款余额按比例拆分
- 具体拆分规则参考 G13 贷款余额项目的说明

### 法人账户透支

- 银行在约定的账户、额度、期限内以透支形式向法人客户提供短期融资和结算便利
- 一般以日间透支形式用款，当日归还，日终不占用授信额度，不产生借款利息

### 当年累计发放贷款年化利息收入

- 当年累计发放贷款所获得的利息收入按年折算
- 计算公式：放款金额 × 实际利息收入 / 实际月数 × 12
- 例：10万元贷款，两个月产生1000元利息，年化利息收入 = 0.1 × 12/2 = 0.6万元
- 与 S71_I[F.] 含义一致，但范围有差异：本表不含买断式转贴现，且没有授信总额限制

## 备注

- S63_Ⅲ 是 1104 系统中以"控股类型"为核心维度的企业贷款报表，与 S63（规模）、S64（行业）、S65（地区）共同构成四维企业贷款观察框架。
- S63_Ⅲ 的控股类型定义极为详细，区分绝对控股（>50%）和相对控股，并包含特殊规则（如投资双方各占50%的处理）。
- S63_Ⅲ 的逾期贷款分档（四档：60天以内/61-90天/91-360天/361天以上）与 S63 的七档分法不同，跨报表对比时需注意口径差异。
- S63_Ⅲ 的 [1.G]（个人经营性贷款余额）= S63_I[1.E]（小微型企业贷款余额）这一等式关系需要进一步确认业务含义——两者口径是否完全一致。
- S63_Ⅲ 包含 PPP 项目贷款和知识产权质押贷款两项特色统计，体现了监管对特定领域的追踪需求。
- S63_Ⅲ 的列 C1（私人控股-小微企业）是 S63_Ⅲ 独有的细分列，C ≥ C1 校验关系体现了私人控股企业中的小微企业子集统计。
- S63_I 中的控股类型列（A/B/C/D）在 S63_Ⅲ 中得到了完整的展开，S63_I 仅包含国有/集体/私人/港澳台商四类，S63_Ⅲ 增加了外商控股（E）和其他（F）。
