# 大集中系统-A1333_A2333-涉农贷款季报表

> [!note] 核心定位
> A1333 和 A2333 涉农贷款季报表是大集中系统中的核心统计报表，用于按季上报涉农领域的各项贷款余额及质量数据。它涵盖了农林牧渔业贷款、农户贷款、农村非金融企业及机关团体贷款以及城市非金融企业及机关团体涉农贷款四大板块，是评估金融机构乡村振兴支持力度和信贷资产质量的关键监管依据。

## 基本信息

- **报表编码**：A1333（法人/汇总级） / A2333（分支/独立级）
- **报表名称**：涉农贷款季报表
- **系统归属**：大集中系统
- **报送频度**：季报
- **原文依据**：[[01-资料库/大集中系统/2026-05-18-A1333_A2333-涉农贷款季报表-原文|A1333、A2333 涉农贷款季报表原文]]

## 指标体系与层级结构

本报表统计指标呈清晰的四树状层级结构，A1333 对应以 `13` 开头的指标编号，A2333 对应以 `23` 开头的指标编号。以下以 A1333（13系列）为例展示指标结构：

### 1. 农林牧渔业贷款情况 (13F01 / 23F01)
从事 GB/T 4754 门类 A（农、林、牧、渔业）所属活动的贷款。
- **13F02 农林牧渔业短期贷款**（期限 $\le$ 1年）
  - **13F08 正常贷款**：正常类（13F09） + 关注类（13F10）
  - **13F11 不良贷款**：次级类（13F12） + 可疑类（13F13） + 损失类（13F14）
- **13F15 农林牧渔业中长期贷款**（期限 > 1年）
  - **13F21 正常贷款**：正常类（13F22） + 关注类（13F23）
  - **13F24 不良贷款**：次级类（13F25） + 可疑类（13F26） + 损失类（13F27）

### 2. 农户贷款情况 (13G01 / 23G01)
发放给符合监管定义的农户的所有贷款。
- **13G02 农户短期贷款**（期限 $\le$ 1年）
  - **按担保方式划分**：
    - **13G08 信用贷款**（其中：13G09 农户小额信用贷款）
    - **13G10 保证贷款**（其中：13G11 农户联保贷款）
    - **13G12 抵押贷款**
    - **13G13 质押贷款**
  - **按质量分类划分**：
    - **13G14 正常贷款**：正常类（13G15） + 关注类（13G16）
    - **13G17 不良贷款**：次级类（13G18） + 可疑类（13G19） + 损失类（13G20）
- **13G21 农户中长期贷款**（期限 > 1年）
  - **按担保方式划分**：
    - **13G27 信用贷款**（其中：13G28 农户小额信用贷款）
    - **13G29 保证贷款**（其中：13G30 农户联保贷款）
    - **13G31 抵押贷款**
    - **13G32 质押贷款**
  - **按质量分类划分**：
    - **13G33 正常贷款**：正常类（13G34） + 关注类（13G35）
    - **13G36 不良贷款**：次级类（13G37） + 可疑类（13G38） + 损失类（13G39）

### 3. 农村非金融企业及机关团体贷款情况 (13H01 / 23H01)
注册地位于农村区域的非金融企业及机关团体的贷款。
- **13H02 农村非金融企业及机关团体短期贷款**（期限 $\le$ 1年）
  - **13H23 正常贷款**：正常类（13H24） + 关注类（13H25）
  - **13H26 不良贷款**：次级类（13H27） + 可疑类（13H28） + 损失类（13H29）
- **13H30 农村非金融企业及机关团体中长期贷款**（期限 > 1年）
  - **13H51 正常贷款**：正常类（13H52） + 关注类（13H53）
  - **13H54 不良贷款**：次级类（13H55） + 可疑类（13H56） + 损失类（13H57）

### 4. 城市非金融企业及机关团体涉农贷款情况 (13I01 / 23I01)
注册地位于城市区域，从事农林牧渔业活动或支持农业农村发展的贷款。
- **13I02 城市非金融企业及机关团体涉农短期贷款**（期限 $\le$ 1年）
  - **13I23 正常贷款**：正常类（13I24） + 关注类（13I25）
  - **13I26 不良贷款**：次级类（13I27） + 可疑类（13I28） + 损失类（13I29）
- **13I30 城市非金融企业及机关团体涉农中长期贷款**（期限 > 1年）
  - **13I51 正常贷款**：正常类（13I52） + 关注类（13I53）
  - **13I54 不良贷款**：次级类（13I55） + 可疑类（13I56） + 损失类（13I57）

## 核心概念与统计边界

### 1. 农户的判定边界
“农户”是指以“户”为统计单位的住户，其判定以**贷款发放时**承贷主体身份为准：
- **纳入范围**：
  - 长期（一年以上）居住在乡镇（不包括城关镇）行政区域内的住户；
  - 长期居住在城关镇所辖行政村范围内的住户；
  - 户口不在本地但在本地居住一年以上的住户；
  - 国有农场的职工、农村个体工商户；
  - 乡镇（不含城关镇）行政区内或行政村范围内的国有机关、学校、企事业单位集体户。
- **排除范围**：
  - 有本地户口，但举家外出谋生一年以上的住户（无论是否保留承包耕地）。

### 2. 农村区域 vs 城市区域
- **农村区域**：指除城市区域以外的区域。
- **城市区域**：包括地级及以上区域的城市行政区、市辖建制镇、县城城区、各类开发区。
  - *县城城区* 指县/县级市人民政府驻地所在的乡镇街道。
  - *开发区* 指国务院或省级政府批准设立的各类经济/高新开发区、园区。

### 3. 企业边界
包括依法在工商部门登记注册的各种经济组织（有限责任、股份、合伙、个人独资等）、非企业法人和农民专业合作社。**不包括**行政机关、事业单位、社会团体和个体工商户。

## 重点校验与表内逻辑关系

在涉农贷款数据报送中，表内各指标存在强校验关系：

### 1. 期限与质量核对关系
各板块的大类余额必须等于短期与中长期之和，且每一子项的质量分类必须轧平。
$$\text{大类总余额} = \text{短期贷款} + \text{中长期贷款}$$
$$\text{短期/中长期贷款} = \text{正常贷款} + \text{不良贷款}$$
$$\text{正常贷款} = \text{正常类} + \text{关注类}$$
$$\text{不良贷款} = \text{次级类} + \text{可疑类} + \text{损失类}$$

- *例如（农林牧渔业贷款）*：
  - $13F01 = 13F02 + 13F15$
  - $13F02 = 13F08 + 13F11$
  - $13F15 = 13F21 + 13F24$

### 2. 农户贷款的“质量 vs 担保”双维度轧平关系
在农户贷款板块（短期/中长期）中，由于同时采集了质量分类与担保方式分类，二者的统计总额必须完全相等。
$$\text{正常贷款} + \text{不良贷款} = \text{信用贷款} + \text{保证贷款} + \text{抵押贷款} + \text{质押贷款}$$

- *例如（农户短期贷款）*：
  - $13G02 = 13G14 + 13G17$ （质量口径）
  - $13G02 = 13G08 + 13G10 + 13G12 + 13G13$ （担保口径）
  - $\Rightarrow 13G14 + 13G17 = 13G08 + 13G10 + 13G12 + 13G13$ （强校验关系）

## 2024 年修订重点

1. **更名与统计范围调整**：
   - 原“农村企业及各类组织贷款”(13H01/23H01)更名为“农村非金融企业及机关团体贷款”；
   - 原“城市企业及各类组织涉农贷款”(13I01/23I01)更名为“城市非金融企业及机关团体涉农贷款”；
   - 相应调整了统计范围以契合非金融企业及机关团体口径。
2. **报送指标终止**：
   - 终止报送了部分原有的附报指标（瘦身监管负荷）。
