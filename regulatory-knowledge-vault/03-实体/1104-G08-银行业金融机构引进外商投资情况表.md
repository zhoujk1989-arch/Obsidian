# 1104-G08-银行业金融机构引进外商投资情况表

## 简介

G08 是 1104 系统中专门统计银行业金融机构引进外商投资情况的专项报表，用于监测境外投资者对银行业金融机构的投资规模、持股比例、留存收益等核心指标，并按机构类型汇总外商投资整体情况。

## 当前结论

- G08 是 1104 系统中专门衡量外商投资情况的报表，覆盖境外投资者详情（前20名）和附注合计（按机构类型）两层结构。
- 季报（法人汇总数据），报送时间为季报第二批次（季后18日）。
- 填报机构范围广泛：涵盖大型银行（含邮储银行）、股份制银行、城商行、民营银行、农商行、外资法人银行、外国银行分行及各类非银金融机构（财务公司、信托、金融租赁、汽车金融、消费金融、金融资产管理公司等）。
- 主表按报告期末持股比例排序，填报前20名境外投资者；报告期内曾有投资但期末余额为0的也需填报。
- 附注按机构类型（外商独资银行、中外合资银行、外国银行分行、引入境外投资者的中资商业银行、引入境外投资者的新型农村金融机构、引入境外投资者的非银行金融机构）汇总统计。
- G08 与 G01 形成表间校验：附注引进外资金额合计 = G01_[25.C]（实收资本），附注所有者权益合计 = G01_[59.C]。
- G08 与 G04 形成表间校验：附注注册资本合计 = G04_[13.A]。

## 重点口径

- **境外投资者排序规则**：按报告期末持股比例排序，填报前20名；报告期内曾有投资但期末余额为0的，填在存量投资者后，按当期利用外资变化金额排序。
- **境外上市机构特殊处理**：如无法获取具体投资人名单，将境外持股平台整体作为填报对手；如可获取，则按交易所持股名单参与前20排序，并在备查栏填写上市平台名称，同时从持股平台中扣除单独列示的投资者。
- **自由港排除规则**：D列（母公司所在国家/地区）不得填写英属维尔京群岛、开曼群岛、萨摩亚、毛里求斯、巴巴多斯、百慕大等自由港。
- **非认缴口径**：F列（引进外资金额）和I列（当年利用外资实际变化金额）均为实际投资金额，非认缴出资金额。
- **币种折算规则**：外币折算为人民币时，按报告期末最后一天国家外汇管理局公布的基准汇价折算；其他货币通过美元套算。

## 重点规则

- **附注机构类型六选一**：外商独资银行、中外合资银行、外国银行分行、引入境外投资者的中资商业银行、引入境外投资者的新型农村金融机构、引入境外投资者的非银行金融机构。填1的行需填报其他统计项目，填0的行其余项目均填0。
- **表内校验**：附注按机构类型（1.1-1.6）的C列合计 ≥ 主表前20名F列合计；附注F列合计 ≥ 主表前20名G列合计。
