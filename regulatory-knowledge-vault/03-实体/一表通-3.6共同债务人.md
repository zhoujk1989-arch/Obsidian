# 一表通-3.6共同债务人

## 简介

采集个人贷款违约90天及以上的共同债务人信息。同一客户有多个共同债务人的分别填列，个人贷款未发生违约的无须填列。不含信用卡违约。

## 关键信息

- 所属系统：一表通
- 报表编码：3.6
- 报表名称：共同债务人
- 当前原文依据：[[01-资料库/一表通系统/2026-04-27-3.6共同债务人-填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- 本表为2026年制度升级整表新增表，属于一表通系统中个人贷款违约相关的业务报表。
- 采集范围仅限"个人贷款违约90天及以上"的共同债务人，信用卡违约不纳入。
- 本表主键由关系ID、机构ID、采集日期三字段联合构成。
- 2026年制度升级中有3处字段变更：共同债务人证件类型由代码类改为文本类、借据ID长度从60扩充到100、新增备注字段。

## 重点口径

| 口径 | 说明 |
|------|------|
| 采集范围 | 个人贷款违约90天及以上的共同债务人，不含信用卡违约 |
| 多共同债务人 | 同一个客户有多个共同债务人的，分别填列 |
| 未违约处理 | 个人贷款未发生违约的，无须填列共同债务人 |
| 关系状态填报 | 当关系类型为多个时，关系状态分别填报，并用英文半角分号隔开 |

## 重点规则

| 规则 | 说明 |
|------|------|
| 主键构成 | 关系ID + 机构ID + 采集日期 |
