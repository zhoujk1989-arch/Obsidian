# 金融基础数据系统-JS_204_SPVFSX_特定目的载体投资发生额信息

## 简介

特定目的载体投资发生额信息，采集金融机构报告期内买卖的特定目的载体投资发生额，以特定目的载体投资交易为单位逐笔报送。

## 关键信息

- 所属系统：金融基础数据系统
- 报表编码：JS_204_SPVFSX
- 报表名称：特定目的载体投资发生额信息
- 报表大类：股权及特定目的载体投资
- 报表细类：特定目的载体投资
- 章节编号：3.6.2.2
- 当前原文依据：[[01-资料库/金融基础数据系统/2026-05-04-JS_204_SPVFSX_特定目的载体投资发生额信息-原文|原文页]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- JS_204_SPVFSX 属于 JS_204 系列（股权及特定目的载体投资大类），与 JS_204_CLGQTZ（存量股权投资信息）、JS_204_GQTZFS（股权投资发生额信息）、JS_204_SPVTZX（存量特定目的载体投资信息）共同构成 JS_204 系列。
- JS_204_SPVTZX 采集报告期末存续的特定目的载体投资（存量），JS_204_SPVFSX 采集报告期内买卖的特定目的载体投资发生额（流量），两者形成"存量 vs 流量"互补，特定目的载体投资覆盖从存量扩展至全量（存量+发生额）。
- 本表是金融基础数据系统首张特定目的载体投资发生额维度报表，也是 JS_204 系列第四张报表，使系统对特定目的载体投资业务的覆盖从"仅存量"扩展至"存量+发生额"全链条。
- 报送粒度为"以特定目的载体投资交易为单位逐笔报送"，与 JS_204_SPVTZX 的"以产品为单位逐笔报送"粒度不同：SPVFSX 以单笔交易为单位（同一产品可能有多笔买卖交易），SPVTZX 以产品为单位（每个产品一条记录）。
- 本表新增"交易方向"（买入/卖出）和"交易日期"字段，是 JS_204 系列中首张具有明确交易方向标识的发生额报表。

## 重点口径

1. **报送范围**：报告期内买卖的特定目的载体投资发生额，以特定目的载体投资交易为单位逐笔报送。不含报告期末存续但未发生交易的存量产品。
2. **特定目的载体类型**：10类码值覆盖银行理财、信托产品、券商资管、基金专户、期货资管、保险资管、金融资产投资公司资管、公募基金、私募基金及其他。99为兜底类。与 JS_204_SPVTZX 完全一致。
3. **运行方式**：4类码值区分开放/封闭与净值/非净值两种维度交叉。与 JS_204_SPVTZX 完全一致。
4. **交易方向**：1 表示买入（认购、申购），0 表示卖出（赎回）。是 JS_204 系列首张具有明确交易方向标识的报表。

## 重点规则

1. **发行人代码为多场景字段**：境内机构填统一社会信用代码（无则组织机构代码），境内个人填身份证件代码，已纳入资管统计制度的填资管产品统计编码，未纳入的填行业管理部门备案编码，境外机构填 LEI 码，境外个人/无 LEI 境外机构填自设编码。共 7 类场景。与 JS_204_SPVTZX 和 JS_204_CLGQTZ 高度一致。
2. **地区代码区分境内外**：境内用 GB/T 2260 县区级代码；境外前 3 位"000"填充 + 后 3 位 GB/T 2659 国别码。港澳台按境外客户填报，不得填报 810000/820000 等行政区划代码。与 JS_204_SPVTZX 和 JS_204_CLGQTZ 一致。
3. **资管产品统计编码优先级**：已纳入《金融机构资产管理产品统计制度》（银发〔2018〕299号）的境内 SPV 优先填统计编码；未纳入或暂未获取的填自设唯一编码。与 JS_204_SPVTZX 一致。
4. **交易金额折人民币**：必填项。外币按金融机构会计核算使用汇率折算，人民币直接填写。与 JS_204_SPVTZX 的"投资余额折人民币"规则一致。
5. **日期格式统一**：认购日期、到期日期、交易日期均为 YYYY-MM-DD 格式。本表比 JS_204_SPVTZX 多一个"交易日期"字段，反映单笔交易的实际发生时间。
