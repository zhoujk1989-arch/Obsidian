# 一表通-2.6客户财务信息

## 简介

采集上报机构授信客户（单一法人、集团、同业等）的最新财务指标。同业客户、个体工商户、小微企业主不填报本报表。对应 EAST 对公客户财务信息表（`IE_002_204`）。

## 基本信息

| 项目 | 内容 |
|------|------|
| 报表编码 | 2.6 |
| 报表名称 | 客户财务信息 |
| 所属系统 | 一表通 |
| 数据域 | 对公客户基础信息 |
| 字段总数 | 24 |
| 主键字段 | 客户ID（B060001）、机构ID（B060002）、财务报表编号（B060021）、采集日期（B060022） |
| 强制校验字段 | 客户ID、机构ID、对公客户名称、财务报表日期、是否审计、报表口径、币种、资产总额、负债总额、净利润、应收账款、其他应收款、流动资产合计、流动负债合计、报表周期、财务报表编号、采集日期 |
| 提示性校验字段 | 存货、现金流量净额、应收账款、其他应收款、流动资产合计、流动负债合计 |
| 非校验字段 | 审计机构、所得税、主营业务收入、报表录入日期、备注 |

- B060014 存货（提示）

- B060017 其他应收款（提示）

### 审计与报表信息（5 字段）

## 一表通与 EAST 差异要点

| 差异类型 | 涉及字段 | 说明 |
|----------|----------|------|
| 日期格式转换 | 财务报表日期(B060004)、采集日期(B060022) | 一表通 YYYY-MM-DD（10位），EAST YYYYMMDD（8位） |
| 码值转换（数字→文本） | 是否审计(B060005)、报表口径(B060007)、报表周期(B060020) | 一表通报数字码值，EAST报中文文本 |
| 机构ID拼接 | 机构ID(B060002) | 一表通需拼接金融许可证号前11位 |
| 新增字段 | 存货(B060014)、流动资产合计(B060018)、流动负债合计(B060019) | 1.0版新增，EAST无对应 |
| 2.1版新增 | 报表录入日期(B060023)、备注(B060024) | 2.1版本新增数据项 |
| 2.1版变更 | 报表口径(B060007)、报表周期(B060020) | 数据类别由"代码类"改为"文本类"，格式由"4!n"改为"anc..60" |

## 填报范围

- **纳入**：单一法人授信客户、集团授信客户、同业授信客户等授信客户的最新财务指标
- **排除**：同业客户（基础信息）、个体工商户、小微企业主（不填报本报表）
- **币种**：按分币种填报，多币种合计折算为人民币填报为 BWB

## 版本演进

| 版本 | 变更 |
|------|------|
| 1.0 | 基础版本，含 21 个字段；新增存货、流动资产合计、流动负债合计 3 个字段 |
| 2.1 | 新增报表录入日期、备注；报表口径和报表周期由代码类改为文本类 |
