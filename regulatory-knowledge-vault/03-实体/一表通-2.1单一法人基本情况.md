## 采集范围

采集业务范围：单一法人客户、集团客户授信中有表内外业务余额或授信额度的成员客户信息。同业客户、个体工商户不填报本报表。以对公名义开立的个体工商户填报本表，同业客户不填报本报表。

## 一表通与EAST差异要点

| 字段 | 差异说明 |
| --- | --- |
| B010026 对公客户类型 | EAST报文本码值，一表通报数字码值；2.1版数据类别由代码类改为文本类 |
| B010033 法定代表人证件类型 | 一表通比EAST码值分类更细，需码值映射；2.1版改为文本类 |
| B010036 财务人员证件类型 | 一表通比EAST码值分类更细，需码值映射；2.1版改为文本类 |
| B010042 上市情况 | 一表通为"上市公司标志-国别代码-上市公司代码"，EAST仅为是否标识 |
| B010044/B010046 评级结果 | 一表通拆分外部/内部评级，EAST合并为一个字段（优先填外部） |
| B010048 首次信贷关系年月 | 一表通统计客户首次获得银行业金融机构贷款的年月，EAST统计客户在我行的首次信贷年月 |
| B010025 行业类型 | 境外机构码值转换：一表通"9999"，EAST"境外" |
| B010023/B010060 日期格式 | 一表通YYYY-MM-DD，EAST YYYYMMDD |
| B010008 全球法人识别编码 | 2.1版本已删除 |
