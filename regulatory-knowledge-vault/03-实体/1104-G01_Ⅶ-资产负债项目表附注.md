# 1104-G01_Ⅶ-资产负债项目表附注

## 简介

G01_Ⅶ《资产负债项目表附注》是 1104 系统中 G01 主表的附注系列报表之一（第Ⅶ部分），主要用于反映填报机构贷款投向分行业情况。

**重要原则**：本表在填列贷款行业投向时，应根据**贷款资金的实际用途**，而不是借款人所属行业填报。

## 当前结论

- G01_Ⅶ 是 G01 附注系列中专门反映贷款投向的报表（第Ⅶ部分）。
- 核心原则：按贷款资金实际用途分类，而非借款人所属行业。
- 境内贷款细分为20个行业（按GB/T4754-2017）+ 个人贷款 + 买断式转贴现 + 买断其他票据类资产。
- 附注项目（高技术产业、数字经济、养老产业等）均为单位贷款，需按国家标准分类。
- 261 版明确：附注项目\[4\]（高技术产业）、\[5\]（数字经济核心产业）、\[6\]（数字化效率提升业）、\[7\]（知识产权密集型产业）均仅包含单位贷款。
- 261 版明确：\[6\] 数字化效率提升业中，一笔贷款不应同时属于数字经济核心产业贷款和数字化效率提升业贷款。

## 重点口径

### 报送口径

| 项目 | 内容 |
|------|------|
| 报送频率 | 月报（境内分支机构汇总），月后6日 |
| 填报币种 | 本外币合计人民币 |
| 数据单位 | 万元（保留两位小数） |

### 填报机构

政策性银行（含开发银行）、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、农村商业银行、村镇银行、外资法人银行、外国银行分行、民营银行、直销银行、农村合作银行、农村信用合作社、农村资金互助社、贷款公司、企业集团财务公司、金融租赁公司、汽车金融公司、消费金融公司。

### 贷款分类体系

#### 主体分类（3项）

| 编号 | 项目 | 说明 |
|------|------|------|
| [1] | 各项贷款 | 贷款、贸易融资、票据融资、融资租赁、买入返售、透支、垫款等 |
| [2] | 对境内贷款 | 使用地为境内（不含港澳台）的贷款 |
| [3] | 对境外贷款 | 使用地为境外（含港澳台）的贷款 |

#### 境内贷款细分（23项）

| 编号 | 项目 | 说明 |
|------|------|------|
| [2.1-2.20] | 20个行业分类 | 按GB/T4754-2017国民经济行业分类标准 |
| [2.10] | 金融业 | 对金融业发放的贷款，**不含同业拆放、贴现、转贴现** |
| [2.21] | 个人贷款（不含经营性） | 信用卡、汽车、住房按揭、其他 |
| [2.21.1] | 信用卡 | 个人信用卡透支垫款 |
| [2.21.2] | 汽车 | 向个人发放的购车贷款 |
| [2.21.3] | 住房按揭贷款 | 贷款用途为购买个人住房且以该住房为抵押 |
| [2.21.4] | 其他 | 除信用卡、汽车、住房按揭外的个人贷款 |
| [2.22] | 买断式转贴现 | 买断未到期商业汇票（银承/商承） |
| [2.23] | 买断其他票据类资产 | 福费廷转让、已议付信用证转让等 |

#### 附注项目（6类）

| 编号 | 项目 | 说明 |
|------|------|------|
| [4] | 高技术产业 | 按《高技术产业分类（2017/2018）》 |
| [5] | 数字经济核心产业 | 按《数字经济及其核心产业统计分类（2021）》 |
| [5.1] | 数字产品制造业 | 01大类数字产品制造业 |
| [5.2] | 数字产品服务业 | 02大类数字产品服务业 |
| [5.3] | 数字技术应用业 | 03大类数字技术应用业 |
| [5.4] | 数字要素驱动业 | 04大类数字要素驱动业 |
| [6] | 数字化效率提升业 | 05大类数字化效率提升业 |
| [7] | 知识产权密集型产业 | 按《知识产权（专利）密集型产业统计分类（2019）》 |
| [8] | 养老产业 | 按《养老产业统计分类（2020）》 |

### 重要说明

| 项目 | 说明 |
|------|------|
| 行业分类标准 | GB/T4754-2017《国民经济行业分类》 |
| 贷款用途判定 | 优先按贷款合同约定资金用途，无法确定时按借款人主营业务 |
| 附注项目限制 | 均为单位贷款，不包含个人贷款 |
| [2.21.3] 住房按揭 | 不包括以个人住房作抵押但用途不为购房的贷款 |
| [2.22] 买断式转贴现 | "买断"指无回购协议，转入方可自由支配票据 |
| [2.23] 买断其他票据 | "买断"指无回购协议，与是否有追索权无关 |
| [5]+[6] 关系 | 一笔贷款不应同时属于数字经济核心产业和数字化效率提升业 | 261版新增 |

## 重点规则

### 表内校验关系

```text
[1.] = [2.] + [3.]                                            各项贷款 = 境内 + 境外
[2.] = Σ(2.1-2.20) + [2.21] + [2.22] + [2.23]                对境内贷款 = 20行业 + 个人 + 买断
[2.21] = [2.21.1] + [2.21.2] + [2.21.3] + [2.21.4]           个人贷款 = 4细分之和
[5] = [5.1] + [5.2] + [5.3] + [5.4]                          数字经济核心产业 = 4子类之和
[4] ≤ [2], [5] ≤ [2], [6] ≤ [2], [7] ≤ [2], [8] ≤ [2]          附注项目 ≤ 对境内贷款
```

### 列校验关系

```text
A ≥ B    A列金额 ≥ B列金额（除[2.22][2.23]以外的各行）
```

### 表间校验关系

```text
[1.B] - [2.21B] ≥ S63_I[1.4.1A] + [1.4.1B] + [1.4.1C] + [1.4.1D] + [1.4.1E]
[1.A] = G01_[62.C]
[2.22A] = G01_III_[1.3.1C]
```
