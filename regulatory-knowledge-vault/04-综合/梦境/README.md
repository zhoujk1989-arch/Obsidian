# 梦境

这里存放系统对“你最近在关注什么、正在变成什么、下一步该往哪里走”的阶段性综合页。

## 定位

- 不是普通资料摘要
- 不是稳定事实页
- 是介于“日志”和“自我模型”之间的中间层

梦境页允许做较大胆的推断和联想，但必须把事实观察和推断明确分开。

## 使用方式

手动触发时，让 LLM：

- 读取首页、个人页、项目页、日志页
- 读取最近更新的资料页和主题页
- 生成一篇新的梦境页

推荐触发口令：

- “做一次梦”
- “做一次关于我的梦”
- “基于最近内容做一轮自我综合”

## 写入原则

- 自动创建新的梦境页
- 不直接修改 `我是谁`、`我的偏好`、`我的目标`、`当前学习地图`
- 如有稳定结论，先在梦境页中列出“建议更新的核心页”
