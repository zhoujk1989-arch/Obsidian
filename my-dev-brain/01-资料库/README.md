# 资料库
这里存放原始资料、导入文本和来源记录。
监管类资料在系统目录下直接存放原文页，结构化提炼统一回写到报表页、系统页或规则页。
`*-原文.md` 默认表示忠实转录页：除 Markdown 排版、表格转写和明显 OCR 断行修复外，不做摘要化改写；如果只能保存节选，必须显式标注 `摘录` 或 `节选`。
建议命名：
- `YYYY-MM-DD-标题.md`
当某一类资料持续增长时，可按系统或专题建立子目录，例如：
- `01-资料库/1104系统/`
- `01-资料库/反洗钱系统/`
建议流程：
1. 放入原始资料
2. 判断应更新的报表页、主题页或实体页
3. 把稳定结论回写到知识页
