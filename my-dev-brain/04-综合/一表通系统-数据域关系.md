# 一表通系统-数据域关系

> 本页承接一表通系统中“数据域之间如何挂接”的关系。
> 系统包含哪些表见 [[04-综合/一表通系统-报表关系]]；单表字段和口径见对应 `03-实体/一表通-*` 报表实体页；字段原文见 [[01-资料库/一表通系统/README]]。

## 当前定位

- 本页只写数据域挂接关系，不维护报表目录。
- 一表通的关系重点不是传统监管报表勾稽，而是机构、客户、产品、协议、交易、账务之间的主键和外键式挂接。

## 典型链路

| 场景 | 数据链路 |
|------|----------|
| 普通贷款 | 机构信息 → 客户信息 → 产品信息 → 贷款协议 → 信贷交易 |
| 房地产贷款 | 机构信息 → 个人/对公客户 → 贷款协议 → 房地产贷款协议 → 信贷交易 |
| 信用卡业务 | 机构信息 → 个人客户 → 卡产品 → 信用卡协议 → 信用卡交易 |
| 不良资产处置 | 机构信息 → 客户/协议 → 不良资产处置 → 可能关联 1104-G12 迁徙口径 |
