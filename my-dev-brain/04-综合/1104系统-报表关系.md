# 1104 系统报表关系

## G01 系列报表关系

G01 系列是 1104 系统的资产负债类报表家族，包含主表和多个附注表，形成"主表 + 附注"的填报结构。

### 报表层级

| 报表 | 编码 | 定位 | 核心视角 |
|------|------|------|----------|
| G01 资产负债项目统计表 | 金监统 G01 | 主表 | 静态余额全貌（本外币合计人民币） |
| G01_V 主要资产负债项目分币种情况表 | 金监统 G01_V | 附注 | 分币种（美元/欧元/日元/港币/英镑）列示 |
| G21 流动性期限缺口统计表 | 银监统0014号 | 独立报表 | 流动性期限结构（剩余期限划分） |

### 报表间关系

#### G01 与 G01_V

- **关系**：G01_V 是 G01 的附注性质补充表
- **关系类型**：补充细化（分币种）
- **数据逻辑**：
  - G01_V 单独列示的项目定义与 G01 主表一致
  - G01_V 未单独列示项目归入"其他资产/其他负债"
  - G01_V 是 G01 的"分维观察"，G01 是"汇总观察"
- **表间校验**：`G01_V[13.A-E 各币种合计] ≤ G01[25.B]`（资产总计）

#### G01 与 G21

- **关系**：G01 与 G21 是"静态余额"与"期限结构"的互补
- **关系类型**：维度互补
- **数据逻辑**：
  - G01 关注"有多少"（时点余额），按会计科目分类
  - G21 关注"什么时候变现"（期限匹配），按剩余期限划分
  - 两者资产端项目存在映射关系（如 G01[4.贷款] → G21[1.贷款]）
- **典型映射**：
  - G01[4.贷款] → G21[1.贷款]（按剩余期限分桶）
  - G01[8.投资] → G21[1.7 投资]（注意交易/银行账户区分）
  - G01[2.存放中央银行款项] → G21[1.1 存放中央银行款项]

### 三张报表的协同视角

| 维度 | G01 | G01_V | G21 |
|------|-----|-------|-----|
| 核心视角 | 静态余额全貌 | 分币种细化 | 流动性期限结构 |
| 货币单位 | 本外币合计人民币 | 分别折算为人民币 | 本外币合计人民币 |
| 维度数 | 1（汇总） | 5（币种_ A-E列） | 10（期限桶） |
| 业务重点 | 项目定义与核对关系 | 币种折算规则 | 期限划分与期限匹配率 |
| 报送频率 | 法人季报（季后13日） | 法人季报（季后18日） | 法人月报（月后13日） |

### 填报逻辑推导

当填报机构需要补充 G01_V 数据时：

1. 从 G01 获取各项目总额（人民币）
2. 拆分出美元、欧元、日元、港币、英镑的原始外币余额
3. 按期末外管局基准汇价进行折算（权益类项目除外）
4. 其他货币通过美元套算确定汇率
5. 对 G01_V 各币种列分别填列，确保表内校验成立
6. 校验 `G01_V[13.A-E合计] ≤ G01[25.B]`

## G01_V 填报要点

### 关键规则回顾

| 类别 | 规则 |
|------|------|
| 汇率确定 | 外币资产/负债：期末外管局基准汇价；权益类：历史汇率；其他货币：美元套算 |
| 保证金归属 | 单位/个人保证金统一归入 G01_V[21.其他负债] |
| 贸易融资边界 | 打包贷款、进口押汇、出口押汇、议付信用证、福费廷 → [5.] |
| 贴现业务 | 风险转移才调整余额，否则计入[19.]卖出回购款项 |
| 境外分支往来 | 境内合计口径下，借方余额填[11.]，贷方余额填[21.] |

### 与 G01 的差异对照

| 项目 | G01 填报方式 | G01_V 填报方式 |
|------|--------------|----------------|
| 现金 | 汇总人民币 | 按币种分列折算人民币 |
| 贷款 | 汇总人民币 | 按币种分列折算人民币 |
| 投资 | 汇总人民币 | 按币种分列折算人民币 |
| 其他资产 | 汇总人民币 | 按币种分列折算人民币 |
| 资产总计 | 本外币合计人民币 | 分币种列示后汇总 |
