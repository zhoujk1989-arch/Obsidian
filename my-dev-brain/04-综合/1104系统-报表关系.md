## 当前结论

- 1104 已经形成"系统页 → 报表页 → 报表实体页 → 原文页"的稳定结构，当前已纳入 30 张报表。
- G31《投资业务情况表》是 1104 系统中专门统计投资业务的报表，采用"底层资产+非底层资产穿透"双层结构，覆盖债券/权益/公募/私募/资管产品全类型。
- G31 的填报机构范围与 G44（杠杆率情况表）高度重合，两者法人口径报送频度均为季后13日。
- G31 第I部分统计直接持有的底层资产（债券/权益/公募），第II部分统计私募基金和资管产品并穿透反映最终投向，两者通过穿透汇总逻辑关联。
- G31_[9.A] = G01_[12.C]，与资产负债主表形成强校验关系。
- 非底层资产穿透后的债券、权益和公募基金投向，会同时体现在第I部分对应行中，形成"底层+穿透叠加"的汇总逻辑。
- G33_Ⅱ《银行账簿利率风险计量报表（标准化计量框架完整版）》是 1104 系统中衡量银行账簿利率风险的核心报表，基于重定价期限计算特定利率冲击情景下经济价值和净利息收入变动。
- G33_Ⅱ 的填报机构限定为大型商业银行（含邮储银行）和股份制商业银行，体现了利率风险计量对资本充足率较高的要求。
- G33_Ⅱ 引入了行为性期权（提前还款权/提前支取权）的标准化计量框架，需要计算基准和情景调整后的提前还款率/提前支取率。
- G33_Ⅰ 作为简化版，不要求对行为性期权进行情景调整建模，按合同约定现金流直接填报；[2.2.1] 核心存款暂不填报。
- G33_Ⅰ 与 G33_Ⅱ 形成互补覆盖：G33_Ⅰ 适用于政策性银行、城商行、民营银行、农商行、外资法人银行；G33_Ⅱ 适用于大型商业银行和股份制商业银行。
- G33_Ⅱ 的嵌入式利率期权剥离和衍生品复式记录法与 G4C-1(b)（市场风险简化标准法）存在交叉引用关系，后续需补充 G4C 系列报表实体页。
- G33_Ⅰ 的衍生品复式记录法同样引用 G4C-1(b)，且 G33_Ⅰ 原文未提及嵌入式利率期权剥离处理，与 G33_Ⅱ 存在差异。
- G32《外汇风险敞口情况表》是 1104 系统中唯一专门衡量外汇风险敞口的报表，采用短边法计算总敞口头寸，覆盖即期、远期、期权和结构性头寸全类型。
- G32 的填报机构范围与 G44（杠杆率情况表）高度重合，两者法人口径报送频度均为季后13日。

- G53 是 1104 系统中地区维度最全面的综合报表，涵盖机构网点、利润、贷款行业分布、资产质量、融资担保贷款、县域业务六大板块，与 S65（企业贷款地区分布）和 S26（城商行省外经营）形成互补。

- G53 第VI部分（县域业务）与 G01 形成多处表间校验：存贷款合计= G01_[61.C]/[62.C]，储蓄存款= G01_[27.C]，个人贷款= G05__[1.A]+G05__[2.A]。
- G53 的报送频度较为复杂：法人汇总数据季报（季后13日），第V部分半年报（半年后18日），第VI部分季报（季后18日）。
- G51 是 1104 系统中国别风险管理的核心报表，按国家/地区维度收集境外债权债务敞口、风险转移、拨备情况，覆盖31个核心字段，季报（季后40日），与 G52 同为季后40日报送的报表。
- G51 的核心逻辑是"直接境外债权 + 风险转移 = 最终境外债权"，通过风险转移净额反映实际的国别风险暴露。
- G51 与 G40 存在表间校验（资本净额），G40 尚未在知识库中建立实体页。
- G51 原文中【12.】应计提国别风险拨备比率的公式与文字说明存在不一致，公式值高于文字说明值，需监管确认。
- G44 是 1104 系统中衡量银行杠杆率的核心报表，基于巴塞尔 III 杠杆率框架，核心特点是非风险敏感性——它不依赖风险权重，而是以简单的总风险暴露作为分母，作为风险加权资本充足率的风险缓释手段。
- G44 杠杆率 = 一级资本净额 / (调整后表内资产 + 衍生产品资产 + 证券融资交易资产 + 表外业务调整后余额)。
- G44 与 G4A 存在强表间校验关系：一级资本净额和一级资本扣减项均从 G4A 取数，G4A 尚未在知识库中建立实体页，后续应优先补充。
- G44 与 G01 存在表间校验关系（法人口径下）：表内总资产和衍生产品资产从 G01 取数。
- G44 与 G4B-2 存在表间校验关系：表外各档转换系数后的资产均从 G4B-2 取数，G4B-2 尚未在知识库中建立实体页，后续应优先补充。
- G44 衍生产品部分（[3.1]-[3.7]）和证券融资交易部分（[4.1]-[4.4]）的计算规则复杂，涉及净额结算、抵质押品扣除、潜在风险暴露等，建议后续结合实际业务场景补充更多填报案例。
- G44 填报机构范围广泛，包括政策性银行、大型商业银行、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用社、村镇银行、外资法人银行、金融租赁公司、汽车金融公司、消费金融公司和金融资产管理公司。
- G44 报送频度为季报，但法人汇总数据（季后18日）和合并报表数据（季后40日）的报送时间不同，是跨口径数据对齐时需特别注意的点。
- 后续新增 1104 报表时，应先补本页的报表行和简述，再决定是否需要新增规则页。
