# 1104-G01_III-存贷款明细报表（一）

## 简介

G01_III《存贷款明细报表（一）》是 1104 系统中的存贷款分品种专项报表，核心功能是收集填报机构本外币存款、贷款分品种的情况，贷款按原始期限分类，存款按主体分类。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银监统0002号
- 报表名称：资产负债项目表附注第III部分：存贷款明细报表（一）
- 当前原文依据：[[01-资料库/1104系统/2026-04-08-G01_III-资产负债项目表附注-填报说明-原文|G01_III 存贷款明细报表（一）填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、规则和常见口径

## 当前结论

- G01_III 以人民币、外币折人民币、本外币合计人民币三列（A/B/C）填报
- 贷款分为七类：短期、中长期、贴现及买断式转贴现、贸易融资、融资租赁、各项垫款、其他贷款
- 存款分为五类：单位存款、个人存款、国库定期存款、临时性存款、其他存款
- 适用机构：14类机构（政策性银行、大型银行、股份制银行、城商行、农商行、外资法人银行等）

## 重点规则

### 表内校验关系

```text
[1.] = [1.1] + [1.2] + [1.3] + [1.4] + [1.5] + [1.6] + [1.7]  （七类贷款之和）
[1.4] = [1.4.1] + [1.4.2]    （贸易融资细分，适用于各列）
[1.6] = [1.6.1] + [1.6.2] + [1.6.3]    （各项垫款之和，C列）
[1.6.1] >= [1.6.1.1]    （等同于贷款的授信 >= 承兑汇票，C列）
[2.] = [2.1] + [2.2] + [2.3] + [2.4] + [2.5]    （五类存款之和）
[2.1] = [2.1.1] + [2.1.2] + [2.1.3] + [2.1.4] + [2.1.5] + [2.1.6] + [2.1.7] + [2.1.8]    （八类单位存款）
[2.1.6] >= [5]    （单位保证金 >= 银行承兑汇票保证金，各列）
[2.2] = [2.2.1] + [2.2.2] + [2.2.3] + [2.2.4] + [2.2.5] + [2.2.6] + [2.2.7] + [2.2.8] + [2.2.9]    （九类个人存款）
[2.2.2] >= [6]    （个人定期 >= 特定养老储蓄，A、C列）
[2.5] >= [2.5.1]    （其他存款 >= 保险公司存款，各列）
[A] + [B] = [C]    （本外币合计 = 人民币 + 外币）
```
