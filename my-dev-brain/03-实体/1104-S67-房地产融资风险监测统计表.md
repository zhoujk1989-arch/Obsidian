# S67-房地产融资风险监测统计表

## 简介

S67《房地产融资风险监测统计表》是金监统系列监管报表之一，用于全面监测银行业金融机构房地产融资风险敞口。报表覆盖房地产贷款全口径（地产开发、房产开发、商业用房购房、个人住房、其他房地产贷款）、表内非信贷资金投向、非保本理财资金投向、其他房地产抵押贷款，以及附注项目（民营房企、重点关注业务、提前还款、代销资管产品、保障性安居工程合计）。报表分第2部分专项统计租赁住房贷款，并包含五级分类、展期、逾期等资产质量维度。

## 当前结论

- S67 是监管层追踪房地产融资风险的核心报表，覆盖从开发端到消费端的全链条房地产融资。
- S67 为月报（月后6日），与 S71 表I/表II、S72 同属第一批次月报频度。
- S67 的填报机构范围广泛，涵盖政策性银行、大型商业银行、股份制银行、城商行、农商行、民营银行、外资银行等几乎所有银行类机构。
- S67 的结构复杂：第一部分涵盖 4 大板块（房地产贷款、表内非信贷资金、非保本理财资金、其他房地产抵押贷款）+ 附注项目；第二部分专项统计租赁住房贷款。
- S67 与 S63 存在明确关联：民营房企统计口径参照 S63_Ⅲ 中"私人控股"范围。
- S67 的逾期处置数据与 G12《贷款质量迁徙情况表》[15.1不良贷款处置总额]保持一致。
- S67 引入了 DSR（偿债收入比）和 LTV（贷款价值比）两个重要风险指标的定义。
- 原文中 [1.2.1] 的校验式写为 `[1.2.1.2.1]+[1.2.1.2.2]+[1.2.1.2.3]+[1.2.1.2.4]`，与正文已定义的 `[1.2.1.1]` 体系不一致，可判断为原文编号异常，且高度疑似笔误。

### 填报机构

政策性银行(含开发银行)、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用社和村镇银行、住房储蓄银行、外资法人银行、外资银行分行、直销银行。

### 房地产贷款定义

与地产开发或房产开发、出售、出租、消费等有关活动的贷款。附注项目中重点关注业务属于本部分的，亦应在此项目填报。

### 关键风险指标定义

**偿债收入比（DSR）：**
```
DSR = (本次贷款的月还款额 + 月物业管理费) / 家庭月均收入
```

**贷款价值比（LTV）：**
```
LTV = 报告期末贷款余额 / 房地产押品市场价值
```
- 填报机构应根据房地产押品价值的波动特性，合理确定价值重估频率，每年至少重估一次。

### 逾期填报规则

1. **按月分期还款贷款**：逾期90天以内，按已逾期部分本金余额填报；逾期90天及以上，按整笔贷款本金余额填报。
2. **借新还旧展期**：逾期天数从原借款合同到期日起算。
3. **正常展期**：逾期天数从展期后的到期日起算。
4. **本息都逾期**：按逾期时间长的期限填报。

### 住房套数区分

按照贷款发放时当地制定的差别化住房信贷政策区分。

### 民营房企统计口径

参照《S63_Ⅲ各控股类型企业融资情况表》中"私人控股"的统计范围。对公贷款包括房地产开发贷款、经营性物业贷款、房地产并购贷款等；个人住房按揭贷款仅统计新建住房，存量房交易不统计。

### 保障性安居工程贷款合计

包括：公共租赁住房贷款、廉租住房贷款、棚户区及垦区危房改造贷款、经济适用住房开发贷款、限价商品住房开发贷款、农村危房改造贷款、游牧民定居工程贷款和保障性租赁住房贷款。

### "保交楼"专项借款

开发性政策性银行发放的"保交楼"专项借款、商业银行发放的"专项借款配套融资"均在 [1.2.1] 住房开发贷款项下填报。

### 重点关注业务-逾期未交付项目

- 仅统计因**资金链断裂**导致的逾期项目（包括但不限于"保交楼"项目）
- 不包括因气象原因、政策原因等导致的逾期
- 按月更新；按月更新存在困难的，至少按每季度更新
- 非季度月末可与上季度月末数据保持一致
