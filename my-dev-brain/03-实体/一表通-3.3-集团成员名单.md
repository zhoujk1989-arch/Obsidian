# 一表通-3.3-集团成员名单

## 简介

3.3《集团成员名单》是一表通系统的集团成员关系表，用于维护集团ID与成员企业之间的映射关系、成员类型、母公司标识及授信敞口已用额度等信息。

## 关键信息

- 所属系统：[[03-实体/一表通系统|一表通系统]]
- 报表编码：3.3
- 报表名称：集团成员名单
- 当前原文依据：[[01-资料库/一表通系统/2026-04-16-3.3-集团成员名单-数据字典-原文|3.3 集团成员名单-数据字典-原文]]
- 知识沉淀位置：本页维护字段分组、关键字段与规则；字段全量清单见原文页

## 当前结论

- 本表是集团客户关系层的核心成员映射表，关键键可视为“关系ID”。
- 2.1 版新增“母公司标识、关联人类别、授信敞口已用额度”，说明成员关系已开始承担集团授信分析职能。

## 字段分组

| 分组 | 代表字段 | 说明 |
|------|----------|------|
| 关系识别 | 关系ID、集团ID、机构ID、成员ID、成员企业名称 | 用于集团与成员映射 |
| 成员属性 | 成员统一社会信用代码、成员类型、登记注册代码、母公司标识、关联人类别 | 用于成员主体属性识别 |
| 授信与状态 | 授信敞口已用额度、关系失效日期 | 用于额度和状态记录 |
| 扩展 | 备注、采集日期 | 用于补充说明与数据快照 |

## 重点规则

- 金额类字段 `20n(2)` 表示数值型金额字段，并保留两位小数。
- 母公司标识引入后，集团内部层级关系可直接在成员表中表达。
