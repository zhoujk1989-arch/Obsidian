# 1104-G18_III-地方政府自主发行债券持有情况统计表

## 简介

G18\_III 是 1104 系统中专门反映银行业金融机构持有地方政府自主发行债券情况的报表，按发行地区、资金用途、发行方式、利率水平、剩余期限等维度细分持有规模。

## 当前结论

- G18\_III 是 1104 系统中专门统计银行业金融机构持有地方政府自主发行债券的报表，核心口径为"新《预算法》和43号文实施以来"具有发行地方债券资格地区自主发行的地方政府债券，不包括43号文以前地方政府以各种方式发行的部分（财政部代发、地方政府自发自还、自发代还等）。
- 报表编码原文仅写"银监统 号"，未给出具体编号，需监管确认。
- 季报（法人汇总数据），季后18日报送，与 G23/G24/G25\_II/G27 等同属第二批次报送。
- 填报机构范围广泛，共15类：政策性银行（含开发银行）、大型银行（含邮储银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用合作社、村镇银行、外资法人银行、外国银行分行、企业集团财务公司、金融租赁公司、汽车金融公司、消费金融公司。
- G18\_III 与 G18《融资工具发行情况表》属于不同报表：G18 统计银行业金融机构自身发行的融资工具，G18\_III 统计持有的地方政府债券。
- G18\_III 与 G52《地方政府融资平台及支出责任债务持有情况统计表》形成互补：G52 监测地方政府隐性债务风险（融资平台），G18\_III 统计地方政府自主发行债券持有情况。

## 重点口径

- **统计范围**：仅含新《预算法》和国发〔2014〕43号文实施以来，具有发行地方债券资格的地区自主发行的地方政府债券。43号文以前地方政府以各种方式发行的部分（财政部代发、地方政府自发自还、自发代还等）不纳入。
- **一般债券 vs 专项债券**：一般债券依据财库〔2015〕64号，为没有收益的公益性项目发行，以一般公共预算收入还本付息；专项债券依据财库〔2015〕83号，为有一定收益的公益性项目发行，以政府性基金或专项收入还本付息。
- **利率水平**：比照基准国债平均收益率为发行日前1至5个工作日相同待偿期记账式国债的平均收益率。
- **专项债券配套融资**：按《关于做好地方政府专项债券发行及项目配套融资工作的通知》要求，为专项债券项目提供的配套贷款额，包括专项债券作为资本金和债务资金的项目。
- **币种折算**：各币种折合人民币填报，按报告期末最后一天外管局基准汇价折算，其他货币通过美元套算。

## 重点规则

- **表内行校验**：1+2+3+4+…36.= 37.（合计数），适用于 A、B、C…J、K 列。
- **表内列校验**：A=B+C（余额合计=一般债券+专项债券），A=D+E+F（按利率水平），A=G+H+I+J（按剩余期限）。
- **表间校验**：G18\_III\_[37.A] ≤ G31\_I\[1.2A\]（G18\_III 全部债券持有合计不超过 G31 投资业务表债券投资）。
