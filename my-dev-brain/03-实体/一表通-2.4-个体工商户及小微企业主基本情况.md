# 一表通-2.4-个体工商户及小微企业主基本情况

## 简介

2.4《个体工商户及小微企业主基本情况》是一表通系统的小微客户主数据表，用于维护个体工商户或小微企业主的客户识别、经营主体信息、经营属性、财务情况和信用关系等内容。

## 关键信息

- 所属系统：[[03-实体/一表通系统|一表通系统]]
- 报表编码：2.4
- 报表名称：个体工商户及小微企业主基本情况
- 当前原文依据：[[01-资料库/一表通系统/2026-04-16-2.4-个体工商户及小微企业主基本情况-数据字典-原文|2.4 个体工商户及小微企业主基本情况-数据字典-原文]]
- 知识沉淀位置：本页维护字段分组、关键字段与规则；字段全量清单见原文页

## 当前结论

- 本表是小微客户主数据表，关键键可视为“客户ID + 机构ID”，并在 2.0 版新增“经营户个人ID”后开始兼顾“客户主体 + 个人经营者”双层识别。
- 与 2.1《单一法人基本情况》相比，本表更强调经营者个人身份、经营地址和简化财务信息。
- 版本演进上，2.0 试用版新增经营主体名称、联系电话，2.0 版新增经营户个人ID，2.1 版新增失效日期和备注，说明小微客户识别与生命周期管理在持续补强。
- 采集日期 B040031 在原始清单中排在 2.1 版新增字段之后，但版本说明仍为 1.0 版，映射时应以字段编码和版本说明为准。

## 重点规则

- 日期类字段存在两种格式：`YYYY-MM-DD` 和 `YYYY-MM`；建模与校验时应区分精确日期字段与年月字段。
- 金额类字段 `20n(2)` 表示数值型金额字段，并保留两位小数。
- 代码类字段当前只有格式约束，缺少取值范围；后续应补代码表或制度说明。
- B040034“经营户个人ID”引入后，后续可与个人客户或经营者主体类资料形成映射关系。
- B040031 虽位于 2.1 版新增字段之后，但版本说明仍为 1.0 版，字段版本应以编码和版本说明为准。
