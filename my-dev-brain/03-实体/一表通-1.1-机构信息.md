# 一表通-1.1-机构信息

## 简介

1.1《机构信息》是一表通系统当前纳入知识库的第一张基础信息报表，用于维护机构标识、机构属性、机构地址、负责人信息以及版本新增字段。

## 关键信息

- 所属系统：[[03-实体/一表通系统|一表通系统]]
- 报表编码：1.1
- 报表名称：机构信息
- 当前原文依据：[[01-资料库/一表通系统/2026-04-16-1.1-机构信息-数据字典-原文|1.1 机构信息-数据字典-原文]]
- 知识沉淀位置：本页直接维护该报表的字段概览、重点规则和常见口径

## 当前结论

- 本表当前提供的是数据字典层信息，核心是字段编码、字段名称、数据类别、数据格式和版本说明。
- 当前字段可分为四组：机构唯一标识、机构属性、联系人与地址、版本新增字段。
- 版本演进上，2.0 版新增 3 个字段，2.1 版新增 3 个字段，说明该表仍处于持续扩展阶段。
- 采集日期字段 A010020 在原始清单中排在 2.1 版新增字段之后，但版本说明仍为 1.0 版，录入和映射时应以字段编码和版本说明为准，而不是以展示顺序为准。

## 重点口径

### 字段类别分层

| 类别 | 代表字段 | 说明 |
|------|----------|------|
| 机构唯一标识 | 机构ID、内部机构号、金融许可证号、统一社会信用代码 | 用于机构唯一识别与系统对接 |
| 机构属性 | 机构类型、机构类别、县域机构标识、机构层级、运营状态 | 用于机构分类和监管属性标记 |
| 地址与联系人 | 机构地址、负责人姓名、负责人工号、负责人联系电话、机构联系电话 | 用于联系信息维护 |
| 政策特色标识 | 科技支行标识、科技特色支行标识、科技金融专营机构标识、自贸区网点标识 | 用于专项属性识别 |
| 版本扩展字段 | A010021-A010026 | 反映 2.0/2.1 版新增需求 |

### 版本变化

| 版本 | 新增字段 | 说明 |
|------|----------|------|
| 1.0版 | A010001-A010020 | 基础机构信息框架 |
| 2.0版 | A010021-A010023 | 新增自贸区网点、承办行机构代码、机构层级 |
| 2.1版 | A010024-A010026 | 新增机构联系电话、科技投入金额、备注 |

## 重点规则

- 代码类、指示器类字段应优先依赖对应代码表或枚举说明，当前原文只提供格式约束，未提供取值范围。
- 日期类字段统一采用 `YYYY-MM-DD` 格式，后续落库与接口映射应保持一致。
- 金额类字段 `20n(2)` 表示数值型金额字段，并保留两位小数。
- 当前字段顺序中 A010020 置于 A010021-A010026 之后，后续做字段映射时不应按行序推断版本，而应按字段编码和版本说明处理。
