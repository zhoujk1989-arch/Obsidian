## 一表通与EAST差异要点

| 字段 | 差异说明 |
| --- | --- |
| B030002 机构ID | 一表通需拼接金融许可证号前11位+内部机构号，EAST为"金融许可证号+内部机构号" |
| B030037 对公客户类型 | EAST报文本码值，一表通报数字码值；需进行码值转换 |
| B030014 法定代表人证件类型 | 一表通比EAST码值分类更细，需码值映射；2.1版数据类别由代码类改为文本类，格式由4!n改为anc..60 |
| B030017 财务人员证件类型 | 一表通比EAST码值分类更细，需码值映射；2.1版数据类别由代码类改为文本类，格式由4!n改为anc..60 |
| B030030/B030032 评级结果 | 一表通拆分外部/内部评级，EAST合并为一个字段（优先填外部） |
| B030009/B030036 日期格式 | 一表通YYYY-MM-DD，EAST YYYYMMDD |
| B030033 首次授信日期 | 一表通为确定日期，EAST为授信年月 |
| B030006 SWIFT编码 | 当且仅当EAST的证件类别为"SWIFT编码"时，可参考 |
