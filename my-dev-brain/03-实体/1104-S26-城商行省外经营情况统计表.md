# S26-城商行省外经营情况统计表

## 简介

S26《城商行省外经营情况统计表》是 1104 系统监管报表之一，旨在推动城市商业银行坚守定位、立足本地、回归本源、专注主业，全面准确反映城商行跨区域经营情况。报表依据《中国银保监会办公厅关于加强城市商业银行跨区域经营监管的通知》（银保监办发〔2022〕74号）建立，包含第I部分（城商行省外经营情况统计表）和第II部分（城商行省外分支机构情况表），以**"法人机构所在省份"**为核心维度，展示城商行省外融资、存款、表外业务、理财投资等指标。季报（季后18日），与 S26_II 形成表间校验关系。

## 当前结论

- S26 是 1104 系统中以**"城商行跨区域经营"**为核心主题的专项报表，专门针对城市商业银行的省外经营情况进行统计监控。
- S26_I 和 S26_II 均为季报（季报第二批次，季后18日），报送频度晚于大多数 1104 报表（季后13日）。
- S26 的核心概念是**"省外"**：指城商行法人机构所在省份之外的地区，且**包括省内的计划单列市**（如浙江城商行在宁波设立的分支机构视为省内分支机构）。
- S26_I 的核心行结构为三层：[1] 各项贷款（按期限/担保方式/业务类型拆分）、[2] 债券投资（按期限/担保方式拆分）、[3] 省外融资合计（按贷款/债券投资/其他表内信用风险资产/表外业务拆分）。
- S26_I 的列维度为 A-R（至少 18 列），其中 A-C 列对应 S26_II 的省外合计 P-R 列，形成表间校验闭环。
- S26 包含多项特色统计：贸易融资、供应链融资、理财投资（母行理财，不含理财子公司）、非标投资、互联网存款（含第三方平台吸收存款）、小微企业贷款、关联授信、前十大非金融企业集团客户。
- S26 的"重点监测口径省外融资"指除贸易融资、供应链融资外，法人及分支机构所在省份以外的非金融企业融资（不含债券投资、理财投资）。
- S26 与 1104 多个报表存在口径引用关系：各项贷款/债券投资/表外业务与 1104 相关报表保持一致，贸易融资与 G01 口径一致，存款与 G01_III 口径一致，互联网存款与 G01_IV 口径一致。

## 重点口径

### 报送口径

| 项目 | 内容 |
|------|------|
| 报送频率 | 季报，季报第二批次（季后18日） |
| 报送口径 | 境内汇总数据 |
| 填报币种 | 本外币合计人民币 |
| 数据单位 | 万元、户 |
| 金额精度 | 保留两位小数 |
| 外币折算 | 按报告期末最后一天外管局基准汇价；其他货币通过美元套算 |

### 填报机构

城市商业银行。

### "省外"定义

- 省外经营：城商行在法人机构所在省份之外开展业务
- **省份包括省内的计划单列市**（关键规则）
- 例：浙江城商行在宁波设立的分支机构为省内分支机构
- 融资投向省份以资金最终使用地为准，如无法确认最终使用地，企业以融资企业注册地为准
- 融资含线上和线下渠道投放的资金

### 境外定义

- 境外：非中华人民共和国（不含港、澳、台地区）境内的地区

### 重点监测口径省外融资

- 除贸易融资、供应链融资外
- 法人及分支机构所在省份以外的非金融企业融资
- **不含**债券投资、理财投资

### 贸易融资与供应链融资

| 业务类型 | 口径依据 |
|------|------|
| 贸易融资 | 与 G01 报表口径一致 |
| 供应链融资 | 银发〔2020〕226号《关于规范发展供应链金融 支持供应链产业链稳定循环和优化升级的意见》 |

### 理财投资

- 理财投资余额为**母行理财**，不包括理财子公司

### 非标投资

- 与《中国银监会关于规范商业银行理财业务投资运作有关问题的通知》（银监发〔2013〕8号）相关定义保持一致

### 小微企业贷款

- 填报口径与 1104 保持一致
- 单户授信 1000 万元（含）以下不含票据融资

### 不良资产

- 与《商业银行金融资产风险分类办法》保持一致

### 存款口径

- 各项存款、单位存款、个人存款、保险公司存款：与 G01 报表第 III 部分存贷款明细表（一）一致
- 单位存款、保险公司存款来源省份以存款人注册地为准

### 互联网存款

- 与《中国银保监会办公厅 中国人民银行办公厅关于规范商业银行通过互联网开展个人存款业务有关事项的通知》（银保监办发〔2021〕9号）保持一致
- 通过第三方平台吸收存款与 G01_IV《资产负债项目表附注》相关定义一致
