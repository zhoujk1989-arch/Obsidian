# 1104-G32-外汇风险敞口情况表

## 简介

G32《外汇风险敞口情况表》统计填报机构报告期末持有外汇情况（包括因从事期权业务而持有的外汇），是 1104 系统中专门用于外汇风险监管的核心报表。采用短边法计算总敞口头寸，覆盖即期、远期、期权、结构性头寸等全类型外汇风险暴露。

## 当前结论

- G32 是 1104 系统中唯一专门衡量外汇风险敞口的报表，覆盖全部币种（含黄金）的即期、远期、期权和结构性头寸。
- 总敞口头寸采用"短边法"计算：分别加总所有币种净多头之和与净空头之和，取绝对值较大者作为总敞口头寸。
- 填报机构范围广泛：政策性银行、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用合作社、外资法人银行、外国银行分行、企业集团财务公司、金融租赁公司。
- 报送频度：季报，法人汇总数据为季后13日内，合并报表数据为季后40日内。
- 黄金在 G32 中视为外汇头寸处理（仅限简化标准法计量市场风险的填报机构），其他贵金属和商品不作为外汇。
- Delta 等值方法用于期权头寸计量：调整后期权头寸 = 本金 × delta（delta 由填报机构专用期权计价模型计算）。
- 结构性外汇头寸扣除需与《商业银行资本管理办法》保持一致，扣除期限至少为6个月。
- 报表采用境内外合并口径：J = F + H + I（法人口径 J = F + H，无海外分行时 J = F）。
- 外币折算为人民币的汇率规则：美元/欧元/日元/港币使用外管局基准汇价，其他货币通过美元套算。

## 重点口径

### 即期资产/负债口径（A/B栏）
- 包含资产负债表内即期的所有外汇资产和负债。
- **排除**：结构性敞口、未到交割日的现货合约、表外交易项目做市价重估引致的资产/负债。
- 有减值准备的资产以总额填报（如外币贷款填贷款总额，非扣除损失准备后的净额）。
- 即期定义：交割日为交易日后第2个工作日（T+2），不包括交易后未到交割日的交易，也不包括已到交割日但未能成功交收的交易。

### 远期买入/卖出口径（C/D栏）
- **包含**：远期外汇合约、外汇期货、外汇掉期、未到交割日的现货（即期）合约。
- **排除**：货币期权合约（期权填 E 栏）。
- 贷款承诺等或有项目原则上不计入，除非涉及外汇买卖或极可能成为坏账需提取专项准备。
- 回购/返售协议按风险转移实质判断：风险转移至买方则证券不填报，回购承诺填为远期购入；风险转移至填报机构则买入证券填为资产，返售承诺填为远期售出。

### 期权头寸口径（E栏）
- **仅**从事会产生外汇风险期权交易的机构填写。
- 采用 Delta 等值方法：调整后头寸 = 本金 × delta（delta 由专用期权计价模型计算）。
- 在即期或远期市场就期权头寸进行的套期，**不填 E 栏**，而填即期/远期栏。

### 结构性外汇头寸口径（G/K栏）
- 与《商业银行资本管理办法》保持一致：商业银行为保护资本充足率不受汇率变化影响而持有的外汇头寸。
- **可纳入**：（1）对外币计量的非并表关联企业投资；（2）对外币计量的并表子公司或分支机构投资；（3）境外分支机构/附属公司经营形成的权益项目（如外币留存收益）。
- **不可纳入**：固定资产和物业、无形资产等资产端项目；法人口径下采用成本法核算的外币长期股权投资对应的附属公司外币留存收益。
- 商业银行吸收的外币资本金如能证明保护资本充足率作用，可申请作为结构性外汇头寸扣除。

### 总敞口头寸口径（[12.]行）
- 采用**短边法**：比较净多头头寸之和与净空头头寸之和，取绝对值较大者。
- 黄金视为外汇之一参与比较。

## 重点规则

### 核对关系（表内校验）

| 校验公式 | 说明 |
|------|------|
| [F] = ([A]-[B]) + ([C]-[D]) + [E] | 境内单币种敞口 = 即期净敞口 + 远期净敞口 + 调整后期权头寸（适用于1-11行） |
| [J] = [F] + [H] + [I] | 合并单币种敞口 = 境内 + 海外分行 + 附属机构（适用于1-11行） |
| [12.F] ≥ 0 | 境内总敞口头寸为非负绝对值 |
| [12.J] ≥ 0 | 合并总敞口头寸为非负绝对值 |
| [10.J] ≥ 0 | 其他净额多头外币的合并敞口为非负 |
| [11.J] ≤ 0 | 其他净额空头外币的合并敞口为非正 |
| [12.F] = max(∑[1-11.F]正数, ∣∑[1-11.F]负数∣) | 境内总敞口头寸 = 短边法计算结果 |
| [12.J] = max(∑[1-11.J]正数, ∣∑[1-11.J]负数∣) | 合并总敞口头寸 = 短边法计算结果 |

### 境内外口径关系规则

| 场景 | J 栏计算方式 |
|------|------|
| 无海外分行和附属公司 | [J] = [F]，[K] = [G] |
| 法人口径（有海外分行但无附属公司） | [J] = [F] + [H]，不填 [I] |
| 合并口径（有海外分行和附属公司） | [J] = [F] + [H] + [I] |

### 汇率折算规则

1. **四种主要货币**（美元、欧元、日元、港币）：使用报告期末最后一天国家外汇管理局公布的基准汇价。
2. **其他货币**：通过套算确定——报告期末最后一天美元对人民币基准汇率 × 同一天上午9时国际外汇市场其他货币兑美元汇率。
