# 一表通-1.3-员工

## 简介

1.3《员工》是一表通系统的人员主数据表，用于维护员工基础身份信息、所属机构、岗位与职务信息、高管标识，以及入职/任职/离职等关键日期字段。

## 关键信息

- 所属系统：[[03-实体/一表通系统|一表通系统]]
- 报表编码：1.3
- 报表名称：员工
- 当前原文依据：[[01-资料库/一表通系统/2026-04-16-1.3-员工-数据字典-原文|1.3 员工-数据字典-原文]]
- 知识沉淀位置：本页直接维护该报表的字段概览、重点规则和常见口径

## 当前结论

- 本表是人员主数据表，核心主键是员工ID，关联键是机构ID。
- 字段结构可分为四组：身份与证件、联系方式、任职与岗位、状态与日期。
- 2.1 版新增字段为离职日期 A030029，用于补全人员生命周期。
- 采集日期字段 A030028 在原始清单中排在 2.1 版新增字段之后，但版本说明仍为 1.0 版，映射时应以字段编码和版本说明为准。

## 重点口径

| 口径点 | 说明 |

| 员工与机构关联 | A030002 关联 1.1《机构信息》中的机构ID |

| 风控字段 | 轮岗日期、强制休假日期常用于内控与合规管理场景 |

## 重点规则

- 日期类字段统一采用 `YYYY-MM-DD`，用于一致性落库与接口映射。
- 指示器类字段（高管标识）应依赖后续代码表或枚举说明，当前原文仅有格式约束。
- A030010“所属部门”格式为 `anc`，原文未给出长度上限，后续需要确认具体约束或系统实现限制。
