# 一表通-6.7贷款展期协议

## 业务定位

采集报送填报机构与客户签订的贷款展期协议。贷款展期是指贷款到期不能归还，经批准办理延长归还时间的贷款。

## 关键字段分组

| 分组 | 字段 | 说明 |
|------|------|------|
| 协议标识 | F070001 协议ID（PK） | 长度不超过60位数字与字母组合 |
| 借据关联 | F070002 借据ID（PK） | 唯一识别借据的编码 |
| 机构归属 | F070010 机构ID（PK） | 前11位=金融许可证号前11位，后13位=内部机构号 |
| 展期要素 | F070003 展期次数 | 延长归还时间的次数 |
| 原协议关联 | F070004 被展期贷款的贷款协议号 | 原贷款协议号 |
| 时间要素 | F070005 展期贷款的到期日期 | YYYY-MM-DD 格式 |
| 金额要素 | F070006 展期贷款的贷款金额 | 20n(2) 格式 |
| 用途要素 | F070007 展期贷款的贷款用途 | 文本类 |
| 利率要素 | F070008 展期贷款的贷款利率 | 20n(6) 百分比格式 |
| 采集标识 | F070009 采集日期（PK） | YYYY-MM-DD 格式 |
| 补充说明 | F070011 备注 | 2.1版新增，英文半角分号分隔 |

## 版本变更

| 版本 | 变更内容 |
|------|----------|
| 1.0版 | 协议ID、借据ID、机构ID、展期次数、被展期贷款协议号、到期日期、贷款金额、贷款用途、贷款利率、采集日期 |
| 2.0试用版 | 新增机构ID字段（F070010） |
| 2.1版 | 协议ID、借据ID数据格式长度延长；新增备注字段（F070011） |

## 填报规则

- 信用卡业务不纳入报送范围（信用卡协议、信用卡交易、信用卡账户状态、信用卡分期状态已覆盖信用卡业务相关信息）。
- 协议ID、借据ID、机构ID、采集日期共4个主键字段，均须强制性校验。
- 展期次数、被展期贷款协议号、到期日期、贷款金额、贷款利率均为强制性校验。
- 展期贷款用途为可选字段。

## 关联关系

- 关联 `6.2贷款协议`：展期贷款对应原贷款协议
- 关联 `6.27贷款协议补充信息`：同属贷款协议信息域
- 关联 EAST `IE_005_501` 信贷合同表：展期协议属于信贷合同范畴
- 关联 EAST `IE_005_503`/`IE_005_504` 信贷借据表：通过借据ID关联
