# 一表通-6.1存款协议

## 概述

| 项目 | 内容 |
|------|------|
| 报表编码 | 6.1 |
| 报表名称 | 存款协议 |
| 所属系统 | 一表通 |
| 业务域 | 协议类（存款协议） |
| 数据表映射 | 个人存款分户账 / 对公存款分户账 |
| 字段总数 | 32 |
| 主键 | F010001 协议ID、F010002 机构ID、F010035 采集日期 |

## 采集范围

填报机构与客户签订的所有存款协议。表内外全口径存款协议，包括委托存款、保证金存款、大额存单等。报数期开始时间前已经销户终结的存款协议，无需填报。银行在内部账户进行核算的存款需按协议进行拆分填报。

## 版本变更要点（2.1版本）

1. 多个字段由"代码类"改为"文本类"：F010004客户类型、F010008存款账户类型、F010018利率定价基础、F010023钞汇类别、F010026账户资金控制情况、F010027协议状态、F010048存款产品类别
2. 删除字段：F010013交易介质、F010015交易介质ID、F010036-F010047（借记卡/存折/其他介质相关）
3. 自定义码值统一改为"00-XX 其他-银行自定义"
4. F010048新增码值"23 委托存款"
5. F010001协议ID长度延长至100
