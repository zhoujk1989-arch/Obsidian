# S74-数字金融情况表

## 简介

S74《数字金融情况表》是金监统系列监管报表之一，用于监测和反映银行业保险业数字化转型的落地成效。报表内容涵盖数字经济产业贷款情况和金融服务数字化转型两大核心维度。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统S74
- 报表名称：数字金融情况表
- 当前原文依据：[[01-资料库/S74系统/2026-04-20-S74数字金融情况表-填报说明-原文|S74数字金融情况表填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- S74 是监管层追踪数字金融发展情况的核心报表，覆盖"产业端"（数字产业贷款）和"服务端"（金融服务数字化转型）两个维度。
- 第I部分数字经济产业贷款与 1104 系统 G01_Ⅶ 的数字经济相关项目存在完全对应的表间校验关系，形成跨系统数据一致性约束。
- 数字经济产业分类依据国家统计局《数字经济及其核心产业统计分类（2021）》（令第33号），与 G01_Ⅶ 口径一致。
- 第Ⅱ部分金融服务数字化转型指标涉及金融科技投入、移动端客户覆盖率和线上业务替代率。
- S74 为季报，季后13日报送，填报机构范围广泛，覆盖主要银行类金融机构及非银金融机构。

| 项目 | 内容 |

| 金额精度 | 保留两位小数 |

### 填报机构

政策性银行(含开发银行)、大型银行（含邮储银行）、股份制商业银行、城市商业银行、农村商业银行、村镇银行、外资法人银行、外国银行分行、民营银行、直销银行、农村合作银行、农村信用合作社、企业集团财务公司、金融租赁公司、信托公司。

### 数字经济产业分类

- 分类标准：国家统计局《数字经济及其核心产业统计分类（2021）》（令第33号）
- 分类层级：数字经济产业 → 数字经济核心产业（01-04大类）+ 数字化效率提升业（05大类）
- 带*行业：仅对应国民经济行业分类小类中的部分活动，需根据贷款实际用途逐笔判断
- 互斥规则：一笔贷款不应同时属于数字经济核心产业贷款和数字化效率提升业贷款

### 金融科技投入口径

```
上一年度金融科技投入金额 = 基础设施建设投入 + 电子设备采购投入 + 软件采购投入 + 系统开发项目投入 + 系统运维投入 + 信息科技咨询投入 + 信息科技人力资源费用 + 其他投入
```

- 非一次性投入：只统计上一年度实际投入部分
- 与 T-Q-1 的关系：[5.总业务笔数] 同 T-Q-1 主要电子交易替代率-全行所有交易笔数；[5.1.线上业务笔数] 同 T-Q-1 主要电子交易笔数-合计

## 重点规则

### 表内校验关系

**行校验：**

```text
[1.A] ≤ [1.1.A] + [1.2.A]                              数字经济产业户数 ≤ 核心产业 + 效率提升业
[1] = [1.1] + [1.2]，适用于 B-F 列                       数字经济产业 = 核心产业 + 效率提升业（金额/余额/利息）
[1.1.A] ≤ [1.1.1.A] + [1.1.2.A] + [1.1.3.A] + [1.1.4.A]  核心产业户数 ≤ 4子类之和
[1.1] = [1.1.1] + [1.1.2] + [1.1.3] + [1.1.4]，适用于 B-F 列  核心产业 = 4子类之和
[2.1.A] ≤ [2.A]                                         移动端个人客户 ≤ 个人客户总数
[3.1.A] ≤ [3.A]                                         移动端法人客户 ≤ 法人客户总数
[5.1.A] ≤ [5.A]                                         线上业务笔数 ≤ 总业务笔数
```

**列校验：**

```text
[C] ≥ [D]                                               贷款余额 ≥ 中长期贷款余额
[C] ≥ [E]                                               贷款余额 ≥ 不良贷款余额
```
