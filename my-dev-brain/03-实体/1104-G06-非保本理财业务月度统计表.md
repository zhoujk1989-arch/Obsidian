# 1104-G06-非保本理财业务月度统计表

## 简介

G06 是 1104 系统中专门统计银行业金融机构非保本理财业务情况的月度报表，分为表I（产品情况）和表II（资产负债情况）两部分，全面覆盖理财产品的募集、存续、兑付、收益以及理财资金的实际投向和负债结构。

- 当前原文依据：[[01-资料库/1104系统/2026-04-22-G06-非保本理财业务月度统计表-填报说明-原文|原文]]

## 当前结论

- G06 是 1104 系统中专门统计非保本理财业务的月度报表，不包括理财子公司（理财子公司数据填入 G06(a) 表）。
- 报表分为表I（产品情况）和表II（资产负债情况）两部分，分别反映理财产品的基本信息和资产负债投向。
- 月报（月后6日），填报机构覆盖政策性银行、大型商业银行、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用合作社、村镇银行、外资法人银行、外国银行分行等。
- 表I 按募集方式（公募/私募）、投资性质（固定收益/权益/商品及衍生品/混合）、运作模式（封闭式净值型/开放式净值型/封闭式非净值型/开放式非净值型）、期限、投资者持有情况、币种等多维度分类统计。
- 表II 按理财资金投向（15大类资产+负债）统计穿透前和穿透后的余额，覆盖现金及存款、同业存单、拆放同业、债券、非标债权、权益类、衍生品、QDII、商品、另类、公募/私募基金、资管产品、委外投资等。
- G06 与 S73（养老金融情况表）第III部分养老理财产品存在表间校验关系（S73 ↔ G06a_I，G06a_I 尚未在知识库中建立）。
- G06 与 G01_I_1（表外业务情况表）存在表间校验关系（G01_I_1 原文提及与 G06_I 的关联）。
- 校验关系极为丰富：表I 有 14 条校验关系（含行项目汇总、分项细分、跨维度相等），表II 有 60+ 条校验关系（含穿透前后、合规新产品、现金管理类、投向汇总等）。
- "合规新产品"（D1/E1/C/D）和"现金管理类理财产品"（E/F）是 G06 的特色统计维度，分别对应资管新规合规情况和现金管理类产品专项统计。
- 表II 的穿透逻辑是核心：穿透前统计直接持有的底层资产，穿透后统计通过私募基金、资管产品、委托投资——协议方式三类资产间接持有的最终底层资产和负债。
- 表II 设有多个专项统计附注：投向地方政府融资平台部分、PPP 项目、市场化债转股、逾期资产、投向房地产业、创业投资基金和政府出资产业投资基金、银行资本补充工具、穿透后投向非标债权的行业归属、待整改资产、非标转标资产。

## 重点口径

- **理财子公司排除**：理财子公司的情况不在本表反映，应填入 G06(a) 表。这是 G06 与 G06(a) 的核心区别。
- **穿透前 vs 穿透后**：穿透前统计直接持有的底层资产、私募基金、资管产品及委托投资——协议方式；穿透后统计通过私募基金、资管产品、委托投资——协议方式三类资产按照穿透原则间接持有的最终底层资产和负债。
- **合规新产品定义**：完全符合《关于规范金融机构资产管理业务的指导意见》（银发〔2018〕106号）、《商业银行理财业务监督管理办法》（银保监会令2018年第6号）等监管规定的理财产品。
- **开放式产品发行统计**：只有首次发行才纳入"本期发行产品数"统计，后续开放期间认购不做统计。
- **收益统计限制**：G（客户端实现收益总额）和 H（银行端实现收益总额）仅按募集方式划分和按投资性质划分需填写。
- **现金管理类产品细分**：开放式净值型下进一步细分为"已完成整改现金管理类产品"和"采用摊余成本法进行核算的现金管理类产品"。
- **外币折算规则**：按照报告期末最后一个交易日国家外汇管理局公布的人民币兑美元、欧元的基准汇价进行折算；美元、欧元、日元以外的其他货币对人民币的折算汇率，以报告期末最后一个交易日美元兑人民币的基准汇率与同一天上午9时国际外汇市场其他货币兑美元汇率套算确定。
- **非标债权定义**：符合资管新规和《标准化债权类资产认定规则》（人民银行 银保监会 证监会 外汇局公告〔2020〕第5号）规定的标准化债权类资产之外的债权类资产。
