# 一表通-2.5-个人客户基本情况

## 简介

2.5《个人客户基本情况》是一表通系统的个人客户主数据表，用于维护个人客户身份识别、人口属性、联系方式、工作信息、收入情况、黑名单状态及涉农特征等信息。

## 关键信息

- 所属系统：[[03-实体/一表通系统|一表通系统]]
- 报表编码：2.5
- 报表名称：个人客户基本情况
- 当前原文依据：[[01-资料库/一表通系统/2026-04-16-2.5-个人客户基本情况-数据字典-原文|2.5 个人客户基本情况-数据字典-原文]]
- 知识沉淀位置：本页维护字段分组、关键字段与规则；字段全量清单见原文页

## 当前结论

- 本表是个人客户主数据表，关键键可视为“客户ID + 机构ID”。
- 与 2.1/2.4 相比，本表更强调自然人身份、家庭/工作属性、黑名单记录和涉农属性。
- 版本演进上，2.0 版新增“个人客户行政区划”，2.1 版新增“新型农业经营主体标识、失效日期、备注”，说明个人客户主数据在行政区划和涉农识别方面持续增强。
- 采集日期 B050036 在原始清单中排在 2.1 版新增字段之后，但版本说明仍为 1.0 版，映射时应以字段编码和版本说明为准。

## 重点规则

- 日期类字段存在两种格式：`YYYY-MM-DD` 和 `YYYY-MM`；建模与校验时应区分精确日期字段与年月字段。
- 金额类字段 `20n(2)` 表示数值型金额字段，并保留两位小数。
- 代码类、指示器类字段当前只有格式约束，缺少取值范围；后续应补代码表或制度说明。
- B050036 虽位于 2.1 版新增字段之后，但版本说明仍为 1.0 版，字段版本应以编码和版本说明为准。
