# 1104-G19-重点关注行业贷款统计表

## 简介

G19《重点关注行业贷款统计表》是 1104 系统中专门统计重点关注行业贷款投向及信用风险质量的报表，覆盖战略性新兴产业（9类）和文化产业领域，按行业分类×五级分类维度展示贷款余额及资产质量。

- 所属系统：[[03-实体/1104系统|1104系统]]

- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- G19 是 1104 系统中专门监测"重点关注行业"贷款投向及资产质量的报表，聚焦战略性新兴产业和文化产业两大领域。
- 填报频度为季报（季后18日），与 G22/G24/G25_Ⅱ/G27 等同属第二批次报送。
- 填报机构范围广泛：政策性银行、大型商业银行（含邮储）、股份制商业银行、城商行、民营银行、农商行、外资法人银行、外国银行分行、直销银行。
- 贷款投向以"资金实际用途"为第一原则，而非借款人所属行业。
- 战略性新兴产业按《战略性新兴产业分类(2018)》分为9类（节能环保、新一代信息技术、生物、高端装备制造、新能源、新材料、新能源汽车、数字创意、相关服务）。
- 文化产业按《文化及相关产业分类（2018）》填报。
- 同一笔贷款只能归属9类战略性新兴产业之一，跨类型的按最主要投向填报。
- 表间校验关系：G19[A] ≥ G19[B]+...+G19[J]（战略性新兴产业合计≤贷款总计），G19[A] ≥ G19[K]（文化产业合计≤贷款总计）。

## 重点口径

### 贷款口径

- 贷款指填报机构对企业融出货币资金形式的资产，主要包括：贷款、贸易融资、贴现、融资租赁、从非金融机构买入返售资产、透支、各项垫款等。
- **不含**买断式转贴现和买断其他票据类资产。

### 行业投向确定原则

- **第一原则**：根据贷款合同中约定的资金用途填报（按实际用途）。
- **第二原则**：如果无法合理确定贷款投向，则按借款人主营业务所属行业填报。
- 个人经营性贷款也按上述原则填报。

### 产业领域分类

- **战略性新兴产业**：9类，分别对应 B~J 列。
- **文化产业**：对应 K 列。
- 带括弧的行业名称：仅括弧中的活动属于文化及相关产业。

## 重点规则

### 战略性新兴产业填报规则

- 同一笔贷款只能填报9类战略性新兴产业之一。
- 同一笔贷款投向不同类型的战略性新兴产业，应整体纳入其中**最主要投向**填报。
- 同一行业对应不同类型的战略性新兴产业产品时，贷款余额应按产品归类分配到不同产业。

### 703行合计规则

- 703行"合计"中，从 B 列（节能环保）到 K 列（文化产业），合计数等于 1-702 行加总减去没有字母的单元格。
- 具体而言：
  - B 列合计数 = 每列白色单元格的数的加总 - B155 非金属矿物制品业项下的行业小类
  - C、E、F、H、I、J 列合计数 = 每列白色单元格的数的加总
  - D 列合计数 = 每列白色单元格的数的加总 - D654 医院项下的行业小类
  - G 列合计数 = 每列白色单元格的数的加总 - G178 常用有色金属冶炼、G184 贵金属冶炼和 G188 稀有稀土金属冶炼项下的行业小类

### 五级分类口径

- 主要依据《贷款风险分类指引》、《商业银行金融资产风险分类办法》。
- 正常类：债务人能够履行合同，没有客观证据表明不能按时足额偿付。
- 关注类：存在不利因素，但债务人目前有能力偿付。
- 次级类：债务人无法足额偿付，或金融资产已发生信用减值。
- 可疑类：债务人已经无法足额偿付，金融资产已发生显著信用减值。
- 损失类：采取所有可能措施后，只能收回极少部分或损失全部。
