# S63-大中小微型企业贷款情况表

## 简介

S63《大中小微型企业贷款情况表》是 1104 系统监管报表之一，也是整个"规模×地区×行业"三维企业贷款校验框架的**核心枢纽**。报表旨在收集各银行在报告日对各类企业贷款的有关情况，包含第I部分（大中小微型企业贷款情况，月报）和第II部分（大中小微型企业不良贷款情况，季报）。S63 覆盖企业贷款的全维度统计：按规模（大/中/小/微/个体）、按信用类型（对公/对私）、按五级分类、按担保方式、按逾期期限、按政策类贷款（专精特新/战略性新兴产业/技术改造/银税合作/无还本续贷/循环贷）等。

## 当前结论

- S63 是整个 1104 企业贷款报表家族的**校验枢纽**，与 S64（行业维度）、S65（地区维度）、S71（普惠金融维度）形成紧密的表间校验网络。
- S63 第I部分为月报（月后6日），第II部分为季报（季后13日），两部分报送频度不同。
- S63 的"企业"定义排除了行政机关、事业单位、社会团体、个体工商户等，个体工商户和小微企业主贷款单独列为[1.1.5]和[3.2]。
- S63 第I部分的核心行结构为两层：按规模维度（[1.1.1]-[1.1.5]）和按信用类型维度（[1.2.1]-[1.2.4]），两层合计均为 [1]。
- S63 第I部分包含 16 个一级行（[1]-[16]），其中 [1] 为贷款合计，[14] 为表内其他授信，[15] 为授信合计（[1]+[14]）。
- S63 第I部分列维度为 A-G（7列），覆盖当年累计发放户数、当年累计发放金额、贷款余额、信用贷款余额、小微型贷款余额、其中：小型、其中：微型。
- S63 第II部分为不良贷款分规模统计，5 个一级行（[1]-[5]），对应大型、中型、小型、微型、个人经营性贷款的不良情况。
- S63 包含详尽的中小微企业划型标准附件（30个行业×2-3个指标），是 S63/S64/S65 企业划型的统一依据。
- S63 与 G01_Ⅱ 形成表间校验：S63 各行业/规模贷款合计 ≤ G01_Ⅱ 各项贷款对应项目。
- S63 与 G01_I_1 形成表间校验：S63[4.1]-[4.4]（银行承兑汇票/跟单信用证/保函/贷款承诺）分别与 G01_I_1 对应项目校验。
- S63 第II部分与第I部分存在内部校验：第II部分合计 = 第I部分不良贷款合计（[1.1.3]+[1.1.4]+[1.1.5]）。

### 填报机构

政策性银行（含开发银行）、大型商业银行（含邮政储蓄银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、外资法人银行、外国银行分行、直销银行、农村信用社、村镇银行、贷款公司、农村资金互助社、企业集团财务公司、信托公司、金融租赁公司、汽车金融公司。

### 企业定义

- 在中华人民共和国境内依法设立、在工商部门登记注册、实行自主经营自负盈亏的各种经济组织
- 包括：有限责任公司、股份有限公司、非公司企业法人、合伙企业、个人独资企业以及其他经营单位
- **排除**：行政机关、事业单位、社会团体、个体工商户等

### 企业划型标准

- 按照工信部联企业〔2011〕300号《中小企业划型标准规定》执行
- 具体划分标准参考国家统计局最新标准（见附件30个行业划型表）
- 企业法人按上述标准执行，企业非法人按授权法人执行
- 新设立企业依据可研报告预计指标划定；无法划型的暂划入中型企业
- 已停产清算中企业参考停产前指标划型
- 企业划型变更时：余额/户数/不良数据按最新划型调整，累放数据仍在原分类

### 贷款口径

- 主要包括：贷款、贸易融资、贴现、融资租赁、从非金融机构买入返售资产、透支、各项垫款等
- 项目1-13不含买断式转贴现
- 贴现：对客户办理的商业票据贴现所融出的资金
- 买断式转贴现：从其他金融机构买断未到期已贴现商业汇票（无回购协议）
- 贸易融资：打包贷款、进口押汇、出口押汇、议付信用证和福费廷等，不含贴现和买断式转贴现

### 担保方式统计原则

- 若贷款存在多种担保方式，遵循**抵（质）押担保方式优先**原则
- 政府性融资担保公司保证贷款：政府性融资担保公司提供保证担保但未要求反担保措施的贷款
- 新型抵质押类贷款：知识产权、股权（股票）、仓单、存货、保单为抵质押物的贷款
- 关系人保证贷款：除关系人保证外无抵质押及其他第三方担保的贷款

### 当年累放贷款金额口径

- 根据企业划型实时调整计算当期各规模企业当年累计发放情况
- 企业划型变更时：划型变更前的累放数据仍在原分类，变更后发生的累放数据放入新分类
- 当年累放贷款年化利息收益 = 放款金额 × 合同年化利率 × (12/实际月数)
- 仅统计当年发放贷款的利息收益，往年发放贷款不计入

### 无还本续贷定义

- 对贷款到期后仍有融资需求又临时存在资金困难的债务人
- 贷款到期前经其主动申请，银行业金融机构提前开展调查和评审
- 在原贷款到期前签订新的借款合同，以新发放贷款结清已有贷款
- **不包括**：贷款到期后的重组、贷款展期等情况

### "专精特新"中小企业定义

- 工信部认定的专精特新"小巨人"企业
- 各省、自治区、直辖市及计划单列市、新疆生产建设兵团中小企业主管部门认定的"专精特新"中小企业
- 与 S70[4."专精特新"中小企业] 含义一致

### 战略性新兴产业贷款

- 包含九类：节能环保、新一代信息技术、生物、高端装备制造、新能源、新材料、新能源汽车、数字创意、相关服务
- 与 G19[战略性新兴产业] 含义一致，仅统计 G19 相应行业中投向战略性新兴产业的贷款
