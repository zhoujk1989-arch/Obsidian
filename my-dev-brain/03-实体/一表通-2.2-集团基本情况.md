# 一表通-2.2-集团基本情况

## 简介

2.2《集团基本情况》是一表通系统的集团客户主数据表，用于维护集团主体识别、母公司映射、注册地址与办公地址、授信类型、内部评级以及授信敞口等信息。

## 关键信息

- 所属系统：[[03-实体/一表通系统|一表通系统]]
- 报表编码：2.2
- 报表名称：集团基本情况
- 当前原文依据：[[01-资料库/一表通系统/2026-04-16-2.2-集团基本情况-数据字典-原文|2.2 集团基本情况-数据字典-原文]]
- 知识沉淀位置：本页维护字段分组、关键字段与规则；字段全量清单见原文页

## 当前结论

- 本表是集团客户主数据表，关键键可视为“集团ID + 机构ID”。
- 字段结构可分为四组：集团识别与母公司映射、地址与更新时间、授信与风险、生命周期与扩展。
- 版本演进上，2.0 试用版新增“母公司客户ID”，说明集团主数据开始与客户主数据建立显式关联；2.1 版进一步补充授信敞口额度、已用额度、失效日期和备注。
- 采集日期 B020019 在原始清单中排在 2.1 版新增字段之后，但版本说明仍为 1.0 版，映射时应以字段编码和版本说明为准。

## 字段分组

| 分组 | 代表字段 | 说明 |
|------|----------|------|
| 集团识别与母公司映射 | 集团ID、机构ID、母公司统一社会信用代码、工商注册编号、母公司客户ID、母公司名称、集团名称、集团成员数 | 用于集团主体识别与母公司映射 |
| 地址与更新时间 | 注册地址、注册地国家地区、注册地行政区划、更新注册信息日期、办公地址、办公地址行政区划、更新办公地址日期 | 用于集团地址信息与更新时间管理 |
| 授信与风险 | 授信类型、风险预警信号、关注事件代码、内部评级结果、授信敞口额度、授信敞口已用额度 | 用于集团授信关系、风险和额度信息维护 |
| 生命周期与扩展 | 失效日期、备注、采集日期 | 用于状态补充与数据快照时间记录 |

## 重点规则

- 日期类字段统一采用 `YYYY-MM-DD` 格式。
- 金额类字段 `20n(2)` 表示数值型金额字段，并保留两位小数。
- 代码类字段目前只有格式约束，缺少取值范围；后续应补代码表或制度说明。
- B020020“母公司客户ID”引入后，2.2 与 2.1 对公客户主数据之间存在显式映射关系，后续可进一步抽“集团-客户映射规则”。
- B020019 虽位于 2.1 版新增字段之后，但版本说明仍为 1.0 版，字段版本应以编码和版本说明为准。
