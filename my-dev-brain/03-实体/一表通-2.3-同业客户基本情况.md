# 一表通-2.3-同业客户基本情况

## 简介

2.3《同业客户基本情况》是一表通系统的同业客户主数据表，用于维护同业客户的机构身份、证照信息、工商属性、资本结构、评级风险和授信关系等信息。

## 关键信息

- 所属系统：[[03-实体/一表通系统|一表通系统]]
- 报表编码：2.3
- 报表名称：同业客户基本情况
- 当前原文依据：[[01-资料库/一表通系统/2026-04-16-2.3-同业客户基本情况-数据字典-原文|2.3 同业客户基本情况-数据字典-原文]]
- 知识沉淀位置：本页维护字段分组、关键字段与规则；字段全量清单见原文页

## 当前结论

- 本表是同业客户主数据表，关键键可视为“同业ID + 机构ID”。
- 与 2.1《单一法人基本情况》相比，本表多了同业机构身份特征字段，如金融许可证件号码、SWIFT 编码、机构类型。
- 版本演进上，2.0 试用版新增“对公客户类型、企业控股类型”，2.1 版新增“其他证件类型/号码、行业类型、失效日期、备注”，说明同业客户口径逐步向通用客户主数据对齐。
- 采集日期 B030036 在原始清单中排在 2.1 版新增字段之后，但版本说明仍为 1.0 版，映射时应以字段编码和版本说明为准。

## 重点规则

- 日期类字段统一采用 `YYYY-MM-DD` 格式。
- 金额类字段 `20n(2)` 表示数值型金额字段，并保留两位小数。
- 代码类、指示器类字段当前只有格式约束，缺少取值范围；后续应补代码表或制度说明。
- SWIFT 编码 `11!an` 为固定长度编码字段，后续校验时应注意长度和字符集。
- B030036 虽位于 2.1 版新增字段之后，但版本说明仍为 1.0 版，字段版本应以编码和版本说明为准。
