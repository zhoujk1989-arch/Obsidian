# 1104-G04-利润表

## 简介

G04《利润表》是反映填报机构一定期间内经营成果的监管会计报表，与财政部发布的有关金融企业会计制度一并阅读。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统G04
- 报表名称：利润表
- 当前原文依据：[[01-资料库/1104系统/2026-04-22-G04-利润表-填报说明-原文|G04 利润表填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- G04 是 1104 系统中专门反映金融机构经营成果的利润类报表，覆盖从利息净收入到净利润的完整利润表结构。
- 三档报送频度：境内分支机构汇总数据（月报，月后6日，G04_I不报送）、法人汇总数据（季报，季后13日）、合并报表数据（季报，季后40日）。
- 金融控股公司仅报送法人汇总数据和合并报表数据。
- 填报机构范围极广：涵盖所有银行类（政策性银行、大型商业银行、股份制、城商行、农商行、民营银行、外资法人银行、外国银行分行等）和非银类（财务公司、信托、金融租赁、汽车金融、货币经纪、消费金融、 AMC、金融资产投资公司、理财公司、金融控股公司）。
- 与 G01 形成表间校验：G04_[13.A净利润] = G01_[57.1C其中：本年利润]。
- 与 G08 形成表间校验：G08 附注所有者权益合计 = G04_[13.A净利润]。

## 重点口径

### 利息收入/支出的口径差异

- 同业往来利息收入/支出在境内/法人汇总数据口径下包含与附属金融机构（独立法人）之间产生的利息，但在合并报表数据口径下不包含。

### 投资收益的口径

- "3.投资收益" = "3.1债券投资收益" + "3.2特定目的载体投资收益" + "3.3股权投资收益" + "3.4贵金属投资收益" + "3.5其他投资收益"，均不含已计入利息收入部分。

### 纳税金额的填列规则

- 纳税金额按实际缴纳（扣除税收返还金额）情况填列，而非按应缴金额填列。
- 示例：2026年一季度应缴增值税5000万元 + 2026年二季度应缴增值税2000万元，若一季度收到上一年度增值税返还500万元，则一季度填列4500万元（5000-500），二季度填列6500万元（4500+2000）。

### 净利润的表间校验

- G04_[13.A净利润] = G01_[57.1C其中：本年利润]，这是利润表与资产负债表之间的核心校验关系。

| 金融控股公司报送范围 | 仅报送法人汇总数据和合并报表数据，不报送境内分支机构汇总数据 | 原文一般说明第4条 |
