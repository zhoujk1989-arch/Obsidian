# S71-银行业普惠金融重点领域贷款情况表

## 简介

S71《银行业普惠金融重点领域贷款情况表》是金监统系列监管报表之一，用于全面、准确反映银行业金融机构对普惠金融重点领域的贷款支持情况。报表结合现有小微企业、三农、巩固脱贫成果等统计指标，建立普惠金融重点领域贷款统计指标体系。

- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- S71 是监管层追踪普惠金融发展情况的核心报表，覆盖"小微企业+三农+消费"三大维度，分为表I、表II、表III三个部分。
- 表I（普惠型小微企业和其它组织贷款）和表II（普惠型涉农贷款）为月报第一批次（月后6日），表III（普惠型消费贷款）为季报第一批次（季后13日）。
- 表I 和表II 在涉农小微企业法人贷款项目上存在完全对应的表间相等关系（II[1.2]=I[1.1]），形成跨表数据一致性约束。
- S71 与 S63（大中小微型企业贷款情况表）、G11_II（资产质量五级分类情况表）存在表间校验关系，当前已具备对应实体页承接表间校验。
- 普惠型小微企业法人贷款授信上限为 3000 万元（含），普惠型涉农贷款中农户经营性贷款授信上限为 500 万元（含），普惠型消费贷款授信上限为 10 万元（含）。
- S71 新增了退役军人创业贷（金发〔2024〕41号）和原扶贫小额信贷等专项统计科目，体现了政策导向。

### 当年累放贷款年化利息收益计算

```
年化利息收益 = Σ(每笔放款金额 × 合同年化利率)

例1：当年放款3笔（30万、40万、50万），年利率6%
     年化利息收益 = 30×6% + 40×6% + 50×6% = 7.2万元

例2：发放10万元贷款，2个月产生1000元利息
     年化利息收益 = 0.1 × 12/2 = 0.6万元
```

- 仅统计当年发放贷款的利息收益，往年发放贷款不计入
- 不考虑资金成本、实际用款时长、贷款罚息收入、其他附加费用、产生不良后的本息损失

### 单户授信总额口径

- 表内外统一授信，包括但不限于：贷款、贸易融资、票据承兑和贴现、透支、保理、担保、贷款承诺、开立信用证
- 无授信总额度管理的，以表内外授信余额和实际审批贷款额度的较大值为准
- 授信总额变更时，贷款余额等数据调整至新分类，但授信变更前累放数据仍在原分类
- 自然人同时获得经营性贷款和消费贷款的，按各自额度分别填报，无需加总
- 对小微企业的授信和对同一企业企业主的授信，不合并计算
