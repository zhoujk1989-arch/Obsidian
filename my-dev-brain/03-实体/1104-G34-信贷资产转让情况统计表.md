# 1104-G34-信贷资产转让情况统计表

## 简介

G34《信贷资产转让情况统计表》是 1104 系统中用于收集银行业金融机构盘活存量信贷资产情况的专项监管报表，按转让方式对信贷资产转让效率、交易平台、登记平台和直接交易对手等四个维度进行跟踪监测。

- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- G34 是 1104 系统中唯一专门监测信贷资产转让情况的报表，核心目标是跟踪银行业金融机构盘活存量信贷资产的规模和结构。
- 本表从四个维度刻画信贷资产转让：**转让效率**（损失/核销/收回现金/自持）、**交易平台**（银登中心/证券交易所/银行间市场/其他）、**登记平台**（银登中心登记金额）、**直接交易对手**（特定目的载体/法人机构/个人投资者）。
- 信贷资产口径同各项贷款，但不包含贸易融资和贴现及买断式转贴现。不含银行法人机构内部贷款转让。
- 本表填报信贷资产转让的**本金**，不含利息。
- 关注类贷款的定义以《贷款风险分类指引》、《商业银行金融资产风险分类办法》的相关要求为标准。
- 报送频度：季报（季后18日），与 G44 法人汇总数据、S26 同为季后18日报送的报表。
- 填报机构范围广泛：政策性银行、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用社、村镇银行、外资法人银行、外国银行分行、财务公司、金融租赁公司、消费金融公司和汽车金融公司。

## 字段概览

> 字段全量定义见原文页。本页只保留分组、关键字段与规则。

### 行项目（转让方式维度）

| 数据项编码 | 数据项名称 | 说明 |
|------|------|------|
| [1] | 直接转让债权 | 出让方将确定的可转让信贷资产向其他机构直接整体性转让贷款所有权对应的本金，不含地方债务置换 |
| [1.1] | 其中：以债转股为目的的 | 出于将贷款转为股权目的，向实施机构（金融资产管理公司、保险资管机构、国有资本投资运营公司等）转让 |
| [1.2] | 其中：不良贷款 | 向其他机构直接转让不良贷款所有权对应的本金，包括转让给四大AMC及地方AMC |
| [1.2.1] | 其中：以债转股为目的的（不良） | 出于将不良债权转为股权目的，向实施机构转让不良贷款所有权对应的本金 |
| [1.3] | 其中：关注类贷款 | 向其他机构直接转让关注类贷款所有权对应的本金 |
| [2] | 信贷资产证券化 | 发起方将信贷资产转让给受托机构，由受托机构以资产支持证券形式向投资机构发行受益证券 |
| [2.1] | 其中：不良贷款 | 将不良信贷资产转让给受托机构发行 ABS |
| [2.2] | 其中：关注类贷款 | 将关注类信贷资产转让给受托机构发行 ABS |
| [3] | 信贷资产收益权转让 | 出让方保留信贷资产所有权，仅转让获取本金、利息和其他约定款项的权利 |
| [3.1] | 其中：不良贷款 | 转让不良信贷资产收益权，保留所有权 |
| [3.2] | 其中：关注类贷款 | 转让关注类信贷资产收益权，保留所有权 |
| [4] | 通过其他方式转让 | 除上述三种方式外的其他转让方式 |
| [4.1] | 其中：不良贷款 | 以其他方式转让不良信贷资产本金 |
| [4.2] | 其中：关注类贷款 | 以其他方式转让关注类信贷资产本金 |
| [5] | 合计金额 | 四种转让方式对应的信贷资产本金总额 |
| [5.1] | 其中：不良贷款 | 四种转让方式对应的不良信贷资产本金总额 |
| [5.2] | 其中：关注类贷款 | 四种转让方式对应的关注类信贷资产本金总额 |
| [6] | 附注：未出表的转让业务 | 名义转让但未实现风险隔离、仍存在于资产负债表内的贷款余额 |

### 列项目 — 转让效率维度（B-E列）

| 数据项编码 | 数据项名称 | 说明 |
|------|------|------|
| [B] | 损失 | 剔除自持、收回现金、收回其他对价、拨备核销后的剩余损失金额 |
| [C] | 核销 | 通过已计提准备金进行核销的金额 |
| [D] | 转让收回现金 | 转让过程中收回的现金金额，不含自持部分 |
| [D1] | 转让收回其他对价 | 转让过程中收回的现金以外其他资产的公允价值，不含自持部分 |
| [E] | 自持 | 转让过程中由本机构继续持有部分的金额 |

### 列项目 — 交易平台维度（F-I列）

| 数据项编码 | 数据项名称 | 说明 |
|------|------|------|
| [F] | 银登中心 | 以银登中心为交易平台转让的信贷资产本金 |
| [G] | 证券交易所 | 以上交所、深交所为交易平台转让的信贷资产本金（不含中证资本监测报价中心、固定收益平台、深圳大宗交易平台及地方金融交易所） |
| [H] | 银行间市场 | 通过银行间市场转让的信贷资产本金 |
| [I] | 其他 | 通过其他交易场所或线下签署协议转让的信贷资产本金 |

**置灰规则（无数据）**：
- 信贷资产证券化（第2行）的 [F]、[I] 列置灰 — 因为 ABS 在银行间债券市场发行和交易
- 信贷资产收益权转让（第3行）的 [G]、[H] 列置灰 — 因为收益权转让须到银登中心登记流转

### 列项目 — 登记平台维度（J列）

| 数据项编码 | 数据项名称 | 说明 |
|------|------|------|
| [J] | 在银登中心登记的金额 | 填报机构在银登中心登记的信贷资产流转业务对应的本金，≤ 对应行的 [A] 列 |

**登记范围**：通过银登中心流转的、各银行业机构间线下达成交易的、银行在其他交易平台进行转让的、银行以债转股为目的转让给实施机构的、银行以处置不良资产为目的转让给四大AMC及地方AMC的信贷资产债权及对应的收益权相关业务。

### 列项目 — 直接交易对手维度（K-V列）

| 数据项编码 | 数据项名称 | 说明 |
|------|------|------|
| [K] | 商业银行理财产品 | 特定目的载体，以理财计划为交易对手 |
| [L] | 信托投资计划 | 特定目的载体，以信托计划为交易对手 |
| [M] | 证券投资基金 | 特定目的载体，以基金为交易对手 |
| [N] | 证券公司资产管理计划 | 特定目的载体，以券商资管计划为交易对手 |
| [O] | 基金管理公司及子公司资产管理计划 | 特定目的载体，以基金资管计划为交易对手 |
| [P] | 其他银行业金融机构 | 法人机构，除商业银行、信托公司和金融资产管理公司以外的其他银行业金融机构 |
| [Q] | 保险公司 | 法人机构，以保险公司为交易对手 |
| [R] | 资产管理公司 | 法人机构，包括金融资产管理公司及地方资产管理公司 |
| [R1] | 其中：金融资产管理公司 | 法人机构，仅填报金融资产管理公司（不含地方AMC） |
| [S] | 其他非银行业金融机构 | 法人机构，以其他非银金融机构为交易对手 |
| [T] | 一般企业 | 法人机构，以一般企业为交易对手 |
| [U] | 其他 | 法人机构，其他类型交易对手 |
| [V] | 个人投资者 | 以个人投资者为交易对手（第5、6行置灰不填） |

**特殊规则**：
- 第5行、第6行对应的 [K-U] 列：填报机构为发起机构，直接交易对手为受托机构
- 第5行、第6行的 [V] 列不填列（ABS 不允许向个人投资者发行）
- 第3行（收益权转让）的 [V] 列不填列

## 重点口径

### 四种转让方式的核心区别

| 转让方式 | 所有权是否转移 | 典型场景 |
|------|------|------|
| 直接转让债权 [1] | **是**，整体性转让贷款所有权 | 将贷款直接卖给其他机构 |
| 信贷资产证券化 [2] | **是**，转让给受托机构（SPV） | 发起 ABS，向投资机构发行受益证券 |
| 信贷资产收益权转让 [3] | **否**，保留所有权，仅转让收益权 | 出让方仍承担信贷资产的所有权风险 |
| 其他方式 [4] | 视具体情况而定 | 不属于上述三类的其他转让安排 |

### 表内核对关系（核心校验）

#### 转让效率维度：各列之和 = A列

```
第1行（及子项）：[A] = [B]+[C]+[D]+[D1]
第2-5行（及子项）：[A] = [B]+[C]+[D]+[D1]+[E]
第6行：[A] ≥ [D]+[D1]
```

#### 交易平台维度：各列之和 = A列

```
第1行（及子项）：[A] = [F]+[I]  （第1行无G、H列）
第2行（及子项）：[A] = [G]+[H]  （第2行无F、I列，置灰）
第3行（及子项）：[A] = [F]+[I]  （第3行无G、H列）
第4行（及子项）：[A] = [F]+[G]+[H]+[I]
第5行（合计）：[A] = [F]+[G]+[H]+[I]
```

#### 直接交易对手维度：各列之和 = A列

```
第1-4行（及子项）：[A] = [K]+[L]+[M]+[N]+[O]+[P]+[Q]+[R]+[S]+[T]+[U]+[V]
第2-3行（及子项）：[A] = [K]+[L]+[M]+[N]+[O]+[P]+[Q]+[R]+[S]+[T]+[U]  （无V列）
第5-6行：[A] = [K]+[L]+[M]+[N]+[O]+[P]+[Q]+[R]+[S]+[T]+[U]+[V]  （但第5、6行的V列置灰）
```

```
[5.A] = [1.A]+[2.A]+[3.A]+[4.A]
[5.1A] = [1.2A]+[2.1A]+[3.1A]+[4.1A]
[5.2A] = [1.3A]+[2.2A]+[3.2A]+[4.2A]
```

#### 子项包含关系

```
[1.1A] ≤ [1.A]、[1.2A] ≤ [1.A]、[1.2.1A] ≤ [1.2A]、[1.3A] ≤ [1.A]
[2.1A] ≤ [2.A]、[2.2A] ≤ [2.A]

```

### 案例解析

**案例1：不良贷款直接转让债权（效率维度）**

> 填报机构持有100万的不良贷款，已计提拨备50万，通过直接转让债权方式转让，收回40万现金，通过拨备核销50万，剩余损失10万。

| 数据项 | 金额（万元） |
|------|------|
| [B] 损失 | 10 |
| [C] 核销 | 50 |
| [D] 转让收回现金 | 40 |
| 校验：[B]+[C]+[D] = 10+50+40 = 100 = [A] | ✓ |

**案例2：正常贷款资产证券化（效率维度）**

> 填报机构持有100万的正常贷款，通过资产证券化，转让收回现金90万，自持5万。

| 数据项 | 金额（万元） |
|------|------|
| [D] 转让收回现金 | 90 |
| [E] 自持 | 5 |
| 校验：[D]+[E] = 90+5 = 95 ≠ 100，差额5万可能计入 [B] 损失或 [C] 核销 | 需补充信息 |

**案例3：多笔交易综合（交易平台+对手方维度）**

> ①通过银登中心以直接转让债权模式向某商业银行出让正常类信贷资产50万；②以债转股为目的，以直接转让债权方式向某实施机构（金融资产管理公司）出让30万（假定在银登中心登记）；③以处置不良资产为目的，以直接转让债权方式向某地方AMC出让不良信贷资产25万；④通过线下协议，以转让信贷资产信托受益权方式向某信托公司出让不良类信贷资产45万；⑤通过银登中心，以转让信贷资产收益权方式向其他银行业金融机构出让不良信贷资产35万。

## 重点规则

- G34 是 1104 系统中唯一专门监测信贷资产转让的报表，四个维度（效率、交易平台、登记平台、交易对手）交叉覆盖，结构复杂。
- 直接转让债权 [1] 不含地方债务置换；信贷资产收益权转让 [3] 出让方保留所有权，仅转让收益权。
- 信贷资产证券化 [2] 的交易平台仅限证券交易所 [G] 和银行间市场 [H]，银登中心 [F] 和其他 [I] 置灰。
- 信贷资产收益权转让 [3] 须在银登中心登记流转，故证券交易所 [G] 和银行间市场 [H] 置灰。
- R1（金融资产管理公司）≤ R（资产管理公司，含地方AMC），R1 不含地方资产管理公司。
- 第6行"未出表的转让业务"的 [V]（个人投资者）列不填列。
- 金额单位：万元，保留两位小数。
