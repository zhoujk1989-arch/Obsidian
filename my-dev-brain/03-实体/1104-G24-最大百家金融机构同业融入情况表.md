# 1104-G24-最大百家金融机构同业融入情况表

## 简介

G24 是 1104 系统中专门统计填报机构对最大百家金融机构同业融入负债情况的报表，用于分析填报机构的流动性风险状况，特别是同业融入资金的结构性与稳定性。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银监统0017号
- 报表名称：最大百家金融机构同业融入情况表
- 当前原文依据：[[01-资料库/1104系统/2026-04-22-G24-最大百家金融机构同业融入情况表-填报说明-原文|G24 原文页]]
- 知识沉淀位置：本页

## 当前结论

- G24 是 1104 系统中专门统计同业融入（负债端）的报表，与 G27（主要负债项目明细表，含同业敞口）形成互补——G27 从额度区间角度看同业敞口分布，G24 从交易对手维度看最大百家金融机构的同业融入明细。
- G24 采用"按交易对手排序"的独特结构：按扣除结算性同业存款后的同业融入余额从大到小排列，最大一家填报在第[1]行，依次至第[100]行，不足100条时补充同业存单数据，仍不满则补0填充满100条。
- G24 的填报机构范围广泛，几乎覆盖所有类型的银行业金融机构和非银行金融机构，是 1104 系统中填报机构覆盖最广的报表之一。
- G24 统计的同业融入业务包括四类：同业拆放、同业存放、卖出回购、委托方同业代付。
- G24 的核心口径是"扣除结算性同业存款后的融入余额"（K列），这是分析流动性风险的关键指标。
- G24 的表间校验关系较少，唯一明确的表间校验是与 G01 的总负债项目校验：`G24_[103.B] = G01_[49.C]`。
- G24 与 G27 存在表间校验依赖：`G27[3.A] ≥ G24[102J]+[102N]`（同业敞口），G24 的同业融入合计+同业存单应不超过 G27 的同业敞口。

| C | 金融机构代码 | 金融许可证上的金融机构代码 |
| D | 同业拆放 | 同业拆借及同业借款融入资金余额（法人合并） |
| E | 其中：同业拆借 | 通过全国统一同业拆借网络融入的无担保资金 |
| F | 其中：同业借款 | 法律法规允许的同业资金借入业务 |
| G | 同业存放 | 吸收其他金融机构存放款项（仅存款类机构） |
| H | 卖出回购 | 返售协议下先卖出再购回融入的资金 |
| I | 委托方同业代付 | 委托其他金融机构向企业客户付款的资金融通 |
| J | 同业融入合计 | 对每一家金融机构的同业融入合计余额 |

| M | 扣除项：结算性同业存款 | 结算性同业银行结算账户存款（不含投融资性） |
| N | 附注：发行的同业存单 | 向各类金融机构发行的同业存单余额 |

### 行定义（汇总维度）

| 行编码 | 行名称 | 说明 |
|------|------|------|
| [1]-[100] | 最大百家金融机构 | 按K列从大到小排列的各家金融机构 |
| [101] | 百家合计 | [1]至[100]行各列之和 |
| [102] | 全部同业合计 | ≥[101]，含百家之外的其他同业融入 |
| [103.B] | 全部负债合计 | 表间校验用，应与 G01_[49.C] 相等 |

## 重点口径

### 同业拆放（D列）

- 以法人为基础合并填报，即将每一家金融机构及其分支机构的所有同业拆入合并在一起，在其法人名称下填报。
- 例如：填报机构从中国工商银行北京分行、中国工商银行石家庄分行拆入的资金，应统一以中国工商银行的名义填报。
- D = E + F，且 D ≥ E + F（允许存在未拆分的子项）。

### 结算性同业存款扣除（M列）

- 依据《关于规范金融机构同业业务的通知》(银发〔2014〕127号) 和《中国人民银行关于加强银行业金融机构人民币同业银行结算账户管理的通知》(银发〔2014〕178号) 填报。
- 仅限存款类金融机构填报。
- 包括结算性同业银行结算账户存款，但不包括投融资性结算账户存款。

### 排序规则

- 按 K 列（扣除结算性同业存款后的融入余额）从大到小排列。
- 满足要求（K列为正值）的记录不足100条时，继续填报持有同业存单但没有其他同业融入敞口的记录，按 N 列排序。
- 补充同业存单数据后仍不满100条，需补0填充满100条记录。

## 重点规则

### 表内核对关系

| 校验关系 | 说明 |
|------|------|
| [J] = [D]+[G]+[H]+[I] | 同业融入合计 = 同业拆放 + 同业存放 + 卖出回购 + 委托方同业代付 |
| [K] = [D]+[G]+[H]+[I]-[M] | 扣除结算性同业存款后的融入余额 = 同业融入合计 - 结算性同业存款 |
| [L] = ([D]+[G]+[H]+[I]-[M]) / [103.B] | 占负债合计比重 = K列 / 全部负债合计 |
| [101] = [1]+[2]+...+[100] | 百家合计 = 各家之和 |
| [102] ≥ [101] | 全部同业合计 ≥ 百家合计 |
| D ≥ E + F | 同业拆放 ≥ 同业拆借 + 同业借款 |
| K1 ≥ K2 ≥ K3 … ≥ K100 | 按K列从大到小排列 |

### 表间核对关系

| 校验关系 | 说明 |
|------|------|
| G24_[103.B] = G01_[49.C] | G24 全部负债合计 = G01 负债及所有者权益合计 |

## 备注

- G24 的报表编码明确为"银监统0017号"，与其他部分 1104 报表原文中仅写"银监统"不同。
- G24 的填报机构范围与 G44（杠杆率情况表）高度重合，但 G24 额外包括外国银行分行、企业集团财务公司、信托投资公司、金融租赁公司、汽车金融公司、消费金融公司、金融资产管理公司和金融资产投资公司。
- G24 是理解银行同业负债集中度的核心报表，交易对手维度的"最大百家"排序有助于识别同业负债依赖度和潜在流动性风险。
- 原文中"[103.B]"作为全部负债合计的引用标识，与 G01_[49.C] 形成表间校验，但原文未明确[103.B]的具体定义，需结合 G01 理解。
