# EAST5.0-IE_007_705-贷款五级形态变动表

## 基本信息

- 报表编码：`IE_007_705`
- 报表名称：贷款五级形态变动表
- 物理表名：`IE_007_705_INC`
- 所属系统：[[03-实体/EAST5.0系统|EAST5.0系统]]
- 频率：月
- 负责人：王超
- 状态：正常
- 自动取数状态：已上线
- 字段总数：15（FIELD = 15，INDICATOR = 0）
- 引用码表数：2

## 业务描述

报送所有表内信贷的五级分类变动信息，与其他表格中五级分类字段一致。需要报送信用卡业务的五级形态变动，填报为信贷合同号=信用卡账号，信贷借据号=信用卡卡号。

## 关键字段与分组

| 分组 | 字段 | 说明 |
|------|------|------|
| 机构标识 | `JRXKZH` 金融许可证号（PK）、`NBJGH` 内部机构号（PK）、`YHJGMC` 银行机构名称 | 标识变动所属银行机构，关联机构信息表 |
| 合同/借据标识 | `XDHTH` 信贷合同号（PK）、`XDJJH` 信贷借据号（PK） | 主键为 `XDHTH` + `XDJJH` + `TZRQ` + `CJRQ`；信用卡业务中信贷合同号=信用卡账号，信贷借据号=信用卡卡号 |
| 客户标识 | `KHTYBH` 客户统一编号、`KHMC` 客户名称 | 客户身份标识 |
| 变动日期 | `TZRQ` 调整日期（PK）、`CJRQ` 采集日期（PK） | YYYYMMDD格式；采集日期为报送期末日期 |
| 五级分类变动 | `YWJFL` 原五级分类、`XWJFL` 新五级分类 | 正常/关注/次级/可疑/损失，引用码表 `EAST_DKWJFL` |
| 变动属性 | `BDYY` 变动原因（银行自定义）、`BDFS` 变动方式（人工/自动，引用 `EAST_BDFS`） | 记录变动触发方式 |
| 经办与备注 | `JBGYH` 经办人工号（自动变动允许为空）、`BBZ` 备注（银行自定义） | 经办员工号关联员工表 |

## 码表引用

| 码表编码 | 码表名称 | 使用字段 |
|----------|----------|----------|
| `EAST_DKWJFL` | 贷款五级分类 | `YWJFL`（原五级分类）、`XWJFL`（新五级分类），5 码值：正常/关注/次级/可疑/损失 |
| `EAST_BDFS` | 变动方式 | `BDFS`（变动方式），2 码值：人工/自动 |

## 口径规则

1. **主键**：`XDHTH`（信贷合同号）+ `XDJJH`（信贷借据号）+ `TZRQ`（调整日期）+ `CJRQ`（采集日期）。
2. **信用卡业务**：信贷合同号填报为信用卡账号，信贷借据号填报为信用卡卡号。
3. **原五级分类/新五级分类**：取值均为正常、关注、次级、可疑、损失，与信贷合同表、信贷借据表等表格中的五级分类字段口径一致。
4. **变动方式**：区分人工变动和自动变动；自动变动的经办人工号允许为空。
5. **变动原因**：中文字典或中文描述，银行自定义。
6. **采集日期**：按月采集的表，采集日期应为月末最后一天。

## 关联关系

- 关联 `IE_001_101` 机构信息表：通过 `JRXKZH`（金融许可证号）和 `NBJGH`（内部机构号）
- 关联 `IE_005_501` 信贷合同表：通过 `XDHTH`（信贷合同号）
- 关联 `IE_005_503` 个人信贷业务借据表 / `IE_005_504` 对公信贷业务借据表：通过 `XDJJH`（信贷借据号）
- 关联 `IE_002_201` 个人基础信息表 / `IE_002_203` 对公客户信息表：通过 `KHTYBH`（客户统一编号）
- 关联 `IE_001_102` 员工表：通过 `JBGYH`（经办人工号）
- 与信贷合同表、信贷借据表等表格的五级分类字段口径一致
