# 一表通-2.7收单商户信息表

## 简介

采集本行因线下线上收单业务而拓展的线下POS终端、线上终端以及二维码商户的信息。同一商户存在多个POS终端的，按多条报送。报数期开始前商户状态为"失效"或"注销"等状态的，不填。对应 EAST 收单商户信息表（`IE_009_906` 或相近编码，具体以 EAST 规范为准）。

## 基本信息

| 项目 | 内容 |
|------|------|
| 报表编码 | 2.7 |
| 报表名称 | 收单商户信息表 |
| 所属系统 | 一表通 |
| 数据域 | 收单业务信息 |
| 字段总数 | 19 |
| 主键字段 | 商户ID（B070001）、客户ID（B070002）、机构ID（B070003）、终端号（B070006）、采集日期（B070017） |
| 强制校验字段 | 商户ID、机构ID、商户名称、是否为POS机特约商户、清算卡号或账号、清算账号类型、清算账户名称、清算账号开户行名称、采集日期 |

## 字段分组

### 身份标识（3 字段，PK）
- B070001 商户ID（PK，强制）
- B070002 客户ID（PK，2.0试用版新增）
- B070003 机构ID（PK，强制）

### 商户基本信息（4 字段）
- B070004 商户名称（强制）
- B070005 是否为POS机特约商户（强制）
- B070006 终端号（PK）
- B070016 商户状态

### 商户分类信息（2 字段）
- B070007 商户类别码（银联MCC码，4位数字）
- B070008 商户类别码名称

### 清算信息（5 字段，含强制）
- B070009 清算卡号或账号（强制）
- B070010 清算账号类型（强制，2.1版）
- B070011 清算账户名称（强制）
- B070012 清算账号开户行名称（强制）
- B070015 商户地区

### 日期字段（4 字段）
- B070013 商户起效日期（默认值 9999-12-31，仅POS商户填）
- B070014 商户失效日期（默认值 9999-12-31，仅POS商户填）
- B070019 终端失效日期（2.1版新增，默认值 9999-12-31）
- B070017 采集日期（PK，强制）

### 备注（1 字段）
- B070018 备注（2.1版新增）

## 一表通与 EAST 差异要点

| 差异类型 | 涉及字段 | 说明 |
|----------|----------|------|
| 机构ID拼接 | 机构ID(B070003) | 一表通需拼接金融许可证号前11位 + 内部机构号 |
| 码值转换（数字→文本） | 是否为POS机特约商户(B070005) | 一表通报数字码值（0/1），EAST报中文文本（是/否） |
| 码值映射（一表通更宽泛） | 清算账号类型(B070010) | EAST多"第三方平台账户"码值；一表通码值比EAST宽泛 |
| 日期格式转换 | 商户起效日期(B070013)、商户失效日期(B070014)、采集日期(B070017) | 一表通 YYYY-MM-DD（10位），EAST YYYYMMDD（8位） |
| 地址格式差异 | 商户地区(B070015) | 一表通填报6位代码，EAST填报中文地址（省+市+县/区） |
| 新增字段 | 客户ID(B070002) | 2.0试用版新增，EAST无对应 |
| 2.1版新增 | 终端失效日期(B070019)、备注(B070018) | 2.1版本新增数据项 |
| 2.1版变更 | 清算账号类型(B070010) | 数据类别由"代码类"改为"文本类"，格式由"4!n"改为"anc..60"，"00-其他"扩展为"00-XX 银行自定义" |

## 填报范围

- **纳入**：本行因线下线上收单业务拓展的线下POS终端、线上终端以及二维码商户
- **排除**：报数期开始前商户状态为"失效"或"注销"等状态的商户
- **多条报送**：同一商户存在多个POS终端的，按多条报送

## 版本演进

| 版本 | 变更 |
|------|------|
| 2.0试用版 | 重组新增表，含16个字段（B070001~B070016） |
| 2.1版 | 新增终端失效日期(B070019)、备注(B070018)；清算账号类型(B070010)数据类别和格式变更 |
