# 1104-G01_I_1-表外业务情况表

## 简介

G01_I_1《表外业务情况表》是 1104 系统中的表外业务专项报表，由**表一**和**表二**两部分组成。

- **表一**：反映填报机构主要表外业务情况，包括担保承诺类、代理投融资服务类、中介服务类和其他类四大类。
- **表二**：反映填报机构非现金管理项下委托贷款按实际投向分类情况。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银监统0002号
- 报表名称：资产负债项目表附注第Ⅰ部分：表外业务情况表
- 当前原文依据：[[01-资料库/1104系统/2026-04-08-G01_I_1-表外业务情况表-填报说明-原文|G01_I_1 表外业务情况表填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、规则和常见口径

## 当前结论

- G01_I_1 由表一和表二组成，表一反映四大类表外业务，表二反映委托贷款投向。
- 表一涵盖担保承诺类、代理投融资服务类、中介服务类、其他类四大类表外业务。
- 表二专注于非现金管理项下委托贷款的行业投向分类。
- 附注项目包括置换预售监管资金的保函（[31.]）。

## 重点规则

### 表一：重要口径说明

| 项目 | 说明 |
|------|------|
| **委托贷款** | 资金由委托人提供，填报机构**不得代垫资金**，风险由委托人承担 |
| **委托投资** | 资金由委托人提供，填报机构**不得代垫资金**，风险由委托人承担；**不含**填报机构发行的表外理财产品 |
| **代理代销** | 按当年累计发生额填报，分信托/资管/保险/基金/理财等细分 |
| **金融衍生品** | 填报名义本金期末余额，**不包括计入**[16.1]代理金融衍生产品交易的衍生品 |
| **融资性保函** | 包括借款保函、融资租赁保函、透支保函、有价证券发行担保、一年期以上延期付款保函等 |
| **非融资性保函** | 包括投标保函、履约保函、预付款保函、海事保函、质量保函等 |

## 常见口径

| 问题 | 结论 | 依据 |
|------|------|------|
| G01_I_1 的组成？ | 表一（四大类表外业务）+ 表二（委托贷款投向） | 第一部分"引言" |
| 委托贷款风险承担？ | 风险由委托人承担，填报机构不得代垫资金 | 第三部分[14.]说明 |
| 委托投资排除范围？ | 不含填报机构发行的表外理财产品 | 第三部分[15.]说明 |
| 金融衍生品填报范围？ | 名义本金期末余额，不含[16.1]代理金融衍生产品交易 | 第三部分[29.]说明 |
| 置换保函限制？ | [31.A] <= [3.2A]（置换保函 <= 非融资性保函） | 第三部分附注[31.] |
