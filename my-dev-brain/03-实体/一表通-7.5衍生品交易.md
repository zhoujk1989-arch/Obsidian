# 一表通-7.5衍生品交易

## 业务定位

采集填报机构采集期内的所有自营、代客即期资金交易及衍生品交易。金融衍生产品是指其价值取决于一种或多种基础资产或指数，在未来某个日期交割的金融合约。合约的基本类型包括即期、远期、期货、掉期（互换）和期权，以及有即期、远期、期货、掉期（互换）和期权中一种或多种特征的结构化金融工具。

## 重点口径

### 交易类型码值（5类+自定义）

- 01 套期保值：定义见《银行业金融机构衍生产品交易业务管理暂行办法》第四条
- 02 代客：卖出衍生品
- 03 代客平盘：含权衍生品执行权利
- 04 做市：衍生品到期按合同约定交割
- 05 自营：远期或互换中按固定利率支付利息
- 00-XX 其他：银行自定义

### 交割方式码值

- 01 全额：到期按约定价格和币种对合约本金全额交付
- 02 差额：到期按轧差后的价格差和币种交付
- 03 净额：中央清算所净额结算
- 04 实物：标的物所有权转移
- 05 现金：按结算价格现金差价结算

### 交易对手大类（12类+其他）

01银行业金融机构 / 02保险业机构 / 03地方金融组织 / 04交易及登记结算类机构 / 05金融控股公司 / 06证券业金融机构 / 07第三方支付公司 / 08特定目的载体 / 09政府部门 / 10公司企业客户 / 11个人客户 / 12境外客户 / 00其他

### 交易状态码值

- 01 新增
- 02 终止
- 03 变更：成交后对原合约内容进行调整，包括部分平盘、重组等
- 04 行权
- 05 估值

### 掉期（互换）拆分报送规则

- 掉期（互换）交易将两端（LEG1、LEG2）拆分为2条记录报送
- 交易编号保持一致，即用2条记录表示1笔交易
- 如：掉期交易按近端、远端拆分为2条记录报送
- 互换交易拆分为收入、支付端2条记录报送

### 估值报送规则

- 衍生品交易存续期内按估值周期填报交易估值信息
- 填报估值金额、估值币种及估值日期，其他字段不变
- 如同一笔交易由多条记录表示，多条记录的估值信息字段保持一致

### 机构方向规则

- 若报送机构既可以为买方也可以为卖方，则按报送机构为卖方填报
- 交易对手方向字段：01买方/02卖方

### 实物贵金属排除规则

- 与1104《G02银行业金融机构衍生产品交易业务情况表》口径保持一致
- 不报送实物贵金属（代销工艺、摆件、采购实物金条等）

### 交易币种规则

- 多币种交易以名义本金对应币种填报
- 遵循GB/T 12406-2008，三位数字字母编码（如CNY）
- 多个币种合计折算为人民币填报为BWB
