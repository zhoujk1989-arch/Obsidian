# 1104-G22-流动性比例监测表

## 简介

G22《流动性比例监测表》是 1104 系统中专门衡量银行流动性资产与流动性负债匹配关系的核心报表，核心指标为流动性比例（LCR 的简化版本），用于对有关机构流动性比例状况进行分析、评价和预测。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银监统0015号
- 报表名称：流动性比例监测表
- 当前原文依据：[[01-资料库/1104系统/2026-04-22-G22-流动性比例监测表-填报说明-原文|G22 流动性比例监测表填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- G22 是 1104 系统中衡量银行流动性风险的基础报表，核心视角是"流动性资产/流动性负债"的匹配关系。
- G22 与 G21（流动性期限缺口统计表）形成互补：G22 以"一个月"为统一期限阈值，G21 按多期限桶（活期、7天内、1个月内、3个月内等）划分。
- G22 与 G25_Ⅰ（LCR）形成流动性风险监管的三层框架：G22 看月度流动性比例（≥100%），G25_Ⅰ 看30天压力情景下的合格优质流动性资产覆盖能力（≥100%），G25_Ⅱ 看1年稳定资金结构（≥100%）。
- G22 的填报机构范围极为广泛，覆盖几乎所有银行类和非银类金融机构（含财务公司、信托公司、金融租赁公司、汽车金融公司、消费金融公司）。
- G22 报送频度较为复杂：境内分支机构汇总数据为月报第一批次（月后6日），法人汇总数据为季报第一批次（季后13日），合并报表数据为季报季后40日内。
- G22 的核心结构为三部分：流动性资产（9个子项）、流动性负债（7个子项）、其他指标（流动性比例、平均/最低比例、质押存款/贷款）。
- G22_[3.] 流动性比例 = 流动性资产总和 / 流动性负债总和 × 100%，是监管核心指标。
- G22 与 G01 形成多处表间校验：现金项目（[1.1]）、黄金项目（[1.2]）直接取值于 G01。
- G22 与 G21 形成表间校验：定期存款（[2.2C]）、向中央银行借款（[2.6C]）、活期存款（[2.1C]）均从 G21 取数。
- 外国银行分行的填报规则与境内机构存在显著差异，主要体现在同业往来轧差方式、流动性资产/负债的扣除规则等方面。
- 同业存单的处理规则明确：持有的一个月内到期的同业存单计入"1.9 其他一个月内到期可变现的资产"，发行的一个月内到期的同业存单计入"2.7 其他一个月内到期的负债"。
- G22 的表内核对关系体现了严格的汇总逻辑和互斥逻辑（如[1.4]和[2.3]的互斥关系）。

## 重点口径

### 期限口径

- "一个月"统一按30天计算（与 G21 的期限桶划分不同）。
- 到期日为原始（合同）到期日，在非营业日到期的视为在紧接的下一个营业日到期。

### 同业往来轧差规则

- 境内机构：同业往来按轧差填报，[1.4] 为资产方净额（≥0），[2.3] 为负债方净额（>0），两者互斥（[1.4]>0 则 [2.3]=0）。
- 外国银行分行：同业往来不轧差，存放同业不分期限。

### 流动性资产的变现方式区分

- [1.7] "一个月内到期的债券投资"：持有到期获得现金兑付。
- [1.8] "在国内外二级市场上可随时变现的证券投资"：通过二级市场出售变现，必须剔除[1.7]避免重复。
- [1.8] 中股票和基金按市场价格50%折扣填报。

### 外国银行分行特殊规则

- 流动性资产应扣除预计不可回收部分，生息资产中用于满足最低监管要求的部分不计入。
- 流动性负债中，冻结存款不计入。
- [2.4] 已发行的债券和[2.6] 向中央银行借款不予填报。
- 同业往来不轧差，存放同业不分期限。
- [1.9] 和[2.7] 中境外联行往来及附属机构往来，资产方净额和负债方净额只能填报一方。

### 同业存单处理

- 持有的一个月内到期的同业存单 → [1.9] 其他一个月内到期可变现的资产。
- 发行的一个月内到期的同业存单 → [2.7] 其他一个月内到期的负债。

### 质押存款与质押贷款

- [8.] 和[9.] 仅填报对应的质押贷款剩余期限在一个月以内的部分。
- [2.1] + [2.2] ≥ [8.]（表内核对关系）。

## 重点规则

### 表内核对关系

- `[1.10] = [1.1] + [1.2] + [1.3] + [1.4] + [1.5] + [1.6] + [1.7] + [1.8] + [1.9]`（流动性资产汇总）
- `[2.8] = [2.1] + [2.2] + [2.3] + [2.4] + [2.5] + [2.6] + [2.7]`（流动性负债汇总）
- `[3.] = [1.10] / [2.8] × 100%`（流动性比例计算）
- `[6.] = [4.] / [5.] × 100%`（平均流动性比例计算）
- `[2.1] + [2.2] ≥ [8.]`（质押存款不超过活期+定期存款）
- `[7.C] ≤ [3.C]`（月度最低流动性比例 ≤ 月末流动性比例）
- 如果`[1.4] > 0`，则`[2.3] = 0`，否则`[2.3] ≥ 0`（同业往来轧差互斥）
- `[C] = [A] + [B]`（本外币合计 = 人民币 + 外币折人民币），不包括[1.4]、[1.10]、[2.3]、[2.8]、[3.]、[4.]、[5.]、[6.]和[7.]行

### 数据精度

- 金额保留两位小数，百分比保留两位小数。

### 填报币种

- 本外币合计人民币数据。
- 外币折算按报告期末最后一天国家外汇管理局公布的基准汇价。
- 四种主要货币（美元、欧元、日元、港币）以外的其他货币，通过套算确定。
