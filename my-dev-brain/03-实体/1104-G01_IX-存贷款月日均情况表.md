# 1104-G01_IX-存贷款月日均情况表

## 简介

G01_IX《存贷款月日均情况表》是 1104 系统中的存贷比监管指标专项报表，核心指标包括**调整后存贷比**、**调整后月日均存贷比**和**存款偏离度**。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银监统0002号
- 报表名称：资产负债项目表附注IX部分：存贷款月日均情况表
- 当前原文依据：[[01-资料库/1104系统/2026-04-08-G01_IX-存贷款月日均情况表-填报说明-原文|G01_IX 存贷款月日均情况表填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、规则和常见口径

## 当前结论

- G01_IX 以人民币、外币折人民币、本外币合计人民币三列（A/B/C）填报
- 核心指标为调整后存贷比（[9]）和调整后月日均存贷比（[10]）
- 存款需加外资法人银行境外母行一年期以上存放净额（+1项）
- 贷款需扣减再贷款等6类特定贷款
- 适用于大型银行等11类机构，财务公司以"0"填报

## 重点口径

### 报送口径

| 项目 | 内容 |
|------|------|
| 报送频率 | 月报（境内分支机构汇总），月后6日 |
| 填报币种 | 人民币、外币折人民币、本外币合计人民币三列（A/B/C） |
| 数据单位 | 万元（保留两位小数） |

### 填报机构

| 类别 | 机构范围 |
|------|----------|
| **按要求计算** | 大型银行（含邮储）、股份制银行、城商行、民营银行、农商行、农合行、农信用社、村镇银行、外资法人银行、外国银行分行 |
| **以"0"填报** | 企业集团财务公司等机构类别以外的机构 |

### 核心指标

| 编号 | 项目 | 计算公式 | 说明 |
|------|------|----------|------|
| [1] | 各项存款 | - | 企业、储蓄、保证金等存款 |
| [2] | 月日均各项存款 | 当月每日各项存款之和/当月天数 | |
| [3] | 各项贷款 | - | 贷款、贸易融资、票据融资等 |
| [4] | 月日均各项贷款 | 当月每日各项贷款之和/当月天数 | |
| [5] | 各项存款（调整后） | [1] + 境外母行一年期以上存放净额 | 外资法人银行适用 |
| [6] | 月日均存款（调整后） | 当月每日调整后存款之和/当月天数 | |
| [7] | 各项贷款（调整后） | [3] - 6项扣减 | 扣减再贷款等6类贷款 |
| [8] | 月日均贷款（调整后） | 当月每日调整后贷款之和/当月天数 | |
| [9] | 调整后存贷比 | [7]/[5] | 监管考核指标 |
| [10] | 调整后月日均存贷比 | [8]/[6] | 监管考核指标 |
| [11] | 存款偏离度 | 按银保监办发[2018]48号计算 | 监管考核指标 |

### 调整项说明

#### 存款调整（+1项）

| 编号 | 调整项 | 说明 |
|------|--------|------|
| + | 外资法人银行吸收的境外母行一年期以上存放净额 | 仅外资法人银行适用 |

> **注意**：大额存单已直接纳入"各项存款"，此处不再重复加回。

#### 贷款调整（-6项）

| 编号 | 扣减项 | 说明 |
|------|--------|------|
| 1 | 支农再贷款、支小再贷款对应的贷款 | |
| 2 | "三农"专项金融债对应的涉农贷款 | |
| 3 | 小微企业专项金融债对应的小微企业贷款 | |
| 4 | 剩余期限>=1年且债权人无权提前偿付的债券对应的贷款 | |
| 5 | 使用国际金融组织或外国政府转贷资金发放的贷款 | |
| 6 | 村镇银行使用主发起行存放资金发放的农户和小微企业贷款 | |

## 重点规则

### 表内校验关系

```text
[C] = [A] + [B]    适用于[1][2][3][4][5][6][7][8]    （本外币合计 = 人民币 + 外币）

[1.A] <= [5.A]     [1.B] <= [5.B]     [1.C] <= [5.C]     （各项存款 <= 调整后存款）
[2.A] <= [6.A]     [2.B] <= [6.B]     [2.C] <= [6.C]     （月日均各项存款 <= 调整后月日均存款）
[3.A] >= [7.A]     [3.B] >= [7.B]     [3.C] >= [7.C]     （各项贷款 >= 调整后各项贷款）
[4.A] >= [8.A]     [4.B] >= [8.B]     [4.C] >= [8.C]     （月日均各项贷款 >= 调整后月日均贷款）

[9.A] = [7.A] / [5.A]    [9.B] = [7.B] / [5.B]    [9.C] = [7.C] / [5.C]    （调整后存贷比）
[10.A] = [8.A] / [6.A]   [10.B] = [8.B] / [6.B]   [10.C] = [8.C] / [6.C]   （调整后月日均存贷比）
```

```text
[1.A] = G01_[61.A]    （各项存款人民币 = G01_[61.A]）

[1.C] = G01_[61.C]    （各项存款本外币合计 = G01_[61.C]）

[3.A] = G01_[62.A]    （各项贷款人民币 = G01_[62.A]）
[3.B] = G01_[62.B]    （各项贷款外币 = G01_[62.B]）

```
