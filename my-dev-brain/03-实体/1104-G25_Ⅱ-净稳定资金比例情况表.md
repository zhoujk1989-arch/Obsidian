# 1104-G25_Ⅱ-净稳定资金比例情况表

## 简介

G25_Ⅱ《净稳定资金比例情况表》是 1104 系统中的流动性监管核心报表，用于衡量银行未来 1 年内稳定资金来源对表内外业务的支撑能力。本表与 G25_Ⅰ（流动性覆盖率）同属一张报表的不同部分，分别从短期（30 天）和长期（1 年）两个时间维度刻画流动性风险。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：银监统0027号
- 报表名称：流动性覆盖率及净稳定资金比例情况表（第Ⅱ部分）
- 当前原文依据：[[01-资料库/1104系统/2026-04-22-G25_Ⅱ-净稳定资金比例情况表-填报说明-原文|G25_Ⅱ 净稳定资金比例情况表填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、规则和常见口径

## 当前结论

- G25_Ⅱ 的核心指标是净稳定资金比例（NSFR）= 可用的稳定资金 / 所需的稳定资金，监管要求不低于 100%。
- NSFR 从两个方向考察流动性：可用稳定资金（负债端）和所需稳定资金（资产端），通过期限结构揭示银行对长期稳定资金的依赖程度。
- 报表将剩余期限划分为三类：<6个月、6-12个月、≥1年，比 LCR 的 30 天视角更精细。
- NSFR 与 LCR（G25_Ⅰ）存在大量概念复用和表间校验关系，两者共同构成流动性风险监管的"短+长"双维度框架。
- 填报机构范围与 G44（杠杆率）高度重合，包括大型商业银行、股份制银行、资产规模 2000 亿以上城商行、民营银行、农商行和外资法人银行。
- 衍生产品部分采用复式记录法：可用资金端的[Ⅰ_7]（衍生产品负债净额）与所需资金端的[Ⅱ_11]（衍生产品资产净额）相互抵减，并设有 20% 的衍生产品附加要求[Ⅱ_13]。
- 表外项目（[Ⅱ_17]）纳入所需稳定资金计算，覆盖可无条件撤销/不可撤销的信用便利、担保信用证贸易融资工具、非契约性义务四类。

## 报表结构

### Ⅰ. 可用的稳定资金（资金来源端）

| 项目 | 含义 | 折算因子方向 |
|------|------|-------------|
| [1.] 一级和二级资本（≥1年） | 合格监管资本 | 高因子 |
| [2.] 不合格资本工具（≥1年） | 不符合合格资本定义的资本工具 | 低因子 |
| [3.] 来自零售和小企业客户的融资 | 稳定存款+欠稳定存款+担保融资 | 中-高因子 |
| [4.] 来自大中型企业、主权、公共部门实体、多边和政策性金融机构的融资 | 业务关系存款+非业务关系存款+担保融资 | 中因子 |
| [5.] 来自央行的融资 | 业务关系存款+非业务关系存款+担保融资 | 高因子 |
| [6.] 来自金融机构的融资 | 业务关系存款+非业务关系存款+担保融资 | 中因子 |
| [7.] 净稳定资金比例衍生产品负债 | 衍生产品负债扣除提供的变动保证金净额 | 特殊计算 |
| [8.] 交易日应付款 | 购买金融工具/外汇/大宗商品的应付款 | - |
| [9.] 相互依存的负债 | 满足相互依存条件的负债 | - |
| [10.] 其他负债和权益 | 以上未包括的所有项目 | - |

### Ⅱ. 所需的稳定资金（资金运用端）

| 项目 | 含义 | 折算因子方向 |
|------|------|-------------|
| [1.] 现金 | 可自由支配的现金（扣除抵押和使用安排部分） | 100% |
| [2.] 存款准备金 | 法定和超额存款准备金 | 100%（部分可能为0） |
| [3.] 对中央银行的债权 | 逆回购、6个月以内央票等 | 高因子 |
| [4.] 住房抵押贷款 | 正常类住房抵押贷款 | 高因子（≤35%风险权重可达85%） |
| [5.] 向个人/非金融机构/主权等发放的贷款（不含住房抵押） | 零售、小企业、大中型企业贷款等 | 中-高因子 |
| [6.] 存放在金融机构的业务关系存款 | 同业业务关系存款 | 低因子或0% |
| [7.] 向金融机构发放的贷款 | 存放同业、拆放同业、转贴现等 | 中因子 |
| [8.] 符合合格优质流动性资产定义的资产（未纳入以上） | HQLA中未被1/2/3覆盖的证券 | 高因子（一级资产95%、2A资产85%、2B资产50%） |
| [9.] 不符合HQLA定义的证券及其他期限<1年的无担保工具 | 股权、短期商票、货基等 | 中因子 |

| [11.] 净稳定资金比例衍生产品资产 | 衍生产品资产扣除收到的现金变动保证金净额 | - |
| [12.] 提供的衍生产品初始保证金及违约基金 | 初始保证金和中央交易对手违约基金 | - |
| [13.] 衍生产品附加要求 | 衍生产品负债金额的20%（自动计算） | - |
| [14.] 交易日应收款 | 出售金融工具/外汇/大宗商品的应收款 | - |
| [15.] 相互依存的资产 | 满足相互依存条件的资产 | - |
| [16.] 其他资产 | 不良贷款、逾期90天以上、股权、固定资产、资本扣减项等 | 低因子或100% |
| [17.] 表外项目 | 信用便利、流动性便利、担保、非契约性义务等 | 中-高因子 |

### 期限列结构

| 列 | 含义 |
|----|------|
| A/B/C | 剩余期限：<6个月 / 6-12个月 / ≥1年 |
| D/E/F | 对应折算因子（可用资金端）或（所需资金端） |
| G/H/I | 折算金额 = 余额 × 折算因子 |
| J | 合计 = G+H+I |

## 重点口径

- **报送口径**：法人口径季报（季后18日，第二批次）；合并报表数据季报（季后40日）
- **填报机构**：大型商业银行（含邮储银行）、股份制商业银行、资产规模在2000亿元以上的城市商业银行、民营银行、农村商业银行和外资法人银行
- **填报币种**：本外币合计人民币，按报告期末最后一天外管局基准汇价折算
- **数据单位**：万元，保留两位小数
- **期限判断原则**：
  - 可用资金端：按可能的最短期限（内嵌期权假定客户最早行权）
  - 所需资金端：按可能的最长期限（内嵌期权假定延长期限的期权被行使）
  - 活期存款填6个月以内；可提前支取定期存款若罚息不显著超过利息损失也填6个月以内
  - 多次还款的长期负债按各期现金流有效剩余期限分别填入不同时间段
- **不良贷款处理**：被划入不良分类（次级、可疑、损失）和逾期90天以上的表内资产统一填入[Ⅱ_16.其他资产]
- **变现障碍定义**：资产被作为抵质押物/信用增级工具、用于支付经营成本、或按规定不得出售转让的资产
- **相互依存条件**：（1）可清楚识别；（2）期限和本金数量相同；（3）银行作为转手方；（4）对手方不能相同

## 重点规则

### 大额存单填报规则
- **不可转让**：面向零售/小企业→[Ⅰ_3.]；面向大中型企业→[Ⅰ_4.]
- **个人与机构间不可转让**：个人部分→[Ⅰ_3.]；机构部分→[Ⅰ_4.]
- **个人与机构间可转让**：统一填入[Ⅰ_4.]
- 1年内到期或允许提前支取的计入E列（<1年），其余计入F列（≥1年）

### 同业存单填报规则
- **发行同业存单**：填入[Ⅰ_6.2]（来自金融机构的融资-非业务关系存款及其他无担保借款）
- **持有同业存单**：填入[Ⅱ_9.]（不符合HQLA定义的证券及其他期限<1年的无担保工具）

### 衍生产品处理
- [Ⅰ_7.] = [Ⅰ_7.1]（衍生产品负债）- [Ⅰ_7.2]（提供的变动保证金）
- [Ⅱ_11.] = [Ⅱ_11.1]（衍生产品资产）- [Ⅱ_11.2]（收到的现金变动保证金）
- 衍生产品负债与资产的抵减关系：[Ⅰ_7.I] = IF([Ⅰ_7.C]≥[Ⅱ_11.C], ([Ⅰ_7.C]-[Ⅱ_11.C])*[Ⅰ_7.F], 0)
- [Ⅱ_13.C] = 0.2 × [Ⅰ_7.1.C]（衍生产品附加要求，自动计算）

### 表间校验关系
- [Ⅱ_1.A] ≤ G01_[1.C]（法人口径下，现金不超过G01资产合计）
- [Ⅰ_1.C] ≤ G4A_([1.A]+[3.A]+[5.A])（可用资金端一级资本不超过G4A对应项）

### LCR ↔ NSFR 表间校验
- 现金：LCR_I.基础数据_[1.1.1A] = NSFR_Ⅱ.所需的稳定资金[1.A]
- 零售和小企业存款：LCR_[2.1.1A]+[2.1.2.1A] ≤ NSFR_Ⅰ.[3.1A]+[3.2A]
- 大中型企业等存款：LCR_[2.1.2.2A]+[2.1.2.3A] ≤ NSFR_Ⅰ.[4.A]+[5.A]
- 存款准备金：LCR_I.基础数据_[1.1.2A] ≤ NSFR_Ⅱ.[2.A]+[2.B]+[2.C]
- 不可无条件撤销信用便利：LCR_[2.1.4.10A] ≤ NSFR_Ⅱ.[17.2A]
- 可无条件撤销信用便利：LCR_[2.1.5.1A] ≤ NSFR_Ⅱ.[17.1A]
- 担保/信用证/贸易融资：LCR_[2.1.5.2A]+[2.1.5.3A]+[2.1.5.4A] ≤ NSFR_Ⅱ.[17.3A]
- 非契约性义务：LCR_[2.1.5.5A] ≤ NSFR_Ⅱ.[17.4A]

## 常见口径

- **问题**：NSFR 和 LCR 的区别是什么？
- **已知结论**：
  - LCR（第I部分）考察30天压力情景下的合格优质流动性资产覆盖，是短期流动性指标
  - NSFR（第Ⅱ部分）考察1年稳定资金来源对资产的支持能力，是结构性/长期流动性指标
  - 两者共同构成"短期压力+长期结构"的完整流动性风险监管框架
- **问题**：可提前支取的定期存款在 NSFR 中如何划分期限？
- **已知结论**：如果提前支取的罚息没有显著超过利息损失，应填入6个月以内；对于内嵌期权的项目按可能的最短期限填报。
- **问题**：住房抵押贷款和商业用房贷款在 NSFR 中如何区分？
- **已知结论**：只有以购买住房为目的并以该套住房为抵押的贷款才填入[Ⅱ_4.住房抵押贷款]；以住房作抵押但用作其他用途的贷款、商业用房贷款以及商住两用房贷款不在此项，应归入[Ⅱ_5.]或其他对应项目。
