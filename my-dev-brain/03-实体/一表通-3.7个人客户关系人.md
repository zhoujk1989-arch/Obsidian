# 一表通-3.7个人客户关系人

## 简介

一表通个人关系层报表，采集个人客户的所有关联关系（个人对个人、个人对对公），共 12 个字段。本人为本人担保的关系不报送。

## 关键信息

| 项目 | 内容 |
|------|------|
| 系统 | 一表通 |
| 报表名称 | 3.7个人客户关系人 |
| 数据域 | 个人贷款违约信息（按报表关系页分类） |
| 字段总数 | 12 |
| 原文页 | [[01-资料库/一表通系统/2026-04-27-3.7个人客户关系人-填报说明-原文]] |

## 当前结论

1. **采集范围**：个人客户的所有关联关系，含个人对个人、个人对对公；本人为本人担保的关系不报送。
2. **社会关系码值**：一表通报码值（01~08 + 00-XX），EAST 报中文；一表通码值比 EAST 更宽泛，需码值映射。
3. **关系人证件类型**：一表通码值分类比 EAST 更细（含大量对公证件码值），需码值映射。
4. **采集日期格式**：一表通为 `YYYY-MM-DD`（10位），EAST 为 `YYYYMMDD`（8位），需格式转换。
5. **2026年制度升级**：社会关系、关系人证件类型由代码类改为文本类；新增备注字段（C070012）。

## 重点口径

### 社会关系码值映射

| 一表通码值 | 含义 |
|------------|------|
| 01 | 配偶 |
| 02 | 子女 |
| 03 | 父母 |
| 04 | 亲属关系 |
| 05 | 朋友 |
| 06 | 同事 |
| 07 | 合作伙伴 |
| 08 | 离异原配偶 |
| 00-XX | 其他-银行自定义 |

### 机构ID生成规则

金融许可证号前11位 + 内部机构号（最长24位，不足不补齐）。理财登记中心统一使用金融许可证。

### 与 EAST 差异

- **社会关系**：EAST 报中文，一表通报码值；一表通码值更宽泛
- **关系人证件类型**：一表通码值分类比 EAST 更细
- **采集日期**：一表通 `YYYY-MM-DD`，EAST `YYYYMMDD`
