# 一表通-6.4-互联网贷款协议

## 简介

一表通协议域互联网贷款专项报表，采集机构与合作方签订的互联网贷款合作协议信息，覆盖业务模式、出资比例、客户数据授权和合作方角色等维度。

## 关键信息

- 报表ID：1994254277826388751
- 物理表名：`T_6_4`
- 排序号：33
- 频率：日
- 负责人：巴启威
- 状态：正常
- 字段总数：15（全部 FIELD，无 INDICATOR）
- 引用码表：1 张（`YBT_YWMS` 业务模式，4码值）

## 业务模式码值（`YBT_YWMS`）

| 码值 | 名称 |
| --- | --- |
| `01` | 本机构主要作为资金提供方共同出资发放贷款 |
| `02` | 本机构主要作为信息提供方共同出资发放贷款 |
| `03` | 银行单独出资合作发放互联网贷款 |
| `04` | 本机构独立开展互联网贷款业务 |

## 关联关系

- 通过 `机构ID`（F01）与 `[[03-实体/一表通-1.1-机构信息|1.1机构信息]]` 关联
- 通过 `协议ID`（F02）与 `[[03-实体/一表通-6.2-贷款协议|6.2贷款协议]]` 关联
- 通过 `借据ID`（F03）与贷款借据关联

## 核心特征

- **互联网贷款专属**：互联网贷款的认定参照《商业银行互联网贷款管理暂行办法》，是协议域中首个互联网贷款专项报表。
- **业务模式4分类**：`YBT_YWMS` 码表（4码值）区分共同出资、信息提供共同出资、单独出资合作、独立开展四种业务模式，是互联网贷款业务的核心区分维度。
- **出资比例双字段**：`F040007`（合作方出资）+ `F040008`（本机构出资）构成互联网贷款的分担结构，可计算各方出资比例。
- **数据授权体系**：`F040009`~`F040011` 三个字段构成客户数据授权书编号+生效日期+终止日期的完整授权链，是互联网贷款中首次出现的客户数据合规维度，反映《个人信息保护法》和互联网贷款管理办法对客户数据使用的监管要求。
- **合作方角色双维度**：`F040012`（风险评价服务）+ `F040013`（担保增信）区分合作方在互联网贷款中的不同角色，仅针对"单独出资合作"模式，可空。
- **合作协议ID的有条件必填**：`F040004` 业务模式为"合作"时必填，"独立"时允许为空，与业务模式码值形成联动。
- **担保责任金额**：`F040006` 是合作方担保责任的金额字段，与 `F040007`（合作方出资）+ `F040008`（本机构出资）共同刻画互联网贷款的信用风险分担结构。
