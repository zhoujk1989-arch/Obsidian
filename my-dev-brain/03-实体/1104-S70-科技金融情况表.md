# S70-科技金融情况表

## 简介

S70《科技金融情况表》是金监统系列监管报表之一，用于真实反映填报周期科技贷款投放情况，以及银行与投资机构合作支持科技创新、科技金融机构建设、科技投资等情况。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统S70
- 报表名称：科技金融情况表
- 当前原文依据：[[01-资料库/1104系统/2026-04-20-S70-科技金融情况表-填报说明-原文|S70科技金融情况表填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- S70 是监管层追踪科技金融发展情况的核心报表，覆盖"科技贷款 + 投贷联动 + 科技金融机构 + 科技投资"四大维度。
- S70 是季报（季后13日），填报机构范围广泛，覆盖银行类和非银类金融机构（信托公司、理财公司、金融资产投资公司）。
- 科技型企业的认定标准与 S71 表I[1.2]普惠型科技型小微企业法人贷款中"科技型小微企业"的定义完全一致（七类科技名单）。
- S70 与 S71、S63、G01_Ⅶ 均存在表间校验关系，但 S63 尚未在知识库中建立实体页。
- S70 第IV部分（科技投资情况表）仅由信托公司、理财公司、金融资产投资公司填报，银行类机构不填报此部分。

### 当年累放贷款年化利息收益计算

```
年化利息收益 = 贷款金额 × 合同年化利率 × (12/实际月数)

例：发放10万元贷款，2个月产生1000元利息
    年化利息收益 = 0.1 × 12/2 = 0.6万元
```

- 仅统计当年发放贷款的利息收入，往年不计入

### 投贷联动合作模式

| 模式 | 说明 | 是否需互动合作 |
|------|------|------|
| "贷款+外部直投" | 外部投资机构直接投资 + 银行合作贷款 | 是，二者自行开展则不纳入 |
| "贷款+远期权益" | 外部投资机构持有远期权益 + 银行合作贷款 | 是 |
| "贷款+直投" | 集团投资公司自有资金直投 + 银行合作贷款 | 是 |
| "贷款+远期权益"（集团） | 集团持有远期权益 + 银行合作贷款 | 是 |

> 银行无论是否属于投贷联动试点银行（银监发〔2016〕14号），均需填报。

- S63（大中小微型企业贷款情况表）是 S70 的重要关联报表，用于校验专精特新中小企业贷款数据，但尚未在知识库中建立实体页，后续需优先补充。

- S70 与 S71 的表间校验关系（[1.1.D]≥S71_I[1.2A] 等）方向为 S70 ≥ S71，表明 S70 科技型企业贷款规模应大于等于 S71 中科技型小微企业贷款规模，符合逻辑（S70 覆盖全部科技企业，S71 仅覆盖小型微型）。
