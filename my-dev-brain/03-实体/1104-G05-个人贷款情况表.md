# 1104-G05-个人贷款情况表

> [!note] 页面角色
> 本页承载 G05《个人贷款情况表》的报表结论、核心口径、分组维度和关键规则。逐字段核对时回原文页。

## 基本信息

- 报表编码：G05（231版）
- 报表名称：个人贷款情况表
- 所属系统：[[03-实体/1104系统|1104系统]]
- 制定依据：《个人贷款管理暂行办法》、《贷款风险分类指引》及其他法律法规
- 报送频度：季报（第一批次，季后13日）
- 报送口径：境内汇总口径
- 数据单位：万元（保留两位小数）
- 填报币种：本外币合计人民币
- 原文页：[[01-资料库/1104系统/2026-04-22-G05-个人贷款情况表-填报说明-原文|G05 原文]]

## 核心定位

G05 是 1104 系统中专门监测个人贷款全貌的报表，覆盖个人贷款的用途、借款人年龄、授信额度、贷款期限、担保方式、支付方式等维度，用于个人贷款监测分析。它与 G01_Ⅶ（贷款投向分行业）形成表间校验，是理解个人贷款结构的核心报表。

## 关键口径

### 个人贷款定义

- 本表个人贷款反映填报机构报告期末发放的使用地为境内的个人贷款。
- 对外国人和港澳台人士发放的使用地为中国境内的个人贷款在此项目填报。
- 本表统计的是**个人非经营性贷款**（不含个人经营性贷款）。
- 个人经营性贷款由 [[03-实体/1104-S63_Ⅲ-各控股类型企业融资情况表|S63_Ⅲ]] 单独统计（[1.G] 个人经营性贷款）。

### 不良贷款认定

- 依据原银保监会《贷款风险分类指导原则》、《关于推进和完善贷款风险分类工作的通知》等文件，按五级分类认定。

### 逾期贷款认定

- 没有按照贷款合同规定的期限偿还本金或利息的各项贷款的本金余额。
- 如果一笔贷款本金和利息都逾期，按逾期时间长的期限进行填报。
- **按月分期还款的个人消费贷款**：逾期90天以内的，按照已逾期部分的本金余额填报；逾期91天及以上的，按照整笔贷款本金余额填报。

### 担保方式优先级

- 若贷款存在多种担保方式，遵循**抵（质）押担保方式优先**的原则。

## 分组维度

### 1.1 按贷款用途（个人非经营性贷款）

| 编号 | 用途 | 说明 |
|------|------|------|
| 1.1.1 | 信用卡 | 个人信用卡透支的垫款 |
| 1.1.1.1 | 汽车分期 | 信用卡应收账款中分期付款购买汽车形成的余额 |
| 1.1.1.2 | 房屋装修分期 | 信用卡应收账款中房屋装修分期付款形成的余额 |
| 1.1.2 | 汽车 | 向个人借款人发放的用于购买汽车的贷款 |
| 1.1.3 | 住房按揭贷款 | 为购买个人住房并以此套住房为抵押的贷款（不含以个人住房作其他用途、商业用房、商住两用房） |
| 1.1.4 | 房屋装修贷款 | 向个人借款人发放的用于房屋装修的贷款 |
| 1.1.5 | 大件耐用消费品贷款 | 用于购买金额较大、使用期限较长的大件耐用消费品的贷款 |
| 1.1.6 | 个人助学贷款 | 向在校学生或其直系亲属、法定监护人发放，用于支付在校期间基本费用（含国家助学贷款和一般商业性助学贷款） |
| 1.1.7 | 其他 | 除以上各用途之外的个人非经营性贷款 |

### 1.2 按年龄（个人非经营性贷款）

| 编号 | 年龄段 |
|------|--------|
| 1.2.1 | 25岁（含）以下 |
| 1.2.2 | 25岁-35岁（含） |
| 1.2.3 | 35岁-45岁（含） |
| 1.2.4 | 45岁-55岁（含） |
| 1.2.5 | 55岁以上 |

### 1.3 按授信额度（个人非经营性贷款）

- 授信总额 = 该借款人名下所有未结清的个人贷款（不含个人经营性）合同额度 + 借款人的信用卡总额度。

### 1.4 按贷款期限

| 编号 | 类型 | 定义 |
|------|------|------|
| 1.4.1 | 短期贷款 | 原始期限在一年及以下 |
| 1.4.2 | 中长期贷款 | 原始期限在一年以上 |

### 1.5 按担保方式

| 编号 | 类型 | 规则 |
|------|------|------|
| 1.5.1 | 信用贷款 | 多种担保方式时，抵（质）押优先 |
| 1.5.2 | 保证贷款 | 同上 |
| 1.5.3 | 抵（质）押贷款 | 优先 |

### 1.6 按支付方式

| 编号 | 类型 | 定义 |
|------|------|------|
| 1.6.1 | 个人自主支付 | 贷款人根据借款人提款申请将资金直接发放至借款人账户，由借款人自主支付给交易对象 |
| 1.6.2 | 受托支付 | 贷款人根据借款人提款申请和支付委托，将资金直接支付给交易对象 |

### 1.7 展期贷款

- 办理贷款展期业务的个人贷款在报告期末的本金余额。

### 1.8 贷款累放情况（不含信用卡）

- 年初至报告期末累计发放的个人贷款金额，不含信用卡。
- 1.8.1 住房按揭贷款：累计发放的个人住房按揭贷款金额。

### 1.9 贷款年化收益（不含信用卡）

- 当年累计发放个人贷款（不含信用卡）所获得的利息收入按年折算后的金额。
- 公式示例：10万元贷款，两个月产生1000元利息收入，年化后 = 0.1×12/2 = 0.6万元。
- **往年发放贷款的利息收入不在此项填报**。
- 1.9.1 住房按揭贷款：当年累计发放个人住房按揭贷款所获得的利息收入按年折算后的金额。

## 列结构

| 列 | 含义 |
|----|------|
| A | 余额（期末） |
| B | 不良余额 |
| C | 本期增加 |
| D | 核销 |

- A ≥ B（余额 ≥ 不良余额），适用于各行
- A ≥ C（余额 ≥ 本期增加），适用于除展期贷款（1.7）以外的各列
- C ≥ D（本期增加 ≥ 核销），适用于除展期贷款（1.7）以外的各列

## 当前结论

- G05 是 1104 系统中专门监测个人贷款全貌的报表，覆盖用途、年龄、授信额度、期限、担保方式、支付方式等维度。
- G05 统计的是**个人非经营性贷款**，个人经营性贷款由 S63_Ⅲ 单独统计。
- G05 与 G01_Ⅶ（贷款投向分行业）形成强表间校验：个人贷款合计余额、各用途余额、中长期贷款余额均分别相等。
- G05 与 S63_Ⅲ（各控股类型企业融资）形成表间校验：个人经营性贷款维度在两个报表间对齐。
- G05 与 G53 第VI部分形成表间校验：G53 县域+城市个人贷款合计 = G05 个人贷款合计（非经营性）+ 个人经营性贷款合计。
- G05 的报送频度为季报（季后13日），与 S63、S70、S71 表III、G01 等第一批次报表一致。
- 填报机构范围广泛，覆盖政策性银行、大型商业银行、股份制商业银行、城商行、民营银行、农商行、外资法人银行等全部类型银行业金融机构。
