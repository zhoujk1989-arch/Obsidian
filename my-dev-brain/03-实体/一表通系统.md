# 一表通系统

## 简介

一表通系统是当前知识库中新增纳入的监管类系统之一，当前已进入第一张基础信息类报表《1.1 机构信息》，后续可继续承接机构、人员、业务、指标等标准化数据项。

- 类型：监管数据采集系统
- 当前状态：已开始整理
- 当前已落地报表：
  - [[03-实体/一表通-1.1-机构信息|一表通-1.1-机构信息]]
- 当前已归档原文：
  - [[01-资料库/一表通系统/2026-04-16-1.1-机构信息-数据字典-原文|1.1 机构信息-数据字典-原文]]

## 当前认识

- 一表通系统当前进入的是“机构信息”基础主数据层，重点不是业务口径，而是字段定义、编码规范、格式约束和版本变更。
- 与 1104 这类报表型系统相比，一表通系统更适合先按“系统页 + 报表页 + 数据字典型报表实体页 + 原文页”的方式组织。
- 1.1 机构信息已经包含基础标识字段、机构属性字段、联系人字段和版本新增字段，可作为后续同类基础数据表的范式。
- 后续如果继续导入一表通其他表，优先在 [[04-综合/一表通系统-报表关系|一表通系统-报表关系]] 中补报表行，再决定是否需要抽主题页。
