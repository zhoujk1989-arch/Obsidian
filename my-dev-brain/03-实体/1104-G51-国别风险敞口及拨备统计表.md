# 1104-G51-国别风险敞口及拨备统计表

## 页面角色

本页承载 G51 报表的结论、重点规则、常见口径和表间校验关系。逐字段核对时回原文页。

## 简介

G51 是 1104 系统中国别风险管理的核心报表，按国家/地区维度收集银行业金融机构的国别风险债权、债务敞口及相关拨备情况。

- **系统**：1104系统
- **报表编码**：G51
- **报表名称**：国别风险敞口及拨备统计表
- **依据**：《银行业金融机构国别风险管理办法》
- **报送频度**：季报，季后40日内（与 G52 同为季后40日）
- **并表口径**：是
- **填报机构**：政策性银行、大型商业银行（含邮政储蓄银行）、股份制商业银行、城市商业银行、外资法人银行、外国银行分行
- **数据单位**：万元，保留两位小数
- **币种**：本外币合计人民币

## 核心概念

### 境外定义

- **境外**：除中华人民共和国以外的其他国家或地区，以及中国香港、澳门和台湾。
- **境外主体**：国家、营业场所在境外的金融机构和非金融企业、居住在境外的个人等。

### 风险转移

风险转移是 G51 的核心逻辑之一，指境外债务人通过担保、保险、信用衍生品、合格抵质押品等手段将债权风险从一国转移到另一国。

- **关键规则**：国别风险转入方所属国家或地区的国别风险评级必须优于转出国别风险评级。
- **净额填报**：风险转入减去风险转出后的净额。净转出为负数，净转入为正数。
- **排除项**：填报机构总行及其并表机构提供的担保不计入风险转移。

### 最终境外债权

最终境外债权 = 直接境外债权 + 风险转移净额

- 风险转移全额覆盖时：体现为对提供风险转移主体国家的债权
- 部分覆盖时：覆盖部分计入相应国家，未覆盖部分仍保留在直接债务人所在国家
- 中国大陆主体提供风险转移时：全额或部分覆盖均在最终债权中不再反映

### 国家分类与拨备比率

| 国家分类 | 代码 | 应计提拨备比率 |

| 低国别风险 | 1 | 0% |
| 较低国别风险 | 2 | 0%（原文公式写0.5%，但说明写0%） |
| 中等国别风险 | 3 | 5%（原文公式写15%） |

- **业务类型**：境外贷款、存放同业/拆放同业/买入返售、存放中央银行、有价证券和基金投资、金融衍生品、对非并表境外机构投资、其他境外债权

## 重点规则

### 债权国别归属规则

1. **离岸中心注册企业**：按主要经营活动所在国家确定国别，而非注册地。
2. **境外企业境内分公司**：不能独立承担法律责任的，视为境外主体，债权填列在总公司所在国家。
3. **中资银行境外分行**：直接境外债权填列在分行所在国家，按风险转移体现为对母行的最终境外债权。
4. **外资银行境内分行**：直接境外债权直接填列在母行所在国家，不作风险转移。

### 外币折算规则

- 美元、欧元、日元、港币：按报告期末最后一天国家外汇管理局公布的基准汇价折算。
- 其他货币：按美元对人民币基准汇率 × 国际外汇市场其他货币兑美元汇率套算。

### 主权债权定义

主权债权包括对境外主权主体的：
1. 贷款
2. 由境外主权主体提供担保的贷款
3. 境外主权主体发行的债券

境外主权主体指：主权国家及其政府机关、行使主权权力的联邦组成单位、国家机构或部门、国家代表。

### 一带一路统计

- [29] 参与共建一带一路项目数量：银行以贷款方式提供融资的项目数量
- [30] 签约金额：已签订未失效的贷款合同金额合计
- [31] 贷款余额：对一带一路项目融资余额
- 校验关系：[31] <= [1.1.2.1]+[1.2.1.2.1]+[1.2.2.2.1]（一带一路贷款余额不超过境外贷款合计）

## 常见口径

### 境外债权 vs 最终境外债权

- **直接境外债权**：按直接债务人所在国家统计，不考虑风险转移。
- **最终境外债权**：扣除风险转移后的债权，反映实际的国别风险暴露。
- 两者差异由【2.风险转移】字段体现。

### 跨境债权 vs 境外分支机构债权

- **跨境债权**：境内机构与境外主体的债权 + 境外机构与第三国客户的债权。
- **境外分支机构债权**：境外机构对其所在国（当地）客户的债权。
- 两者合计 = 直接境外债权。

### 当地货币债权 vs 非当地货币债权

- 仅适用于境外分支机构债权（【1.2】项下）。
- 区分标准：计价货币是否为当地政府的法定货币。

### 国别风险拨备计算

应计提国别风险拨备 = 应计提拨备比率 × 应计提拨备的国别风险资产

- 应计提拨备的国别风险资产 = 境外债权 + 担保承诺等表外业务（考虑风险转移和风险缓释）
- 如满足办法三十一条相关规定，可不计提国别风险准备。
