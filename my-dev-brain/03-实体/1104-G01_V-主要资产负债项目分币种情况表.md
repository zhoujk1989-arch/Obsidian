# 1104-G01_V-主要资产负债项目分币种情况表

## 简介

G01_V《主要资产负债项目分币种情况表》是 1104 系统中 G01 主表的附注性质补充报表，用于在 G01 汇总数据基础上，按美元、欧元、日元、港币、英镑五个币种分别列示主要资产负债项目的人民币折算金额。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统 G01_V
- 报表名称：主要资产负债项目分币种情况表
- 报表定位：G01 主表附注补充
- 当前原文依据：[[01-资料库/1104系统/2026-04-08-G01_V-主要资产负债项目分币种情况表-填报说明-原文|G01_V 主要资产负债项目分币种情况表填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、规则和常见口径

## 当前结论

- G01_V 是 G01 主表的"分币种细化视角"，不是独立报表，而是对 G01 数据的补充维度观察。
- 本表为空白报表时（没有外币业务），分币种列可以不填；有外币业务时，需按币种折算后填列。
- 本表的项目定义与 G01 主表保持一致，未单独列示的项目归入"其他资产/负债"合并填报。
- 本表仅做简单汇总，不考虑抵消分录和审计调整等特殊事项。

## 核心特性

| 特性 | 说明 |
|------|------|
| 报送频率 | 法人汇总数据（季报），季后18日 |
| 填报币种 | [A.]美元、[B.]欧元、[C.]日元、[D.]港币、[E.]英镑，均折算为人民币 |
| 汇率规则 | 外币资产/负债：期末外管局基准汇价；权益类：历史汇率；其他货币：美元套算 |
| 校验关系 | 表内满足会计恒等式（资产=负债+权益），表间 `G01_V[13.A-E合计] ≤ G01[25.B]` |

## 重点口径

### 报送口径

- **报送对象**：政策性银行、大型商业银行（含邮储）、股份制银行、城商行、民营银行、农商行、农合行、农信用社、村镇银行、外资法人银行、外国银行分行、直销银行、财务公司、信托公司、金融租赁公司、汽车金融公司、消费金融公司
- **报送频度**：季报（法人汇总），季后18日
- **数据单位**：万元（保留两位小数）

### 币种折算规则

| 项目类型 | 汇率确定方式 |
|----------|--------------|
| 外币资产（除权益类） | 报告期末最后一天外管局公布的基准汇价 |
| 权益类项目 | 按历史汇率，差额按会计制度处理 |
| 其他货币 | 期末美元基准汇率 × 当天9时国际外汇市场其他货币兑美元汇率 |

### 项目边界说明

| 项目 | 归属说明 | 注意事项 |
|------|----------|----------|
| 贷款 | [4.] | 不含贸易融资和贴现 |
| 贸易融资 | [5.] | 包括打包贷款、进口押汇、出口押汇、议付信用证、福费廷 |
| 贴现 | [6.] | 实质重于形式，风险转移才调整余额 |
| 保证金 | [21.]其他负债 | 单位/个人保证金均不归入存款类项目 |
| 央行回购 | [19.]卖出回购款项 | 不在[16.]向央行借款中反映 |

## 重点规则

### 表内校验关系

```text
[8.] = [8.1] + [8.2] + [8.3]           投资 = 债券 + 股票 + 其他
[11.] ≥ [11.1] + [11.2]                其他资产 ≥ 同业存单 + 衍生资产
[13.] = Σ(1-11) - [12.]                资产总计 = 资产合计 - 减值准备
[21.] ≥ [21.1] + [21.2]                其他负债 ≥ 发行同业存单 + 衍生负债
[22.] = Σ(14-21)                       负债合计
[25.] = [22.] + [23.] + [24.]          负债及所有者权益合计
[13.] = [25.]                          资产总计 = 负债及所有者权益合计（会计恒等式）
```

### 表间校验关系

```text
G01_V[13.A] + G01_V[13.B] + G01_V[13.C] + G01_V[13.D] + G01_V[13.E] ≤ G01[25.B]
```

> 说明：G01_V 分币种资产合计不得超过 G01 主表的资产总计，因为 G01_V 是 G01 的补充细化视角。

## 关联报表

### G01 主表

- **关系**：G01_V 是 G01 的附注补充表
- **作用**：在 G01 汇总数据基础上，增加币种维度观察
- **项目一致性**：G01_V 单独列示项目定义与 G01 主表一致

### G21 流动性期限缺口统计表

- **关系**：维度互补
- **G01_V → G21**：G01_V 的各币种资产可进一步按剩余期限划分（需参考 G21 的期限桶规则）
- **典型映射**：
  - G01_V[A-E列][4.贷款] → G21[1.贷款]（按剩余期限分桶）
  - G01_V[A-E列][8.投资] → G21[1.7 投资]（注意交易/银行账户区分）
