# 一表通-8.1-贷款借据

## 简介

表8.1贷款借据是一表通系统交易域中的贷款借据明细报表，用于采集填报机构信贷业务所对应的全部借据信息，包括个人贷款借据、对公贷款借据及委托贷款借据。

## 当前结论

1. **填报范围**：填报机构信贷业务所对应的借据信息，涵盖个人贷款借据、对公贷款借据及委托贷款借据。
2. **字段结构**：仅7个字段，结构极简，以借据ID为唯一标识，聚焦借据核心状态与金额。
3. **贷款状态码表 `YBT_DKZT`（6码值）**：01-正常 / 02-核销 / 03-转让 / 04-结清 / 05-逾期 / 06-其他。
4. **逾期标识码表 `YBT_SF`（2码值）**：0-否 / 1-是。
5. **借款余额口径**：包含逾期未还本金；已转让/核销贷款余额为0；贴现余额以填报机构实际采用的会计记账方式为准。
6. **减值准备口径**：包括专项准备、特种准备和一般准备三种；组合计提时按计提比例计算该笔贷款对应金额。
7. **利率口径**：填报年化利率的百分率值（如5表示百分之五）。

## 重点口径

- **贷款状态与逾期标识的关系**：贷款状态为"05-逾期"时，逾期标识应为"1-是"；但逾期标识为"1-是"时，贷款状态不一定是"05-逾期"（可能仍为"01-正常"但已逾期）。
- **已转让/核销贷款余额**：无论是否结清，已转让或核销的贷款余额统一填0。
- **利率填报格式**：以百分率数值填报，如年利率5%填"5"，不是"0.05"。
