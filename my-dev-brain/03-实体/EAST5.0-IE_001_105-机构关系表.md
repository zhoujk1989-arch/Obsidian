# EAST5.0-IE_001_105-机构关系表

## 简介

`IE_001_105` 机构关系表是 EAST5.0 系统中的机构基础信息表，用于报送银行机构之间的层级关系，包括总行、分行、支行等实体机构以及省级汇总、地市级汇总、法人汇总、分行汇总等汇总层级。

## 当前结论

- 本表用于描述银行机构之间的层级关系，核心字段为 `NBJGH`（内部机构号）+ `CJRQ`（采集日期）作为主键，通过 `SJGLNBJGH`（上级管理内部机构号）关联上级机构。
- 本表与 `IE_001_101` 机构信息表存在关联关系：本表的 `YHJGDM` 银行机构代码关联机构信息表的 `YHJGDM` 银行机构代码；本表的 `YHJGMC` 银行机构名称关联机构信息表的 `YHJGMC` 银行机构名称。
- 本表的 `SJGLJGDM`、`SJGLNBJGH`、`SJGLJGMC` 分别对应上级管理机构的银行机构代码、内部机构号和机构名称，没有上级机构的填写为 `0` 或空。
- 本表未引用任何码表，所有字段均为银行自定义描述。
- 报送范围应包括所有进行账务处理的机构，至少应包括总行、分行、支行等实体机构和省级汇总、地市级汇总、法人汇总、分行汇总。

## 字段分组

| 分组 | 代表字段 | 说明 |
| --- | --- | --- |
| 机构识别 | `YHJGDM` 银行机构代码、`NBJGH` 内部机构号、`JRXKZH` 金融许可证号 | 12位人行支付行号或SWIFT行号、内部唯一识别、金融许可证号 |
| 机构名称 | `YHJGMC` 银行机构名称 | 需填写全称，与公章/金融许可证登记名称一致 |
| 上级机构 | `SJGLJGDM` 上级管理机构代码、`SJGLNBJGH` 上级管理内部机构号、`SJGLJGMC` 上级管理机构名称 | 没有上级机构时代码/内部机构号填 `0`，名称填空，总行本级上级机构为总行法人名称 |
| 备注与采集 | `BBZ` 备注、`CJRQ` 采集日期 | 银行自定义备注，YYYYMMDD格式 |
