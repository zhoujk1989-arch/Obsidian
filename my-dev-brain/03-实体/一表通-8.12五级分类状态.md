# 一表通-8.12五级分类状态

## 简介

8.12五级分类状态报表采集包括贷款、信用卡在内的信贷资产及非信贷资产五级分类状态记录，用于反映资产风险分类的变动情况。

## 关键信息

| 项目 | 内容 |
|------|------|
| 报表编码 | 8.12 |
| 报表名称 | 五级分类状态 |
| 系统 | 一表通 |
| 字段总数 | 15 |
| 主键字段 | 协议ID、细分资产ID、机构ID、当前五级分类、采集日期（5个） |
| 强制性非空校验字段 | 机构ID、当前五级分类、减值准备、币种、采集日期（5个） |
| 2.1版新增字段 | 审查员工ID、审批员工ID、币种、备注 |
| 2.1版修改字段 | 协议ID（长度anc..60→anc..100）、细分资产ID（长度+说明）、经办员工ID（说明） |

## 数据域

信贷资产风险分类 / 五级分类

## 关键字段与分组

### 核心分类字段
- **当前五级分类**（H120005）：01正常/02关注/03次级/04可疑/05损失/00未分级，强制性校验
- **原五级分类**（H120006）：分类变动前状态，正常借据可置空
- **变动方式**（H120007）：01人工/02自动
- **变动原因**（H120008）：银行自定义，未变动填报"未变动"

### 资产标识字段
- **协议ID**（H120001）：信用卡填账号，投资业务填协议ID，无协议置空
- **细分资产ID**（H120002）：借据ID/投资标的ID/同业业务ID等
- **机构ID**（H120003）：金融许可证号前11位+内部机构号

### 时间与金额
- **调整日期**（H120004）：五级分类调整实际日期
- **采集日期**（H120013）：数据采集日期，强制性校验
- **减值准备**（H120014）：计提减值准备金额，无需计提填0，强制性校验

### 人员与辅助
- **经办员工ID**（H120009）/ **审查员工ID**（H120010）/ **审批员工ID**（H120011）
- **币种**（H120012）：GB/T 12406-2022 三位编码，强制性校验
- **备注**（H120015）：2.1版新增

## 码值说明

### 五级分类码值
| 码值 | 含义 | 定义 |
|------|------|------|
| 01 | 正常 | 债务人能够履行合同，没有客观证据表明本金、利息或收益不能按时足额偿付 |
| 02 | 关注 | 存在一些可能产生不利影响的因素，但债务人目前有能力偿付 |
| 03 | 次级 | 债务人无法足额偿付本金、利息或收益，或金融资产已发生信用减值 |
| 04 | 可疑 | 债务人已经无法足额偿付，金融资产已发生显著信用减值 |
| 05 | 损失 | 采取所有可能措施后只能收回极少部分或损失全部 |
| 00 | 未分级 | 未进行五级分类的金融资产 |

### 变动方式码值
| 码值 | 含义 |
|------|------|
| 01 | 人工 |
| 02 | 自动 |

## 一表通与EAST差异

- **五级分类码值**：一表通填报数字码值，EAST填报中文码值，需转换
- **日期格式**：一表通为YYYY-MM-DD（10位），一表通内部为YYYYMMDD（8位），需转换
- **机构ID**：一表通需要拼接金融许可证号前11位
