# 1104-G11_I-资产质量五级分类情况表（第Ⅰ部分）

## 简介

G11_I 是 G11《资产质量五级分类情况表》的第Ⅰ部分"按行业分类的贷款（按贷款投向）"，专门反映填报机构发放境内各项贷款的行业投向分布及资产质量五级分类情况，覆盖各项贷款、逾期贷款、重组贷款、展期贷款和并购贷款等核心项目。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统G11\_I号
- 报表名称：资产质量五级分类情况表（第Ⅰ部分）
- 当前原文依据：[[01-资料库/1104系统/2026-04-22-G11_I-资产质量五级分类情况表（第Ⅰ部分）-填报说明-原文|原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- G11_I 是 1104 系统中专门反映**贷款投向行业分布**的核心报表，与 G11_II（资产质量及准备金）互为补充，两者通过"各项贷款"项目形成强校验。
- 填报频度：**季报**（法人机构汇总数据），季后 13 日内。
- 填报机构范围广泛，但**货币经纪公司不需要报送第Ⅰ部分**。
- 核心填报原则：贷款行业投向按**资金实际用途**填报，而非借款人所属行业。
- 报表覆盖 20 个国民经济行业分类（GB/T4754-2017），并细分为个人贷款（信用卡/汽车/住房按揭/助学贷款/其他）等子项。
- 新增 261 版并购贷款细分：控制型并购贷款 + 参股型并购贷款，含境外并购贷款子项。

## 重点口径

### 五级分类定义（与 G11_II 共享）

| 类别 | 定义 |
|------|------|
| 正常类 | 债务人能够履行合同，没有客观证据表明本金、利息或收益不能按时足额偿付 |
| 关注类 | 虽然存在可能对履行合同产生不利影响的因素，但债务人目前有能力偿付 |
| 次级类 | 债务人无法足额偿付本金、利息或收益，或金融资产已经发生信用减值 |
| 可疑类 | 债务人已经无法足额偿付本金、利息或收益，金融资产已发生显著信用减值 |
| 损失类 | 在采取所有可能的措施后，只能收回极少部分金融资产，或损失全部金融资产 |

- **正常资产** = 正常类 + 关注类
- **不良资产** = 次级类 + 可疑类 + 损失类
- **逾期资产**：独立于五级分类，按合同期限兑付情况统计

### 贷款行业投向填报原则

- **按资金实际用途**填报，而非借款人所属行业。
- 根据贷款合同中约定的资金用途填报。
- 如果无法合理确定贷款投向，按借款人**主营业务所属行业**分类。
- 个人经营性贷款也按此原则填报。

### 逾期贷款填报规则

- 逾期贷款按本金或利息逾期时间分 7 档统计。
- **按月分期还款的个人消费贷款**：逾期 90 天内按已逾期部分本金余额填报；逾期 91 天及以上按整笔贷款本金余额填报。
- **借新还旧/展期贷款**：逾期天数从原借款合同到期日起算（借新还旧展期）或从展期后到期日起算（正常展期）。
- 本金和利息都逾期时，按**逾期时间长的期限**填报。

### 重组贷款认定规则

- 重组贷款核心条件：**债务人发生财务困难** + 商业银行作出有利于债务人的合同调整。
- 债务人财务困难情形包括：本金利息逾期、偿债能力下降预计现金流不足、债务已被分为不良、无法在其他银行以公允价格融资、证券退市风险等。
- 合同调整包括：展期、宽限期、利息转本金、降低利率、减少偿付、押品置换、置换等。
- **重组贷款不应划分为正常类**，若特殊情况导致正常类非 0，需向监管员解释。
- 多次重组后未计入不良贷款的余额需书面说明原因。

### 并购贷款定义（261版新增）

- 并购贷款用于支付并购交易价款和费用。
- **控制型并购贷款**：实现实际控制、合并或参股目标企业/资产。
- **参股型并购贷款**：实现参股目标企业/资产。
- **境外并购贷款**：境内并购方通过受让股权、认购新增股权、收购资产或承接债务等方式，实际控制/合并/参股**境外**目标企业或资产。

## 重点规则

### 表内核对关系

| 关系 | 说明 |
|------|------|
| A = B + E = C + D + F + G + H | 列维度：账面余额 = 正常 + 不良 = 正常类 + 关注类 + 次级类 + 可疑类 + 损失类 |
| B = C + D | 正常资产 = 正常类 + 关注类 |
| E = F + G + H | 不良资产 = 次级类 + 可疑类 + 损失类 |
| 1. = 2. + 3. = 第Ⅱ部分1. | 各项贷款 = 境内 + 境外 = 第Ⅱ部分各项贷款（适用于 A-H 列） |
| 2. = 2.1 + 2.2 + ... + 2.23 | 对境内贷款 = 各行业分类合计 |
| 4. = 4.1 + 4.2 + ... + 4.7 | 逾期贷款 = 各逾期档合计 |
| 4. ≤ 1. | 逾期贷款不超过各项贷款 |
| 5.5 = 5.1 + 5.2 - 5.3 - 5.4 | 期末重组贷款 = 年初 + 新重组 - 不再认定 - 收回等（仅 A 列） |
| 8. = 8.1 + 8.2 | 并购贷款 = 控制型 + 参股型 |
| 8.1.1 ≤ 8.1 | 境外控制型并购贷款 ≤ 控制型并购贷款合计 |
| 8.2.1 ≤ 8.2 | 境外参股型并购贷款 ≤ 参股型并购贷款合计 |
