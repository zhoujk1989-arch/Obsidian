# 1104-G14_IV-非同业集团客户及经济依存客户大额风险暴露情况表

## 简介

G14_IV 是 1104 系统 G14《大额风险暴露统计表》的第 IV 部分，专门反映银行非同业集团客户及经济依存客户中大额风险暴露客户明细情况，用于监测银行对非同业集团客户和经济依存客户的风险暴露集中程度。

- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- G14_IV 是 G14 系列大额风险暴露报表家族中专门针对"非同业集团客户 + 经济依存客户"两类特殊客户群体的明细报表。
- 本部分客户仅填报考虑合格风险缓释作用后风险暴露超过一级资本净额 2.5% 的客户。
- 客户数量上限为 70 家：不超过 70 个则填报全部，超过 70 个只填报前 70 家（按 D 列降序）。
- 同一单一法人客户同时属于非同业集团客户和经济依存客户的，按两类客户分别填报。
- 填报机构范围：政策性银行、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、农村信用社、村镇银行、外资法人银行。**企业集团财务公司、消费金融公司、农村资金互助社不需填报第 IV 部分**。
- 季报（法人汇总：季后 18 日内；合并报表：半年报：半年后 40 日内）。
- 表间校验：71.D = G40_[2.A]（一级资本净额强校验）。

## 重点口径

- **大额风险暴露客户判定标准**：仅填报考虑合格风险缓释作用后风险暴露超过一级资本净额 2.5% 的客户。
- **客户数量上限**：如果大额风险暴露客户不超过 70 个，填报全部；超过 70 个只填报前 70 家，按 D 列降序排列。
- **双重身份客户**：同一单一法人客户同时属于非同业集团客户和经济依存客户的，按两类客户分别填报（即可能占 2 行）。
- **客户类型分类**：仅填报两类——非同业集团客户、经济依存客户。

## 重点规则

### 风险暴露分类口径

- **一般风险暴露（F）** = 各项贷款（F1）+ 债券投资（F2）+ 其他一般风险暴露
- **特定风险暴露（G）** = 资产管理产品（G1）+ 资产证券化产品（G2）
- 其中资产管理产品进一步细分为：信托产品（G11）、非保本理财（G12）、证券业资管产品（G13）
- **交易账簿风险暴露（H）**：详见 G14_I 相应指标填报说明
- **交易对手信用风险暴露（I）**：详见 G14_I 相应指标填报说明
- **潜在风险暴露（J）** = 银行承兑汇票（J1）+ 跟单信用证（J2）+ 保函（J3）+ 贷款承诺（J4）+ 其他潜在风险暴露

### 风险暴露汇总口径

- **考虑风险缓释作用的风险暴露总和（D）** = F + G + H + I + J + K
- **不考虑风险缓释作用的风险暴露总和（M）** = D + L（L 为风险缓释转出，转入为负数）

### 比例计算口径

- **风险暴露占比（E）** = D / 71.D（一级资本净额）
- **不可豁免风险暴露占比（E1）** = D1 / 71.D
