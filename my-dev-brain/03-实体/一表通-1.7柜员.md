## 当前结论

- 本表采集填报机构所有柜员，包括实体柜员和虚拟柜员。
- 虚拟柜员必须有柜员号，员工ID允许为空。
- 机构ID、柜员号、采集日期是理解本表的核心定位字段；岗位编号用于连接岗位信息，员工ID用于连接员工信息。
- 字段全量清单保留在原文页，本页只保留影响报送、关联和口径判断的字段规则。

## 关键字段规则

| 字段/字段组 | 规则 |
| --- | --- |
| 机构ID（A070001） | 前11位与金融许可证号相同，后13位为内部机构号。 |
| 柜员号（A070002） | 实体柜员和虚拟柜员均需填写。 |
| 员工ID（A070003） | 实体柜员用于关联员工；虚拟柜员允许为空。 |
| 柜员类型（A070004） | 银行自定义，需体现具体虚拟柜员类型，如自助银行、网银等。 |
| 是否实体柜员（A070012） | `0`=否，`1`=是。 |
| 岗位编号（A070005） | 与岗位信息关联；同一员工多个岗位时用英文半角分号隔开。 |
| 柜员权限级别（A070006） | `01`=高柜，`02`=低柜，无法枚举时用 `00-XX` 自定义。 |
| 柜员状态（A070008） | `01`=在岗，`02`=离岗，无法枚举时用 `00-XX` 自定义。 |
| 上岗日期（A070007） | 填柜员就任本岗位日期，不是员工入职日期；默认值为 `9999-12-31`。 |
| 失效日期（A070009） | 柜员状态转为注销或无效的日期。 |
| 备注（A070010） | 多事项用英文半角分号隔开。 |
| 采集日期（A070011） | 数据表字段信息的采集日期，格式为 `YYYY-MM-DD`。 |
