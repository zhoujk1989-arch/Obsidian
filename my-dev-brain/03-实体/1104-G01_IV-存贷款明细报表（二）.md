# 1104-G01_IV-存贷款明细报表（二）

## 简介

G01_IV《存贷款明细报表（二）》是 1104 系统中的存贷款期限结构专项报表，核心功能是收集填报机构存款、贷款原始期限情况，用以反映资产负债期限结构和流动性情况。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统G01_IV
- 报表名称：资产负债项目表附注第Ⅳ部分：存贷款明细报表（二）
- 当前原文依据：[[01-资料库/1104系统/2026-04-08-G01_IV-资产负债项目表附注-填报说明-原文|G01_IV 存贷款明细报表（二）填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、规则和常见口径

## 当前结论

- G01_IV 以本外币合计人民币（C列）填报
- 核心结构为"9类期限 + 3组关键项目（A/B/C）+ 互联网存款附注"
- 填报原则：按原始（合同）期限分类，不考虑提前支取/还款
- 适用机构：15类机构（政策性银行、大型银行、股份制银行、城商行、农商行、外资法人银行等）

### 期限分类规则

| 项目 | 规则说明 |
|------|----------|
| 基本原则 | 按原始（合同）期限分类，不考虑提前支取/还款 |
| 逾期处理 | 超90天非应计贷款仍按原始期限填列 |
| 整数边界 | 含上限不含下限（如"一年至二年"=含二年不含一年） |
| 时间单位 | 月、年按自然月/年统计 |
| 特殊贷款 | <ul><li>贴现/转贴现：贴现日距到期日时间</li><li>分期付款：合同起始日到最终到期日</li><li>分笔发放：按每笔放款日或合同最后到期日</li></ul> |

### 互联网存款口径

| 项目 | 说明 |
|------|------|
| 定义 | 不通过网点/柜面/开卡机等实体渠道，远程开立II类账户吸收的存款 |
| 排除范围 | I类户客户关联的II类账户不计入 |
| 细分项 | <ul><li>[11.1] 个人定期存款</li><li>[11.2] 个人活期存款</li><li>[11.a] 第三方平台引流存款</li><li>[11.a.1] 第三方平台个人定期存款</li><li>[11.a.2] 第三方平台个人活期存款</li></ul> |

## 重点规则

### 表内校验关系

```text
[10.] = [1.] + [2.] + [3.] + [4.] + [5.] + [6.] + [7.] + [8.] + [9.]  （九类期限之和）
[A] >= [B]    （各项存款 >= 储蓄存款）
[11.] = [11.1] + [11.2]    （互联网存款 = 定期 + 活期）
[11] >= [11.a]    （互联网存款 >= 第三方平台存款）
```

### 表间校验关系

```text
[10.A] = G01_[61.C]    （各项存款本外币合计 = G01_[61.C]）
[10.B] = G01_[27.C]    （储蓄存款本外币合计 = G01_[27.C]）
[10.C] = G01_[62.C]    （各项贷款本外币合计 = G01_[62.C]）
```
