# 1104-G01_Ⅵ-托管业务情况表

## 简介

G01_Ⅵ《托管业务情况表》是 1104 系统中的专项报表，用于反映填报机构的托管业务情况。托管是商业银行作为独立第三方，按照法律法规规定和托管合同约定，为所托管产品在中华人民共和国境内提供财产保管及相关服务的行为。

**重要说明**：资金监管业务不属于托管业务，但在本表中列报。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统 G01_Ⅵ号
- 报表名称：托管业务情况表
- 当前原文依据：[[01-资料库/1104系统/2026-04-08-G01_Ⅵ-托管业务情况表-填报说明-原文|G01_Ⅵ 托管业务情况表填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、规则和常见口径

## 当前结论

- G01_Ⅵ 是托管业务专项报表，按托管产品类型分11+1项列示。
- 填报对象不包括政策性银行、农村合作银行、农信用社、村镇银行、民营银行等。
- 重要说明：资金监管业务不属于托管业务，但在本表中列报（[13.]）。
- 私募投资基金托管按净资产余额填报，不按合同金额。

## 重点口径

### 报送口径

| 项目 | 内容 |
|------|------|
| 报送频率 | 月报（境内分支机构汇总），月后6日 |
| 填报币种 | 本外币合计人民币 |
| 数据单位 | 万元（保留两位小数） |

### 填报机构

大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、农村商业银行、外资法人银行、外国银行分行、信托公司。

### 托管业务分类（11项）

| 编号 | 项目 | 说明 |
|------|------|------|
| [1] | 证券投资基金托管 | 封闭式、开放式证券投资基金 |
| [2] | 基金公司客户资产管理托管 | 基金管理公司私募资管计划、资产支持专项计划 |
| [3] | 证券公司客户资产管理托管 | 证券公司私募资管计划、资产支持专项计划 |
| [4] | 银行理财托管 | 商业银行及理财子公司理财产品 |
| [5] | 信托财产保管 | 证券投资类、权益投资类、信托贷款类信托计划 |
| [6] | 私募投资基金托管 | 分[6.1]证券、[6.2]股权、[6.3]其他（净资产余额） |
| [7] | 保险资产托管 | 保险资金运用、保险资管产品 |
| [8] | 养老金托管 | 分[8.1]企业年金、[8.2]职业年金、[8.3]养老金产品 |
| [9] | QDII类产品托管 | QDII、RQDII、QDLP、QDIE等"走出去"投资 |
| [10] | QFI类产品托管 | QFII、RQFII、QFLP、GDR等"走进来"投资 |
| [11] | 其他产品托管 | 期货、债转股投资计划等 |

### 重要说明

| 项目 | 说明 |
|------|------|
| 资金监管业务 | 不属于托管业务，但在本表[13.]中列报 |
| 私募基金统计 | 按净资产余额填报，不按合同金额（保险私募、QDLP/QDIE、QFLP不包含） |
| 保险资管产品 | 包括组合类产品、保险债权计划、保险股权计划、保险私募基金、保险资产支持计划 |
| 养老金产品统计 | 必须向人社部产品备案 |
