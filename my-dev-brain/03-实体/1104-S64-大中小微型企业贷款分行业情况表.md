# S64-大中小微型企业贷款分行业情况表

## 简介

S64《大中小微型企业贷款分行业情况表》是 1104 系统监管报表之一，旨在收集各银行在报告日对小微型企业贷款按行业分类的有关情况。报表包含第I部分（大中小微型企业贷款分行业情况）和第II部分（大中小微型企业不良贷款分行业情况），按行业维度×企业规模维度交叉展示贷款余额、发放额、不良等指标。季报（季后13日），与 S63 形成表间校验闭环，并与 G01_Ⅶ、S71 存在表间校验关系。

## 关键信息

- 所属系统：[[03-实体/1104系统|1104系统]]
- 报表编码：金监统S64
- 报表名称：大中小微型企业贷款分行业情况表
- 当前原文依据：[[01-资料库/1104系统/2026-04-20-S64-大中小微型企业贷款分行业情况表-填报说明-原文|S64大中小微型企业贷款分行业情况表填报说明-原文]]
- 知识沉淀位置：本页直接维护该报表的提炼结论、重点规则和常见口径

## 当前结论

- S64 是 1104 系统中以"行业"为核心维度的企业贷款报表，弥补了 S63（按企业规模汇总）和 S65（按地区汇总）缺乏行业分布视角的空白。
- S64 与 S63、S65 共同构成"规模维度 × 地区维度 × 行业维度"的三维企业贷款校验框架。
- S64 包含两部分：第I部分为大中小微型企业贷款分行业情况，第II部分为大中小微型企业不良贷款分行业情况，两部分均为季报（季后13日）。
- S64 的行业分类按照 GB/T 4754-2017《国民经济行业分类》标准执行，企业从事两种或以上经济活动时按增加值份额最大的主要活动确定行业。
- S64 第I部分有 20 个行业分类行（[1.1]-[1.20]），加上合计行 [1] 共 21 行。
- S64 第I部分的列维度为 A-H（8列），覆盖贷款余额、当年累计发放、不良贷款等指标。
- S64 第I部分与 S63_I[1] 存在表间相等校验关系（不含E列），第II部分与 S63_I[1.1.3]+S63_I[1.1.4]+S63_I[1.1.5] 存在表间相等校验关系（不含E列）。
- S64_I[1.E]（小微企业贷款余额）≤ S71_I[1.A0]+[3.A0]（普惠型小微企业贷款+普惠型个体工商户和小微企业主贷款），体现 S64 小微企业口径 ≥ S71 普惠型口径的关系。
- S64_I 各行业贷款合计（A/B/C/D/F列）≤ G01_Ⅶ[1.各项贷款A]，体现行业贷款与 G01_Ⅶ 贷款投向的校验关系。
- S64 包含首贷户统计（当年首次贷款的首贷户户数、当年首贷户首次贷款金额、涉农首贷户），这是 S64 区别于 S63/S65 的特殊统计项。
- S64 第I部分 [6.1.1]≤[6.1]（普惠型小微企业贷款中的信用贷款 ≤ 普惠型小微企业贷款），[6.2.1]≤[6.2]（普惠型小微企业贷款中的中长期贷款 ≤ 普惠型小微企业贷款）。

### 填报机构

政策性银行(含开发银行)、大型商业银行（含邮储银行）、股份制商业银行、城市商业银行、民营银行、农村商业银行、农村合作银行、外资法人银行、外国银行分行、直销银行、农村信用社、村镇银行、贷款公司、农村资金互助社、企业集团财务公司、信托公司、金融租赁公司、汽车金融公司。

### 行业分类标准

- 按照 GB/T 4754-2017《国民经济行业分类》执行
- 企业从事一种经济活动：按该活动确定行业
- 企业从事两种或以上经济活动：按增加值份额最大的主要活动确定；如难确定，则依据销售收入、营业收入或从业人员确定

### 企业划型标准

- 按照工信部联企业〔2011〕300号《中小企业划型标准规定》执行
- 企业法人按上述标准执行，企业非法人按授权法人执行
- 银行业金融机构应随时跟踪企业情况，及时调整划型分类

### 贷款口径

- 主要包括：贷款、贸易融资、贴现、融资租赁、从非金融机构买入返售资产、透支、各项垫款等
- **不含**：买断式转贴现和买断其他票据类资产

### 首贷户统计

| 项目 | 说明 |
|------|------|
| 首贷户定义 | 从银行业金融机构首次获得贷款的客户，通过人行征信系统查询无征信记录 |
| 当年首次贷款的首贷户户数 | 截至报告期末当年首次发放贷款的首贷户户数合计 |
| 当年首贷户首次贷款金额 | 截至报告期末当年对首贷户首次发放贷款的金额合计 |
| 首贷户再次贷款 | 某一首贷户首次贷款结清后，再次发放的贷款不计入首贷户统计 |
| 涉农首贷户 | 注册地位于农村区域，或注册地位于城市区域但贷款用于农林牧渔业及支农等用途的首贷户 |

> 首贷户示例：1月向首贷户A首次发放100万，2月向A发放200万（非首贷），3月向首贷户B首次发放50万。则1月末/2月末/3月末的首贷户户数分别为1/1/2户，首贷金额分别为100/100/150万元。

行校验：
[1] = [1.1]+[1.2]+[1.3]+[1.4]+[1.5]+[1.6]+[1.7]+[1.8]+[1.9]+[1.10]+[1.11]

列校验：
[F.] ≥ [G.] + [H.]，适用于 1、1.1-1.20 各行

```text
行校验：
[1] = [1.1]+[1.2]+...+[1.20]，适用于 A-H 列

列校验：
[F.] ≥ [G.] + [H.]，适用于 1、1.1-1.20 各行
```

| 关系 | 说明 |
|------|------|
