## 业务链路摘要

- 按 `原始材料/业务需求/EAST5.0/060_自营资金交易信息表.md` 的字段映射，将一表通来源表加工为 EAST5.0 `自营资金交易信息表`。
- 表级规则（Excel第1464行）：三部分 UNION ALL：
  1. **投资交易**（主表 T_7_7）→ LEFT JOIN T_9_2(投融资标的)/T_8_8(投资情况)/T_6_21(投资协议)/T_5_1(产品业务基本信息,自营标识='01')/T_1_1(机构信息) → 过滤交易金额>=0
  2. **同业交易**（主表 T_7_6）→ LEFT JOIN T_8_7(同业存量情况)/T_9_2(投融资标的)/T_5_1(产品业务基本信息,自营标识='01')/T_1_1(机构信息) → 过滤非同业存放(!=08) + 非结算性存放同业(!=07020)
  3. **融资交易**（主表 T_7_12）→ LEFT JOIN T_8_9(融资情况)/T_9_2(投融资标的)/T_1_1(机构信息)/T_4_2(科目信息) → 过滤非大额存单(NOT IN '011')
- SQL 草案采用按 `P_DATA_DATE` 清理后重插的方式；增量边界按当月采集日期过滤。
- 2026-05-10 重构校准：消除全部 NULL 占位和 ON 1=1 占位，补齐全部 JOIN 条件、字段映射、码值 CASE 转换、日期格式转换、金额 CAST、备注多源拼接。
