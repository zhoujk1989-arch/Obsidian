## 1. 目标

- 目标 EAST 表：`关联关系表`

## 4. 字段级取数逻辑说明

### 4.1 日期类字段

- `采集日期`：默认值：报告日，数据格式转成yyyymmdd

### 4.2 码值/枚举转换字段

- `关系类型`：码值转换：按照如下进行码值转换：
01 -> 母公司
02 -> 子公司
03 -> 与该企业受同一母公司控制的其他企业
04 -> 与该企业实施共同控制的投资方
05 -> 对该企业施加重大影响的投资方
06 -> 该企业的合营企业
07 -> 该企业的联营企业
08 -> 该企业的主要投资者个人及与其关系密切的家庭成员
09 -> 该企业或其母公司的关键管理人员及与其关系密切的家庭成员
10 -> 该企业主要投资者个人、关键管理人员或与其关系密切的家庭成员控制、共同控制或施加重大影响的其他企业
11 -> 供应链上下游
12 -> 担保关系
00-XX -> 其他-XX
- `关联人类别`：按如下码值转换：
01 -> 自然人
02 -> 国有企业
03 -> 民营企业
04 -> 政府机关
05 -> 事业单位
06 -> 社会团体
07 -> 境外机构
00-XX -> 其他-XX
- `关联人证件类别`：按照码值表进行转换，1999-XX和2999-XX转为其他-XX
