## 4. 字段级取数逻辑说明

### 4.1 日期类字段

- `票据出票日期`：加工映射：日期转YYYYMMDD格式
- `票据到期日期`：加工映射：日期转YYYYMMDD格式
- `贴现日期`：加工映射：日期转YYYYMMDD格式
- `采集日期`：加工映射：日期转YYYYMMDD格式

### 4.2 码值/枚举转换字段

- `票据状态`：代码转化：CASE WHEN T1.PJZT = '01' THEN '正常'
      WHEN T1.PJZT = '02' THEN '卖断'
      WHEN T1.PJZT = '03' THEN '解付'
      WHEN T1.PJZT = '04' THEN '垫款'
      WHEN T1.PJZT = '05' THEN '核销'
当票据状态为'00-XX'时，赋值为'其他-XX'，其中'XX'为银行自定义
