## SQL 加工逻辑（2026-05-09 重构后）

### 源表
- `T_7_8`（一表通不良资产处置）：主表，提供核销基础字段（细分资产ID/协议ID/币种/资产类型/处置本金金额/处置日期等）
- `T_1_1`（一表通机构信息）：通过机构ID截取关联，获取金融许可证号

### 关联条件
1. **LEFT JOIN subq_agg（子查询）**：
   - `TRIM(src.G080003) = TRIM(sa.XFZC_ID)` — 细分资产ID
   - `TRIM(src.G080004) = TRIM(sa.XY_ID)` — 协议ID
   - `TRIM(src.G080022) = TRIM(sa.BZ)` — 币种
   - `TRIM(src.G080006) = TRIM(sa.ZCLX_CODE)` — 资产类型
   - 子查询已按分组取最新记录（rn=1）

2. **LEFT JOIN T_1_1 org**：
   - `SUBSTR(TRIM(src.G080002), 12) = SUBSTR(TRIM(org.A010001), 12)` — 机构ID从第12位截取
   - `org.A010020 = V_DATA_DATE` — 采集日期=跑批日期

### WHERE 过滤条件
- `src.G080023 <= V_DATA_DATE`：采集日期不晚于跑批日期（全量表范围）
