# regulatory-data-lab 智能体总章程

本仓库是一个由 LLM 持续维护的监管报表知识库，用来把监管报表 SQL 文件、表结构结论和本地技术理解沉淀为：

1. 业务可读的自然语言口径
2. 可持续维护的字段与表级血缘
3. 可复用的报表、数据表、系统和概念知识

本次重构后，业务主目录固定为：

- `报表业务口径/`
- `概念/`
- `数据表/`
- `血缘/`
- `sql/`
- `模板/`

运维层保留在根目录和隐藏目录中：`.agents/`、`工具/`、`AGENTS.md`、`CLAUDE.md`、`索引.md`、`日志.md`、`说明.md`。

SQL 文件统一放入 `sql/<系统名>/`。本仓库不再维护独立，不再保留原始 SQL、表结构、业务需求或语法参考材料目录。

具体规则以 `AGENTS.md` 为准。

Hermes/Claude 类智能体执行血缘影响分析、血缘维护、SQL 来源去向维护或上下游影响分析时，应同时引用 `.agents/skills/regulatory-data-lab-lineage/`。
