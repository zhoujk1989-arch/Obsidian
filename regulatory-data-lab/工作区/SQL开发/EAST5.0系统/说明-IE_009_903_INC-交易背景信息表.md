## 4. 表级规则

```text
### 2.1 表级规则（Excel第 1349 行）

主表：【商业单据】T1
过滤条件：采集日期 = 报告日
内关联子查询：（
select 【协议与单据对应关系】所有字段
from 【协议与单据对应关系】 T1
left join 【贷款协议】 DK
on T1.协议ID = DK.协议ID  and T1.采集日期 = DK.采集日期 and DK.贷款协议起始日期 在本月
left join 【协议与单据对应关系】LST 上月末 协议ID、单据ID
on T1.协议ID = LST.协议ID and T1.单据ID = LST.单据ID
where T1.采集日期 = 报告日 且 （DK.协议ID非空 或 LST.协议ID为空）   /*本月生效合同 或 上月未报送过*/ 
）T11
关联条件：T1.单据ID = TT1.单据ID 且 T1.协议ID = TT1.协议ID 且 T1.采集日期 = TT1.采集日期 且 TT1.业务种类 in （'03','08','09','10','00'） /*限制为保理融资、承兑汇票、保函、信用证、其他业务*/
```
