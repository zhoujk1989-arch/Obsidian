## 7. 与旧版草案的主要差异

| 差异项 | 旧版 | 新版 |
| --- | --- | --- |
| JOIN 条件 | `ON 1 = 1` + TODO 占位 | 已按业务需求文档逐一补齐 |
| 码值转换 | 部分字段仍为裸值 | 全部 58 个字段码值已转为 CASE |
| 利率类型 | 错误转为 DECIMAL | 改为 VARCHAR 码值转换 |

| 还款方式 | NULL 占位 | 从 T_8_15 还本方式实现 |
| 五级分类去重 | 无 | 窗口函数 ROW_NUMBER 取最新 |
| 展期次数汇总 | 无 | 子查询 SUM 按借据ID分组 |
| 授信情况取最新 | 无 | 窗口函数 ROW_NUMBER 取最新 |
| 备注拼接 | 仅取 T_6_27 备注 | CONCAT_WS 拼接 5 个来源备注 |
| 筛选条件 | 无 | 上月末+本期贷款状态 IN ('01','05') |
| 上月末贷款状态 | 无 | 计算上月末日期，EXISTS 子查询 |
| 校验 SQL | 5 项检查 | 15 项检查（含码值、金额、缺口字段） |
