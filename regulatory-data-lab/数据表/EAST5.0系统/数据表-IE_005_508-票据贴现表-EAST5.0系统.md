# 数据表-IE_005_508-票据贴现表-EAST5.0系统

## 系统归属

- 所属系统：EAST5.0系统
- 所属库/域：EAST5.0 监管采集接口/数据字典域
- 是否存在跨系统同名表：待确认
- 文件路径是否位于正确系统目录：是

## 业务含义

- `IE_005_508` 是 EAST5.0 `票据贴现表` 的本地表结构定义，表注释为"票据贴现表"。
- 本表当前按 `监管采集接口表` 维护，主要用于记录监管采集字段、主键线索和字段填报说明。
- SQL 草案已实现从一表通 T_6_13（票据协议）和 T_1_1（机构信息）的完整映射，含码值转换、日期格式转换和 SUBSTR 加工。

## 主键与核心标识

- `CJRQ`：采集日期，字段注释标识为 `PK`。
- `PJHM`：票据号码，字段注释标识为 `PK`。
- `XDJJH`：信贷借据号，字段注释标识为 `PK`。
- `XDHTH`：信贷合同号，字段注释标识为 `PK`。
- `NBJGH`：内部机构号，字段注释标识为 `PK`。
- `BZ`：币种，核心归属或业务标识字段。
- `YHJGMC`：银行机构名称，核心归属或业务标识字段。
- `JRXKZH`：金融许可证号，核心归属或业务标识字段。

## 直接上游对象

- [[数据表-T_6_13-票据协议-一表通系统]]：主来源表，提供贴现业务全部字段。
- [[数据表-T_1_1-机构信息-一表通系统]]：关联来源，提供银行机构名称和金融许可证号。
