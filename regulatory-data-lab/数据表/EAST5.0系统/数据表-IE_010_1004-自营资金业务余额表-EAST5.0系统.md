# 数据表-IE_010_1004-自营资金业务余额表-EAST5.0系统

## 系统归属

- 所属系统：EAST5.0系统
- 所属库/域：EAST5.0 监管采集接口/数据字典域
- 是否存在跨系统同名表：待确认
- 文件路径是否位于正确系统目录：是

## 业务含义

- `IE_010_1004` 是 EAST5.0 `自营资金业务余额表` 的本地表结构定义，表注释为"自营资金业务余额表"。
- 本表记录截至报送日期有余额的自营资金业务及所持有的头寸信息。分三部分：投资情况（T_8_8）、同业存量情况（T_8_7）、融资情况（T_8_9）。

## 主键与核心标识

- `CJRQ`：采集日期，字段注释标识为 `PK`。
- 本表无单字段 PK，实际唯一性由采集日期+业务来源记录决定。
