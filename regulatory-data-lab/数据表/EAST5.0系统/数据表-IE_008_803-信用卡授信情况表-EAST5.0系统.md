# 数据表-IE_008_803-信用卡授信情况表-EAST5.0系统

## 系统归属

- 所属系统：EAST5.0系统
- 所属库/域：EAST5.0 监管采集接口/数据字典域
- 是否存在跨系统同名表：待确认
- 文件路径是否位于正确系统目录：是

## 业务含义

- `IE_008_803` 是 EAST5.0 `信用卡授信情况表` 的本地表结构定义，表注释为"信用卡授信情况表"。
- 本表报送信用卡持卡人当月授信情况。客户额度按同一账户下的总额度报送，不以单张信用卡额度报送。对主副卡客户（一账户多个客户）的情况，本表只报送主卡人的授信信息。同一客户多个账户的，按多条分别报送。同一账户多个币种共享额度的，按记账币种统一折算填报。状态为"销户"的账户在报送最后状态的次月可不再报送。

## 主键与核心标识

- `CJRQ`：采集日期，字段注释标识为 `PK`。
- `KHTYBH`：客户统一编号，字段注释标识为 `PK`。
- `XYKZH`：信用卡账号，字段注释标识为 `PK`。
- `NBJGH`：内部机构号，核心归属或业务标识字段。
- `YHJGMC`：银行机构名称，核心归属或业务标识字段。
- `JRXKZH`：金融许可证号，核心归属或业务标识字段。
- `ZJHM`：证件号码，核心归属或业务标识字段。
- `BZ`：币种，核心归属或业务标识字段。
