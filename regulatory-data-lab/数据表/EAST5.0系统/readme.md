# EAST5.0系统数据表目录说明

## 目录职责

本目录保存 EAST5.0 系统的源表、接口表、结果表和稳定中间数据集页面。

## 应包含

- EAST5.0 源表和结果表的字段口径、主键、码值和落地状态。
- 表级上下游摘要。
- 与 `报表业务口径/EAST5.0系统/`、`血缘/EAST5.0系统/`、`sql/EAST5.0系统/` 的互链。
- SQL 校准后形成的字段映射结论和缺口字段说明。

## 不应包含

- 存储过程全文。
- 只属于报表页的业务背景长说明。
- 未经 DDL、SQL 或既有知识页支持的字段结论。

## 状态判断

字段来源、码值和关键依赖未闭环时保持 `draft`。跑数或数据库语法未验证的 SQL 相关结论，应在 Open Questions 中保留风险。
