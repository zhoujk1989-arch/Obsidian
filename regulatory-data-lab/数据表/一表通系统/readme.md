# 一表通系统数据表目录说明

## 目录职责

本目录保存一表通系统表结构、接口表和报送数据集页面。

## 应包含

- 一表通报送表的字段口径、代码类字段和填报约束。
- 分户账、机构、客户、交易等稳定业务对象的表内指标说明。
- 外部监管来源 wikilink 和既有知识页证据。
- 与一表通报表页、血缘页和系统页的互链。

## 不应包含

- 大段填报说明原文。
- 单独码值页或字段页。
- 与一表通无直接主责关系的其他系统表。

## 特别约束

字段口径和指标口径优先落在本目录对应数据表页中；报表页只保留摘要和链接。
