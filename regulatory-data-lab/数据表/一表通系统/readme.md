# 一表通系统数据表目录说明

## 目录职责

本目录保存一表通系统表结构、接口表和报送数据集页面，作为表结构事实和字段主档。

## 应包含

- 表级基础信息：系统、表名、中文名、表类型、主题域。
- 表结构事实：字段清单、数据类型、长度、精度、是否非空、主键、索引或唯一约束。
- 字段主档：字段编码、中文名、DDL 注释、格式、是否必填。
- 码值/枚举：代码类字段取值范围、含义、特殊值、空值规则。
- 技术加工状态、SQL 产出状态和上下游对象统一查看对应血缘页。
- 证据摘要：DDL、字段字典、接口说明、外部实体来源等，不复制全文。
- 引用入口：加工本表或产出本表的 SQL、对应业务需求页、对应血缘页。

## 不应包含

- 大段填报说明原文。
- 一表通报表的业务需求加工规则。
- SQL 的完整 JOIN、过滤、CASE、UNION、聚合逻辑。
- 上下游摘要信息或完整字段级血缘。
- 单独码值页或字段页。
- 与一表通无直接主责关系的其他系统表。

## 特别约束

字段结构、格式、必填性、码值/枚举和落地状态优先落在本目录对应数据表页中；加工规则、业务需求和完整血缘分别由业务需求页和血缘页承载。
