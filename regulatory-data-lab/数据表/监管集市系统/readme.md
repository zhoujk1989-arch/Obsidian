# 监管集市系统数据表目录说明

## 目录职责

本目录保存监管集市系统下的机构、客户、公共维表或集市加工表页面，作为表结构事实和字段主档。

## 应包含

- 表级基础信息：系统、表名、中文名、表类型、主题域。
- 表结构事实：字段清单、数据类型、长度、精度、是否非空、主键、索引或唯一约束。
- 字段主档：字段编码、中文名、DDL 注释、格式、是否必填。
- 码值/枚举：代码类字段取值范围、含义、特殊值、空值规则。
- 字段落地状态：本地表结构是否存在、SQL 是否已明确产出、是否存在缺口。
- 证据摘要：DDL、字段字典、接口说明等，不复制全文。
- 引用入口：加工本表或消费本表的 SQL、对应业务需求页、对应血缘页。

## 不应包含

- 目标报表系统的完整业务口径。
- SQL 的完整 JOIN、过滤、CASE、UNION、聚合逻辑。
- 上下游摘要信息或完整字段级血缘。
- 未确认字段结构或字段约束的推断性结论。
- 与监管集市无关的源系统表。

## 联动要求

当本目录表作为其他系统 SQL 的上游来源时，本页只引用相关 SQL 或血缘页，不展开“我去到哪”的完整链路。
