# 监管集市系统数据表目录说明

## 目录职责

本目录保存监管集市系统下的机构、客户、公共维表或集市加工表页面。

## 应包含

- 集市表的业务角色、主键、核心字段和有效期规则。
- 作为其他监管系统来源表时的输出去向。
- 与相关血缘页、目标数据表页和系统页的互链。

## 不应包含

- 目标报表系统的完整业务口径。
- 未确认字段来源的推断性映射。
- 与监管集市无关的源系统表。

## 联动要求

当本目录表作为其他系统 SQL 的上游来源时，必须在本页记录“我去到哪”，目标数据表页记录“我从谁来”。
