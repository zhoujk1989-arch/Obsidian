# 金融基础数据系统数据表目录说明

## 目录职责

本目录保存金融基础数据系统接口表、专题数据集和结果表页面。

## 应包含

- JS 类表或专题表的字段口径、主键和报送范围。
- 代码类字段、专题标签和业务场景映射。
- 与金融基础数据系统报表页、血缘页和系统页的互链。

## 不应包含

- 五篇大文章等外部监管材料全文。
- 与本系统无关的其他监管系统表。
- 没有证据的字段闭环结论。

## 状态判断

专题来源、字段映射或证据路径待补时保持 `draft`，并在 Open Questions 中写明缺口。
