# 金融基础数据系统数据表目录说明

## 目录职责

本目录保存金融基础数据系统接口表、专题数据集和结果表页面，作为表结构事实和字段主档。

## 应包含

- 表级基础信息：系统、表名、中文名、表类型、主题域。
- 表结构事实：字段清单、数据类型、长度、精度、是否非空、主键、索引或唯一约束。
- 字段主档：字段编码、中文名、DDL 注释、格式、是否必填。
- 码值/枚举：代码类字段、专题标签字段的取值范围、含义、特殊值、空值规则。
- 技术加工状态、SQL 产出状态和上下游对象统一查看对应血缘页。
- 证据摘要：DDL、字段字典、接口说明、外部实体来源等，不复制全文。
- 引用入口：加工本表或产出本表的 SQL、对应业务需求页、对应血缘页。

## 不应包含

- 五篇大文章等外部监管材料全文。
- 金融基础数据报表或专题数据集的业务需求加工规则。
- SQL 的完整 JOIN、过滤、CASE、UNION、聚合逻辑。
- 上下游摘要信息或完整字段级血缘。
- 与本系统无关的其他监管系统表。
- 没有证据的字段闭环结论。

## 状态判断

表结构事实、字段主档、码值/枚举或证据路径待补时保持 `draft`，并在 Open Questions 中写明缺口。
