# 一表通系统SQL目录说明

## 目录职责

本目录保存一表通系统相关的 SQL 草案、加工脚本和校验脚本。

## 应包含

- 一表通报送表、接口表或校验规则相关 SQL。
- SQL 文件头部的来源、目标、字段覆盖和验证状态。
- 与一表通数据表页、血缘页、报表页的互链。

## 不应包含

- 其他系统主目标 SQL。
- 仅用于对话说明且无法复用的临时代码。
- 没有证据来源的字段映射。

## 联动要求

SQL 变更后必须同步检查一表通数据表页字段口径和血缘页双向边。
