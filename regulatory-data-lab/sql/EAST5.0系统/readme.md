# EAST5.0系统SQL目录说明

## 目录职责

本目录保存 EAST5.0 系统报表相关的存储过程草案、修复版和校验 SQL。

## 应包含

- EAST5.0 结果表加工 SQL、存储过程和校验脚本。
- 文件头部说明报表、目标表、依赖源表、字段覆盖、缺口和验证状态。
- 对应数据表页、血缘页和报表业务口径页的路径或链接提示。

## 不应包含

- 非 EAST5.0 主目标的 SQL。
- 没有字段级核对记录的最终版声明。
- 无来源说明的批量生成脚本。

## 状态判断

未在 GBase 或目标数据库完成语法验证和跑数验证的 SQL，只能标注为草案或待验证，不得写成生产可用结论。
