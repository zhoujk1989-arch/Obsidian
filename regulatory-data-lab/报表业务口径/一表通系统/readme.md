# 一表通系统报表业务口径目录说明

## 目录职责

本目录保存一表通系统下报表的业务口径页。

## 应包含

- 一表通报表的业务对象、报送边界和填报口径。
- 对分户账、机构、客户、交易等业务场景的可读解释。
- 外部 `regulatory-knowledge-vault` 来源 wikilink。
- 与一表通数据表页、血缘页和系统页的互链。

## 不应包含

- 完整监管原文。
- 数据表完整字段字典。
- 只用于 SQL 开发的临时推断。

## 特别约束

涉及分户账类报表时，优先使用“是否存在分户账核算/最细一级分户账号”作为稳定边界。
