# 金融基础数据系统报表业务需求目录说明

## 目录职责

本目录保存金融基础数据系统下报表或专题数据集的业务需求加工规则页。

页面用于把金融基础数据系统 SQL 或转换规则翻译成业务人员可评审的自然语言规则，并保留可反向生成 SQL 的加工约束。

## 应包含

- 金融基础数据报表或专题数据集的整表加工目标和报送模式。
- 来源选择、关联条件、过滤条件、专题标签规则和去重规则。
- 目标字段或指标的加工规则，不写业务含义解释。
- 五篇大文章等专题数据进入或排除报送范围的加工规则。
- 对应 SQL 文件引用。
- `regulatory-knowledge-vault` 中对应报表业务实体 wikilink。

## 不应包含

- 字段或指标的业务含义解释。
- 接口表字段字典全文。
- 完整字段级血缘。
- 数据表页、血缘页、系统页等本仓库其他知识页的正文引用。
- 未建立页面的占位链接。
- 与本系统无主责关系的其他系统报表说明。

## 联动要求

新增专题报表或数据集时，页面正文只引用对应 SQL 文件和外部报表业务实体；系统页同步检查作为 agent 内部维护动作处理。
