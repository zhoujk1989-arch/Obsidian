# 金融基础数据系统报表业务口径目录说明

## 目录职责

本目录保存金融基础数据系统下报表或专题数据集的业务口径页。

## 应包含

- 金融基础数据报送对象、业务范围和指标口径。
- 五篇大文章等专题数据的业务场景说明。
- 外部来源 wikilink、相关数据表页和血缘页链接。

## 不应包含

- 接口表字段字典全文。
- 未建立页面的占位链接。
- 与本系统无主责关系的其他系统报表说明。

## 联动要求

新增专题报表或数据集时，应检查 `概念/概念-系统-金融基础数据系统.md` 是否同步入口和覆盖摘要。
