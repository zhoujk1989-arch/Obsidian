# EAST5.0系统报表业务口径目录说明

## 目录职责

本目录保存 EAST5.0 系统下各张监管报表的业务口径页。

## 应包含

- EAST5.0 报表的报送对象、报送范围和过滤条件。
- 结果表字段或指标的业务解释。
- 与 `数据表/EAST5.0系统/`、`血缘/EAST5.0系统/`、`sql/EAST5.0系统/` 的互链。
- 监管外部来源的 Obsidian wikilink 或本仓库既有证据链接。

## 不应包含

- EAST5.0 存储过程全文。
- 源表 DDL 全量字段清单。
- 未核实来源的字段级血缘结论。
- 非 EAST5.0 主系统对象；跨系统依赖只在正文说明。

## 状态判断

未完成 SQL 语法验证、跑数验证或仍有阻塞性字段缺口时，页面保持 `draft`。
