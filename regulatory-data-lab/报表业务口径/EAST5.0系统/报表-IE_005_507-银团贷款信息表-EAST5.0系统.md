## 4. 字段级取数逻辑说明

### 4.1 日期类字段

- `采集日期`：默认值：报告日，数据格式转成yyyymmdd

### 4.2 码值/枚举转换字段

- `银团成员类型`：加工映射：将【银团贷款协议】.【银团成员类型】中码值按如下替换：

'02'转化为'代理行'
'03'转化为'参加行'
'00-XX'转化为'其他-XX'，其中'XX'为银行自定义
