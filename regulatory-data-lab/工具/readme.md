# 工具目录说明

## 目录职责

本目录保存仓库维护所需的本地工具脚本。

## 应包含

- 外部知识库检索工具。
- SQL 血缘脚手架。
- 可重复运行、输入输出清晰的仓库维护脚本。

## 不应包含

- 一次性临时脚本。
- 大型外部来源材料。
- 业务知识页面。
- 无说明、无参数约束、无输出说明的脚本。

## 使用要求

工具输出只能作为脚手架或候选证据。涉及业务口径、字段血缘和状态判断时，仍需回写到对应知识页并记录 `日志.md`。
