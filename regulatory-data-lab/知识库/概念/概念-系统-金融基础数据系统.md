# 概念-系统-金融基础数据系统

## 系统定位

金融基础数据系统（Financial Basic Data System）是中国人民银行主导的金融基础数据采集体系，核心产出为 JS_XXX_ 编码的监管报送表。当前版本为采集规范 V2.1。

## 系统边界

- 数据来源：金融机构内部业务系统（贷款合同、担保合同、客户信息等）
- 数据流向：金融机构 → 金融基础数据采集系统 → 人民银行
- 与一表通系统/EAST5.0 的关系：待确认是否存在映射或交叉报送
