## 业务定义

- 这张报表回答的业务问题：报送对公客户的财务信息，包括资产、负债、利润、现金流等核心财务指标。
- 统计对象：对公客户（含单一法人客户、同业客户、集团客户）
- 统计时点/周期：按财务报表日期统计，报送当月新增的财报记录

## 统计口径

### 纳入口径

- 当月在源系统（一表通 T_2_6）中新收集的对公客户财务信息
- 通过客户统一编号关联单一法人基本情况、同业客户基本情况、集团基本情况获取客户名称

### 排除口径

- 非授信客户
- 负债类客户
- 离岸对公客户
- 机关事业单位
- 未达到税务部门建账标准的个体工商户
- 注：当前 SQL 草案尚未实现排除逻辑，字段来源待确认

### 统计粒度

- 每条记录代表一个对公客户在某一财务报表日期下的财务信息

### 输出维度

- 按客户 + 财务报表日期 + 财务报表编号 + 采集日期 维度输出

## 整表报送范围

- 本表覆盖对公客户的财务信息报送，包含资产总额、负债总额、净利润、所得税、税前利润、主营业务收入、现金流量净额、应收账款、其他应收款等核心财务指标
- 增量报送：每月报送源系统中新收集的财务信息
- 客户名称取数优先级：单一法人对公客户名称 → 同业客户名称 → 集团名称

## 技术实现摘要

- 主源表：T_2_6（一表通客户财务信息），按 B060023（财报录入日期）筛选当月新增
- 关联补充：T_2_1（单一法人基本情况）、T_2_3（同业客户基本情况）、T_2_2（集团基本情况）、T_1_1（机构信息）
- 主要过滤：财报录入日期在当月范围内
- 主要加工：日期格式转换（YYYY-MM-DD → YYYYMMDD）、码值转换（是否审计 0/1 → 否/是）、派生字段（税前利润 = 净利润 + 所得税）
