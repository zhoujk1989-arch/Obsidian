- 本表报送对象：国家外汇管理局公布汇率的主要币种，填报各主要外币与人民币的折算汇率。其他货币对人民币的折算汇率，以当天美元兑人民币的基准汇率与同一天国际外汇市场其他货币兑美元汇率套算确定。机构可只填报常见的外币信息，但至少应包含其它表中填报使用过的所有外币。

## 技术实现摘要

- SQL 草案：`工作区/SQL开发/EAST5.0系统/PROC_EAST_IE_010_1001_INC_HLXXB_草案.sql`
- 主要处理：
  1. 主表 T_10_2 INNER JOIN T_1_2（机构关系）ON K020001=A020001 AND A020002='0'（仅取顶层机构）
  2. LEFT JOIN T_1_1（机构信息）ON A020001=A010001（补充机构名称、内部机构号、许可证号）
  3. WHERE SUBSTR(K020002,7,6)=LEFT(P_DATA_DATE,6)（汇率ID截取第七位到最后时间点等于当月）
  4. 直接映射：WBBZ(K020003)、ZBBSL(K020005)、BBZ(K020010)
  5. 加工映射：HLRQ(SUBSTR(K020002,7,8))、YHJGMC(A010005)、NBJGH(A010002)、JRXKZH(A010003)
  6. 规则映射：WBSL(CAST(100 AS DECIMAL(20,6)))、BBBZ('CNY')
  7. 参数赋值：CJRQ(P_DATA_DATE)
  8. 缺口字段：SENSITIVEFLAG/GSFZJG 置 NULL
- 当前限制：SQL 草案尚未在 GBase 环境执行语法校验和跑数验证，不能视作生产已验证口径。
