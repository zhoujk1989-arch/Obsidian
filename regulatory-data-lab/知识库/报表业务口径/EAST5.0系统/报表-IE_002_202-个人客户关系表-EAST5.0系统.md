## 业务定义

- 这张报表回答的业务问题：本行个人客户的所有关联关系，包括个人对个人、个人对对公的各类关联关系。本人为本人担保的关系不报送。
- 统计对象：一表通个人客户关系人（T_3_7）中，截至采集日有效的关系记录，以及上一采集日至采集日期间解除、失效、终结的所有终态关系记录。
- 统计时点：截至采集日。

## 统计口径

- 纳入口径：
  - 从一表通 T_3_7（个人客户关系人）中取采集日期等于跑批日期的记录
  - 关联 EAST5.0 IE_002_201（个人基础信息表）获取客户姓名、证件类别、证件号码、金融许可证号、内部机构号
  - 用 KHTYBH 关联 IE_002_201，按 NBJGH 升序取第一条去重
- 排除口径：
  - 解除关系日期非空且早于上一采集日的关系记录（已在上个周期前解除，不再报送）
  - 本人为本人担保的关系（KHTYBH = GXRKHTYBH 且 GXLX = '担保'）
  - 客户统一编号（KHTYBH）为空或空串的记录
- 统计粒度：每条记录代表一对客户关系在某一采集日期下的关联状态
- 输出维度：客户、关系人、关系类型、采集日期

## 整表报送范围

- 覆盖业务对象：个人客户的所有关联关系，包括个人对个人、个人对对公的相关关联关系。本人为本人担保的关系不报送。
- 来源系统：核心系统、客户关系系统等各类信息系统采集的个人客户关系信息。
- 报送模式：全量表，报送截至采集日有效的数据，以及上一采集日至采集日期间结清、失效、终结等所有视为终态的数据。

## 技术实现摘要

- 主要源表：一表通 `T_3_7`（个人客户关系人）、EAST5.0 `IE_002_201`（个人基础信息表）
- 关系圈定来源：T_3_7 当期记录，排除提前解除和自担保
- 主要过滤条件：采集日期过滤、解除日期≥上一采集日终态保留、KHTYBH 非空、排除自担保
- 码值转换：GXRZJLB（1999-XX / 2999-XX → 其他-XX）、GXZT（解除日期 ≤ 跑批日期 → 无效，否则有效）
- 存储过程：`PROC_EAST_IE_002_202`

## 字段或指标特殊规则

- 主键联合唯一约束：GXLX + CJRQ + KHTYBH + GXRZJHM。
- IE_002_201 LEFT JOIN 按 KHTYBH 去重，ROW_NUMBER() ORDER BY NBJGH 取第一条；若 IE_002_201 中同一客户跨多机构存在多条记录，JRXKZH 和 NBJGH 以机构号最小为准。
- IE_002_201 可能因机构关联缺失导致 JRXKZH、NBJGH 为空，当前不做强制校验。
- 关系状态（GXZT）加工中"解除日期 ≤ 跑批日期"判定为无效，跑批日期为存储过程入参 p_data_date 转换后的 DATE 值。
- 本人为本人担保的判断：KHTYBH = GXRKHTYBH 且 GXLX = '担保'。当前假设 T_3_7.C070004（社会关系）存储中文'担保'，如为编码需替换对应码值。
- 上一采集日 = DATE_SUB(p_data_date, INTERVAL 1 DAY)，与 PROC_EAST_IE_002_201 保持一致。
- `SENSITIVEFLAG`、`GXRKHLB` 和 `GSFZJG` 三个字段当前无映射来源，均置空。
