## SQL 加工逻辑（2026-05-10 重构后）

### 主表与关联
- FROM `T_9_4`（商业单据）src
- INNER JOIN 子查询 T11（协议与单据对应关系 T_3_8）：
  - ON `src.J040001 = T11.C080003`（单据ID）
  - AND `src.J040009 = T11.C080002`（协议ID）
  - AND `src.J040008 = T11.C080007`（采集日期）
- LEFT JOIN `T_6_2` DK2（贷款协议）：ON `T11.C080002 = DK2.F020001 AND T11.C080007 = DK2.F020063`
- LEFT JOIN `T_6_11` s1（信用证协议）：ON `T11.C080002 = s1.F110001 AND T11.C080007 = s1.F110038`（加采集日期条件）
- LEFT JOIN `T_6_13` s2（票据协议）：ON `T11.C080002 = s2.F130001 AND T11.C080007 = s2.F130049`（加采集日期条件）
- LEFT JOIN `T_6_10` s3（贸易融资协议）：ON `T11.C080002 = s3.F100002 AND T11.C080007 = s3.F100026`（加采集日期条件）
- LEFT JOIN `T_1_1` s4（机构信息）：ON `src.J040002 = s4.A010001 AND s4.A010020 = V_DATA_DATE`
- LEFT JOIN `T_6_12` s5（保函及其他担保协议）：ON `T11.C080002 = s5.F120001 AND T11.C080007 = s5.F120028`（加采集日期条件）

### 子查询 T11（表级规则 Excel第1349行）
- FROM `T_3_8`（协议与单据对应关系）
- LEFT JOIN `T_6_2` DK（贷款协议）：ON 协议ID + 采集日期 + 贷款协议起始日期在本月
- LEFT JOIN `T_3_8` LST（上月对应关系）：ON 协议ID + 单据ID + 上月末采集日期
- WHERE `T_3_8.C080007 = V_DATA_DATE` AND（DK.协议ID非空 OR LST.协议ID为空）

### WHERE 过滤
- `src.J040008 = V_DATA_DATE`（商业单据采集日期 = 报告日）
- `T11.C080008 IN ('03','08','09','10','00')`（限制业务种类为保理融资/承兑汇票/保函/信用证/其他业务）
- `s4.A010001 IS NOT NULL`（确保机构信息关联存在）

### 码值 CASE 转换
1. **YWZL（业务种类）**：`T11.C080008` → 13 分支 CASE + '00%' 通配 → '其他-XX' + ELSE ''
2. **DJZL（单据种类）**：`src.J040006` → 13 分支 CASE + '00%' 通配 → REPLACE('00','') + ELSE ''

### 日期转换
- **CJRQ（采集日期）**：`CONCAT(YEAR(src.J040008), LPAD(MONTH(),2,'0'), LPAD(DAY(),2,'0'))`

### 金额 CAST
- **DJJE**：`CAST(NULLIF(TRIM(src.J040005), '') AS DECIMAL(20,2))`
- **HTJE**：`CAST(NULLIF(TRIM(COALESCE(s1.F110009, s2.F130020, s3.F100006, s5.F120009)), '') AS DECIMAL(20,2))`
