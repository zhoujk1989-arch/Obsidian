## 使用说明

- 查某张 EAST 表从哪里来：先看“EAST 目标到一表通来源”，再进入对应单表血缘页看字段级边。
- 查某张一表通表影响哪些 EAST 表：看“一表通来源到 EAST 下游”，再进入源表页的“EAST5.0 SQL 草案下游消费”章节。
- 当前依赖关系用于 SQL 草案评审和影响分析；投产前必须补齐 JOIN、码值 CASE、增量边界和跑数验证。
