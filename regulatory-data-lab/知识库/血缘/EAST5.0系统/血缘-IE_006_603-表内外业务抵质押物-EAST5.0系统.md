## 业务链路摘要

- 按 `原始材料/业务需求/EAST5.0/043_表内外业务抵质押物.md` 的字段映射，将一表通来源表加工为 EAST5.0 `表内外业务抵质押物`。
- 表级规则：取报送日期为当月，通过担保协议ID关联生成担保合同号，取不为保证金担保的数据作为报送范围。
- SQL 草案采用按 `P_DATA_DATE` 删除后重插的全量重跑方式。
