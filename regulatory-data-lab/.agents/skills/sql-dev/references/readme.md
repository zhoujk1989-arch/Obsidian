# sql-dev references目录说明

## 目录职责

本目录保存 SQL 开发技能使用的参考材料、检查清单或示例说明。

## 应包含

- 可复用的 SQL 开发参考规则。
- 字段映射、血缘抽取、校验 SQL 编写的示例或清单。
- 与本仓库监管报表知识页结构一致的参考说明。

## 不应包含

- 监管原文全文。
- 具体任务产物。
- 正式 SQL 草案或知识页。

## 维护要求

参考材料只能辅助判断，不能替代对 DDL、SQL、外部来源 wikilink 和既有知识页的实际核对。
