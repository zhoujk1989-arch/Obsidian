# sql-dev agents目录说明

## 目录职责

本目录保存 SQL 开发技能内部可复用的子智能体说明或分工提示。

## 应包含

- 面向 SQL 校准、字段映射、血缘核对等子任务的角色说明。
- 与 `../SKILL.md` 和 `.agents/workflows/SQL开发.md` 一致的执行约束。

## 不应包含

- 具体系统的 SQL 草案。
- 正式知识页。
- 与仓库级 `AGENTS.md` 冲突的工作方法。

## 维护要求

新增子智能体说明前，应确认该分工能被多次复用；一次性任务分工不应沉淀到本目录。
