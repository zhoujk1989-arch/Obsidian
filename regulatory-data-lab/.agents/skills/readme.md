# skills目录说明

## 目录职责

本目录保存可复用的专门能力说明，用于支撑复杂任务。

## 应包含

- SQL 开发、字段映射、血缘抽取等专门能力。
- 技能入口说明和必要的辅助脚本或示例。

## 不应包含

- 单次任务产物。
- 知识库正式页面。
- 与 workflow 重复的完整执行流程。

## 维护要求

技能应被 workflow 明确调用；如果某项能力只服务一个流程，优先写入对应 workflow，不单独建 skill。
