# 模板目录说明

## 目录职责

本目录保存报表业务口径页、数据表页、血缘页、概念页和 SQL 开发的标准模板。

## 应包含

- 可直接复制使用的页面模板。
- SQL 开发模板。
- 与 `AGENTS.md`、`.agents/rules/页面写作规则.md` 保持一致的章节结构。

## 不应包含

- 具体报表、数据表、血缘或概念实例。
- 临时草稿。
- 与当前仓库结构冲突的旧模板。

## 维护要求

修改模板后，应同步检查 `AGENTS.md`、`.agents/rules/页面写作规则.md` 和 `日志.md` 是否需要更新。
