## 系统边界

- 起始系统：一表通系统
- 目标系统：EAST5.0系统
- 是否跨系统血缘：是
- 目标对象：`IE_005_508` `票据贴现表`

## 业务链路摘要

- 按 历史业务需求材料 的字段映射，将一表通来源表加工为 EAST5.0 `票据贴现表`。
- 表级规则：取日期在当月且剔除上月状态为失效的数据
- 关联逻辑：T_1_1（机构信息）与 T_6_13（票据协议）通过 机构ID（A010001 = F130003）关联
- 过滤条件：T_6_13.F130049（采集日期）= P_DATA_DATE
- SQL 草案采用按 `P_DATA_DATE` 清理后重插方式；具体投产方式待验证。
