## 业务链路摘要

- 按 历史业务需求材料 的字段映射，将一表通来源表加工为 EAST5.0 `自营资金业务余额表`。
- 本表分三部分 UNION ALL：投资情况（T_8_8）、同业存量情况（T_8_7）、融资情况（T_8_9）。
- 每部分独立完成 JOIN 关联、WHERE 过滤和字段映射，并关联上月末数据判断终态纳入。
- SQL 草案采用按 `P_DATA_DATE` 清理后重插方式；所有 JOIN 条件和 WHERE 过滤已按业务需求补齐。
