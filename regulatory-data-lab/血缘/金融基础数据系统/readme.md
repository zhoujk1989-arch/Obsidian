# 金融基础数据系统血缘目录说明

## 目录职责

本目录保存金融基础数据系统数据表的表级上下游、字段级血缘和代码字段加工使用摘要。

## 应包含

- 当前金融基础数据系统表标识：系统、表名、中文名、表类型。
- SQL 证据：加工本表或消费本表的 SQL 文件、过程、视图。
- 表级上游关系：上游表到当前表。
- 表级下游关系：当前表到下游表。
- 字段级血缘：源表、源字段、目标表、目标字段、处理逻辑、关系类型、SQL 定位、确认状态。
- 代码字段加工使用摘要：代码转换、专题标签加工、默认值、特殊值、异常处理和 SQL 定位。
- 血缘缺口：缺 SQL、缺源字段、缺目标字段、缺码值规则、缺确认。

## 不应包含

- 外部监管材料全文。
- 金融基础数据报表业务信息或专题业务解释。
- 业务需求页里的加工规则全文。
- 数据表字段字典全文。
- 没有 SQL 或可确认加工链路证据的结论。
- 其他系统的完整血缘主页面。

## 状态判断

SQL 证据、表级上下游、字段级血缘、代码字段加工摘要或缺口状态未闭环时保持 `draft`。
