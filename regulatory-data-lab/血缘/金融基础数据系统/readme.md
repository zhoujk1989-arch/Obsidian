# 金融基础数据系统血缘目录说明

## 目录职责

本目录保存金融基础数据系统专题数据、接口表和结果表的来源去向关系。

## 应包含

- 金融基础数据系统表级链路和字段级加工逻辑。
- 专题标签、代码转换和跨系统来源依赖。
- 与金融基础数据系统数据表页、报表页和系统页的互链。

## 不应包含

- 外部监管材料全文。
- 没有源表或知识页证据的结论。
- 其他系统的完整血缘主页面。

## 状态判断

来源文件、字段映射或双向互链未闭环时保持 `draft`。
