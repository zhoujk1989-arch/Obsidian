# 一表通系统血缘目录说明

## 目录职责

本目录保存一表通系统报送表、接口表和业务对象之间的血缘关系。

## 应包含

- 一表通报送链路的表级和字段级来源。
- 分户账、机构、客户、交易等业务对象的上下游关系。
- 与一表通数据表页、报表页和系统页的互链。

## 不应包含

- 监管填报说明全文。
- 只属于业务口径页的解释。
- 未确认来源的字段级映射。

## 特别约束

对跨系统来源只记录必要链路和证据链接，不通过共享目录混放其他系统对象。
