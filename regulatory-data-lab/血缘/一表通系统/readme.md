# 一表通系统血缘目录说明

## 目录职责

本目录保存一表通系统数据表的表级上下游、字段级血缘。

## 应包含

- 当前一表通表标识：系统、表名、中文名、表类型。
- SQL 证据：加工本表或消费本表的 SQL 文件、过程、视图。
- 表级上游关系：上游表到当前表。
- 表级下游关系：当前表到下游表。
- 字段级血缘：源表、源字段、目标表、目标字段、处理逻辑、关系类型、SQL 定位、确认状态。
- 血缘缺口：缺 SQL、缺源字段、缺目标字段、缺码值规则、缺确认。

## 不应包含

- 监管填报说明全文。
- 一表通报表业务信息或报送口径解释。
- 业务需求页里的加工规则全文。
- 数据表字段字典全文。
- 未确认来源的字段级映射。

## 特别约束

对跨系统来源只记录表级边、字段级边和 SQL 证据；不通过共享目录混放其他系统对象。
